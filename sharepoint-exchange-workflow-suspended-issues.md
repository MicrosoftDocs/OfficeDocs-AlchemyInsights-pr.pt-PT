# <a name="workflows-in-sharepoint"></a>Fluxos de trabalho no SharePoint

<p>Se os fluxos de trabalho do SharePoint não estão a enviar mensagens de correio electrónico, a organização poderá ter encontrado os limites de remetente Exchange Online.&nbsp;</p> <p>'Fluxo de trabalho é suspenso' mensagem de erro poderá ocorrer se tiver um dos seguintes itens:</p> <ul> <li>Ter um fluxo de trabalho no SharePoint Online que está a utilizar o SharePoint 2010 ou o tipo de plataforma de fluxo de trabalho do SharePoint de 2013.</li> <li>O fluxo de trabalho está configurado para enviar uma mensagem de correio electrónico personalizado a mais de 200 utilizadores cada vez, mais de 10.000 destinatários por dia ou mais de 30 mensagens por minuto.</li> <li>Quando executar o fluxo de trabalho, a mensagem de correio electrónico não é enviada e repare a mensagem de erro <strong>interno de estado estiver definido como suspenso</strong> ou <strong>não foi possível enviar para um destinatário</strong> é apresentado.</li> </ul> <p>Para mais informações, consulte o seguinte <a href="https://support.office.com/en-us/article/-daily-email-limit-has-exceeded-and-your-workflow-has-been-suspended-or-unable-to-send-to-a-recipient-error-in-a-sharepoint-online-workflow-89d02169-5fa6-4259-affc-73edb6ca9fb6?ui=en-US&amp;rs=en-US&amp;ad=US">artigo</a>.</p>
