# <a name="error-404-when-accessing-files-in-sharepoint-or-onedrive"></a>Erro 404 quando acede a ficheiros no SharePoint ou OneDrive

<strong>Erro 404 </strong>é recebida quando os utilizadores estão a tentar aceder a um site ou ficheiro no SharePoint ou OneDrive. Isto é frequentemente causado por um site ou ficheiro ou grupo a obter o nome mudado, movido ou eliminado.
Os utilizadores ocorrerá um erro 404 a tentar aceder a colecção de sites de raiz e ter sido eliminado.

Para resolver o erro 404 para um Site que tenha sido mudado, movido ou eliminado: para sites clássicos que existem no Centro de administração de clássico, consulte <a href="https://docs.microsoft.com/en-us/sharepoint/restore-deleted-site-collection">Restaurar uma colecção de sites eliminada</a>.&nbsp;</li> <li>Para modernas sites (comunicação, ligados por grupo, ou outros sites) que existem na nova pré-visualização de centro de administração, consulte <a href="https://docs.microsoft.com/en-us/sharepoint/view-and-restore-deleted-sites-in-new-admin-center">e restauro de vista eliminada sites o novo Centro de administração do SharePoint. 

Para resolver o ficheiro (ou outro item) que foi mudado, movido ou eliminado Ir para o site SharePoint ou OneDrive e visualizar a ' Reciclagem ' o conteúdo de Site, consulte [Restaurar itens na Reciclagem de um site SharePoint](https://support.office.com/en-us/article/Restore-items-in-the-Recycle-Bin-of-a-SharePoint-site-6df466b6-55f2-4898-8d6e-c0dff851a0be).
