# <a name="give-users-access-to-sharepoint-site"></a>Dar aos utilizadores acesso ao SharePoint site

<p>A funcionalidade de pedido de acesso permite às pessoas a pedir acesso ao conteúdo que não têm actualmente permissão para ver. Como proprietário do site, pode configurar a funcionalidade para enviar correio, quando alguém pedidos de acesso a um site. Para mais informações, visite <a href="https://support.office.com/en-us/article/set-up-and-manage-access-requests-94b26e0b-2822-49d4-929a-8455698654b3">definir uma cópia de segurança e gerir pedidos de acesso</a>.</p>  <p>Eis algumas hiperligações comuns sobre os pedidos de acesso:</p>  <ul>  <li><a href="https://support.office.com/en-us/article/-Access-Denied-to-Access-Requests-list-or-Request-approval-failed-when-you-process-a-pending-request-in-SharePoint-Online-3ec71fe1-9111-4a4c-84a5-bdfe05c2e558">"Acesso negado" à lista de pedidos de acesso ou "Pedir a aprovação falhou" quando processa um pedido pendente no SharePoint Online</a></li>  <li><a href="https://support.office.com/en-us/article/set-up-and-manage-access-requests-94b26e0b-2822-49d4-929a-8455698654b3#bk_enableallow">Definições de pedidos de acesso de alteração</a></li>  <li><a href="https://support.office.com/en-us/article/set-up-and-manage-access-requests-94b26e0b-2822-49d4-929a-8455698654b3#__toc374462557">Aprovar ou recusar pedidos de acesso para um site</a></li>  <li><a href="https://support.office.com/en-us/article/set-up-and-manage-access-requests-94b26e0b-2822-49d4-929a-8455698654b3#__toc374462558">Gerir convites para utilizadores externos (apenas Online SharePoint)</a></li>  <li><a href="https://support.office.com/en-us/article/set-up-and-manage-access-requests-94b26e0b-2822-49d4-929a-8455698654b3#__toc334189260">Retirar os convites</a></li>  <li><a href="https://support.office.com/en-us/article/set-up-and-manage-access-requests-94b26e0b-2822-49d4-929a-8455698654b3#__toc374462560">Reenviar um convite</a></li>  </ul>
