# <a name="dial-pad-is-missing-in-microsoft-teams"></a>O bloco de discagem está faltando nas equipes da Microsoft 

Se o seu bloco de discagem estiver faltando no Microsoft Teams, verifique o seguinte:

- Certifique-se de que o usuário tenha sido atribuído uma [licença de equipes.](https://docs.microsoft.com/MicrosoftTeams/assign-teams-licenses)
- Certifique-se de que o usuário tenha um plano de [chamada](https://docs.microsoft.com/MicrosoftTeams/calling-plan-landing-page) atribuído.
- Habilite os usuários para o [Enterprise Voice](https://docs.microsoft.com/skypeforbusiness/skype-for-business-hybrid-solutions/plan-your-phone-system-cloud-pbx-solution/enable-users-for-enterprise-voice-online-and-phone-system-voicemail#to-enable-your-users-for-phone-system-in-office-365-voice-and-voicemail).
