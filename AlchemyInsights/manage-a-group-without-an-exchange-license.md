# <a name="manage-a-group-without-an-exchange-license"></a>Gerir um Grupo sem Licença de Câmbio

Se a sua conta de administração não tiver uma caixa de correio, poderá obter um erro sobre uma licença de produto em falta quando tentar editar um Microsoft 365 Group.
  
Como você é o administrador inquilino, é possível para si editar um grupo, mesmo que não tenha uma licença de troca. Basta ir ao [Exchange Admin Center](https://outlook.office365.com/ecp.aspx) \> **Groups** para gerir o Grupo Microsoft 365.
  
Se quiser editar Grupos no centro de administração, adicione uma licença De Troca à sua conta de administração. Tem de esperar até que a caixa de correio seja totalmente criada antes de poder editar um grupo.
