# <a name="use-microsoft-edge-to-clear-the-browsing-data-stored-by-internet-explorer"></a>Utilize o Microsoft Edge para limpar os dados de navegação armazenados pelo Internet Explorer

Esta funcionalidade só está disponível se a sua organização tiver ligado o modo Internet Explorer (IE). Note que limpar os dados de navegação do IE não afetará os seus dados de navegação no Microsoft Edge.
*
1. No Microsoft Edge, selecione **Definições e muito mais ...**  >  **Configurações**  >  **Privacidade e serviços.**

1. Em **dados de navegação claros para o Internet Explorer,** **selecione Escolha o que limpar**.

1. Selecione os tipos de dados que pretende limpar.

1. Selecione **Limpar agora**.
