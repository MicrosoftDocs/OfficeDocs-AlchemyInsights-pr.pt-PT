# <a name="control-the-access-from-unmanaged-devices"></a>Controlar o acesso a dispositivos não geridos

Como SharePoint ou administrador global no Microsoft 365, pode bloquear ou limitar o acesso ao conteúdo sharePoint e OneDrive a partir de dispositivos não geridos. Para obter mais informações, consulte [o Acesso ao Controlo a partir de dispositivos não geridos.](https://docs.microsoft.com/sharepoint/control-access-from-unmanaged-devices)
