# <a name="enable-the-custom-domain-for-dkim"></a>Ativar o domínio personalizado para DKIM

Depois de criar os registos CNAME para os seus domínios personalizados, tem de ativar o domínio.

Para ativar o domínio, execute os seguintes passos:

1. Navegue para o [centro de administração Exchange](https://outlook.office365.com/ecp/).
2. No painel esquerdo, selecione **proteção > dkim**.
3. Selecione o domínio e, em seguida, em **Mensagens de Sinais para este domínio com assinaturas DKIM**, clique em **Ativar**. Repita este passo para cada domínio.
