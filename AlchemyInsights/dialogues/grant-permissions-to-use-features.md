# <a name="grant-permissions-to-use-features-in-the-office-365-security--compliance-center"></a>Conceder permissões para utilizar funcionalidades no Office 365 Security & Compliance Center

Para conceder permissão a um utilizador para executar uma tarefa de conformidade, adicione o utilizador ao grupo de funções apropriado no Centro de Conformidade de Segurança & de Segurança do Office 365.

Para obter mais informações, consulte [permissões necessárias para utilizar funcionalidades no Centro de Conformidade & de Segurança](https://docs.microsoft.com/microsoft-365/security/office-365-security/permissions-in-the-security-and-compliance-center).
