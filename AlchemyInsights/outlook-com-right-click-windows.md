# <a name="how-to-right-click-on-a-tablet"></a>Como botão direito do rato num Tablet PC

Para o botão direito do rato sobre uma superfície ou no Windows tablet, toque e mantenha a sua selecção durante alguns segundos. Quando libertar, será apresentado o menu de contexto.
