# <a name="learn-about-the-progressive-web-apps-pwas-on-microsoft-edge-chromium"></a>Conheça as Aplicações Web Progressivas (PWAs) no Microsoft Edge (Chromium)

As Aplicações Web Progressivas (PWAs) são websites progressivamente melhorados para funcionarem como aplicações nativas em plataformas que as suportam. Os PWAs combinam as melhores qualidades das aplicações web e nativas.

Do ponto de vista de uma plataforma web, [os PWAs no Microsoft Edge (Chromium)](https://docs.microsoft.com/microsoft-edge/progressive-web-apps-chromium/#pwas-on-microsoft-edge-chromium) são baseados em padrões. Permitem que os utilizadores instalem aplicações diretamente a partir do navegador, e não necessitam de implantação ou registo baseados na Microsoft Store. Os PWAs de desktop são suportados em qualquer plataforma onde o Microsoft Edge (Chromium) esteja disponível.
