# <a name="troubleshooting-outlook-mobile-setup-for-a-shared-mailbox"></a>Solução de problemas para dispositivos móveis do Outlook para uma caixa de correio compartilhada

Não pode aceder à caixa de correio partilhada a partir da aplicação Outlook que está instalada no dispositivo móvel. Quer esta funcionalidade? Deixe a equipe de Engenharia outlook saber. [Vote aqui!](https://go.microsoft.com/fwlink/?linked=862116)
  
Você só pode usar uma caixa de correio compartilhada em um dispositivo móvel, abra seu navegador, inscrevendo-se no Office 365 e, em seguida, acesse o Outlook na web. Para obter instruções, [consulte adicionar uma caixa de correio compartilhada no Outlook na web](https://support.office.com/article/add-a-shared-mailbox-to-outlook-on-the-web-98b5a90d-4e38-415d-a030-f09a4cd28207). A partir do Outlook na Web, poderá aceder à caixa de correio partilhada.
