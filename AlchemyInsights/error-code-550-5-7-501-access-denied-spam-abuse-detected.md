# <a name="550-57501-access-denied-spam-abuse-detected"></a>550 5.7.501 acesso negado, abuso de spam detectado

Normalmente, esta mensagem ocorre quando os utilizadores enviam mensagens de correio electrónico de endereços IP utilizando o domínio *. onmicrosoft.com* inicial atribuído a novos inquilinos no Office 365. A maneira mais fácil de resolver esse problema é:

1. [Adicione um domínio ao seu locatário](https://docs.microsoft.com//office365/admin/setup/add-domain).

2. [Altere o endereço de e-mail principal dos seus usuários](https://docs.microsoft.com//office365/admin/add-users/change-a-user-name-and-email-address) para o novo domínio personalizado que você acabou de adicionar.
