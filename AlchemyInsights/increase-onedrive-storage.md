# <a name="increase-onedrive-storage"></a>Aumentar o armazenamento oneDrive

Como administrador global ou SharePoint na Microsoft 365, pode definir o espaço de armazenamento OneDrive para um utilizador específico diretamente do centro de administração da Microsoft.  

Para obter mais informações, consulte [alterar o espaço de armazenamento OneDrive de um utilizador específico.](https://docs.microsoft.com/onedrive/change-user-storage)
