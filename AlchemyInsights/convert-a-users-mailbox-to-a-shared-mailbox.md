# <a name="convert-a-user-mailbox-to-a-shared-mailbox"></a>Converter uma caixa de correio de utilizador numa caixa de correio partilhada.

Quando converte a caixa de correio de um utilizador individual para uma caixa de correio partilhada, todos os itens de e-mail e calendário existentes são retidos numa caixa de correio partilhada onde várias pessoas podem aceder. Também pode converter uma caixa de correio partilhada de volta para uma caixa de correio (privada) do utilizador.
  
Para mais informações, consulte [Converter uma caixa de correio para uma caixa de correio partilhada](https://docs.microsoft.com/office365/admin/email/convert-user-mailbox-to-shared-mailbox).
