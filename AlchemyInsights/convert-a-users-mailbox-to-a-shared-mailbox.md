# <a name="convert-a-user-mailbox-to-a-shared-mailbox"></a>Converter uma caixa de correio do utilizador para uma caixa de correio partilhada

Quando converter caixa de correio de um utilizador individual para uma caixa de correio partilhada, todos os itens de calendário e correio electrónico existente são mantidos numa caixa de correio partilhada, onde podem aceder a várias pessoas. Também pode converter uma caixa de correio partilhada para uma caixa de correio (privados) do utilizador.
