# <a name="convert-a-user-mailbox-to-a-shared-mailbox"></a>Converter uma caixa de correio do utilizador para uma caixa de correio partilhada

Quando converter caixa de correio de um utilizador uma caixa de correio partilhada, todo o correio electrónico existente e o calendário é mantida. Só é na caixa de correio partilhada, onde várias pessoas consigam aceder-lhe em vez de uma pessoa. Posteriormente, pode converter uma caixa de correio partilhada para uma caixa de correio (privados) do utilizador.
