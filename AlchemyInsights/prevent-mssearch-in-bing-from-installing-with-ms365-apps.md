# <a name="prevent-microsoft-search-in-bing-from-installing-with-microsoft-365-apps"></a>Impedir que a Pesquisa do Microsoft em Bing seja instalada com aplicações microsoft 365

Para evitar que o serviço de fundo do Microsoft Search em Bing instale com as Aplicações Microsoft 365 para empresa, utilize uma das seguintes opções:

- A ferramenta de implementação do office ou a política de grupo
- Microsoft Endpoint Configuration Manager (atual filial) ou Microsoft Intune

Para saber mais, consulte [Como evitar que o serviço de fundo do Microsoft Search em Bing seja instalado com aplicações microsoft 365 para empresa.](https://go.microsoft.com/fwlink/?linkid=2151946)
