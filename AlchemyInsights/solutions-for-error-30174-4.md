# <a name="error-code-30174-4-when-installing-office"></a>Código de erro 30174-4 quando instalar o Office

Poderá ver este erro se o software antivírus, firewall, definições de proxy ou ligação impedirem a instalação do Office. Eis algumas coisas que pode tentar corrigir o problema.
  
- Utilizar uma ligação com fios

- Desactivar temporariamente as definições de proxy

- Desactivar temporariamente o software antivírus
