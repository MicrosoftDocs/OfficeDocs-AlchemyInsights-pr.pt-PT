# <a name="fix-problems-with-sharepoint-libraries-mapped-to-network-drives"></a>Corrigir problemas com bibliotecas do SharePoint mapeadas para unidades de rede

Quando navega para uma unidade de rede mapeada, pode ver uma das seguintes mensagens:
  
- **\\O caminho não é acessível. Pode não ter autorização para usar este recurso de rede. Contacte o administrador deste servidor para saber se tem permissões de acesso.**

- **Acesso negado. Antes de abrir ficheiros neste local, tem primeiro de adicionar o web site à sua lista de sites fidedignos, navegar no site e selecionar a opção de iniciar sessão automaticamente.**

[Obtenha ajuda para resolver as unidades de rede mapeadas.](https://docs.microsoft.com/sharepoint/support/administration/troubleshoot-mapped-network-drives)
  
Mapear uma biblioteca como unidade de rede é temporário e suportado apenas no Internet Explorer. Em vez disso, [sincroniza os ficheiros SharePoint com o novo cliente de sincronização OneDrive](https://support.office.com/article/6de9ede8-5b6e-4503-80b2-6190f3354a88.aspx) que inclui [ficheiros on-demand](https://support.office.com/article/0e6860d3-d9f3-4971-b321-7092438fb38e.aspx). Aceda a todos os seus ficheiros no OneDrive sem utilizar o espaço de armazenamento local.
