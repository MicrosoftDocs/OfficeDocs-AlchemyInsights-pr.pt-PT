# <a name="how-to-customize-permissions-for-a-sharepoint-list-or-library"></a>Como personalizar permissões para uma lista ou biblioteca sharePoint

Como administrador ou proprietário de uma biblioteca, lista ou levantamento, pode alterar permissões para permitir que um grupo de pessoas aceda aos dados de que necessitam, restringindo outros.

[Personalize permissões para uma lista ou biblioteca sharePoint](https://support.office.com/article/customize-permissions-for-a-sharepoint-list-or-library-02d770f3-59eb-4910-a608-5f84cc297782)
