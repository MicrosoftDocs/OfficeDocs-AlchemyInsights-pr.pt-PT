# <a name="get-business-assist"></a>Obter Assistência para Empresas

1. Na ação centro de administração do Microsoft 365, vá para **Assistência Empresarial de**  >  **Subscrições.**

1. **Selecione Business Assist**  >  **Buy** e siga o assistente para comprar Assistência para Empresas.

Para saber mais, consulte [o Assistência empresarial para o Microsoft 365](/microsoft-365/admin/misc/business-assist).
