# <a name="the-underlying-connection-was-closed-error-in-sharepoint"></a>Erro de "A ligação subjacente foi encerrada" no SharePoint

Se estiver a receber o erro "A ligação subjacente foi fechada" no SharePoint, poderá estar relacionado com a depreciação de TLS 1.0/1.1. Para mais informações, consulte estes artigos:

- [Preparando-se para o TLS 1.2 no Office 365 e Office 365 GCC](https://docs.microsoft.com/microsoft-365/compliance/prepare-tls-1.2-in-office-365?view=o365-worldwide)

- [Erros de autenticação ocorrem se o cliente não tiver suporte TLS 1.2](https://review.docs.microsoft.com/sharepoint/troubleshoot/administration/authentication-errors-tls12-support)
