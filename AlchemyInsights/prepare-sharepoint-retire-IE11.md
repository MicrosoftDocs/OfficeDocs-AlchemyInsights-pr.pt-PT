# <a name="prepare-your-sharepoint-environment-for-the-retirement-of-internet-explorer-11"></a>Preparar o seu ambiente do SharePoint para a desmoção do Internet Explorer 11

Apesar de o suporte para o Internet Explorer 11 continuar, mudamos os recursos de desenvolvimento do browser para [o browser Microsoft Edge Browser](https://www.microsoft.com/edge/business). 

Para obter mais informações, consulte Preparar o seu ambiente do SharePoint para a despromoção do [Internet Explorer 11 para Microsoft 365 e serviços.](/sharepoint/prepare-ie11)
