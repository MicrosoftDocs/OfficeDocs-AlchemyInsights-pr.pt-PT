# <a name="how-do-i-turn-off-skype-in-outlookcom"></a>Como desactivar Skype em Outlook.com

Não é possível remover o Skype de Outlook.com, mas pode desactivar as notificações:

1. Abra **definições de** > **Geral** > **[notificações](https://outlook.live.com/mail/options/general/notifications)**. 
2. Em **função de conversação e chamada**, desactive os tipos de notificação que já não pretende.

Para mais informações sobre como utilizar o Skype, leia a [falar para familiares e amigos Skype em Outlook.com](https://support.office.com/article/83c6a5b1-3921-479c-b9e9-e753ce59c1fa).
