# <a name="cant-add-or-change-teams-profile-picture"></a>Não é possível adicionar ou alterar a imagem do perfil das equipas

Para adicionar ou alterar uma imagem de perfil, os utilizadores devem estar totalmente habilitados com o seguinte:

- Exchange Online
- SharePoint Online
- Capacidade de criar grupos Microsoft 365

Consulte a administração das suas equipas sobre a obtenção das permissões necessárias. Para obter instruções, consulte [Alterar a sua imagem em Equipas](https://support.microsoft.com/office/change-your-picture-in-teams-7a711943-9248-420e-b814-c071aa8d9b9c).
