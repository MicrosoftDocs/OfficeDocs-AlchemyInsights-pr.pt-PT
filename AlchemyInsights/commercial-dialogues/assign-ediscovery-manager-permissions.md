# <a name="assign-ediscovery-manager-permissions"></a>Atribuir permissões de gestor de eDiscovery

Para atribuir permissões eDiscovery Manager, execute os seguintes passos:

1. No [Office 365 Security & Compliance Center](https://sip.protection.office.com/), selecione **Permissões > gestor de eDiscovery**.
2. No painel **eDiscovery Manager,** ao lado **do eDiscovery Manager,** clique em **Edit**.
3. No painel **de edição Escolha eDiscovery Manager,** clique em **Escolher o gestor eDiscovery**.
4. No painel **Escolha eDiscovery Manager,** clique **em + Adicionar** e, em seguida, selecione os nomes dos utilizadores.
5. **Selecione adicionar > feito > guardar**.
