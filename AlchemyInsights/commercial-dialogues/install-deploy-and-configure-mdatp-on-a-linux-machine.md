# <a name="install-deploy-and-configure-mdatp-on-a-linux-machine"></a>Instale, implemente e configuure o MDATP numa máquina Linux

1. Certifique-se de que está inscrito no MDATP e tem acesso ao [portal MDATP.](https://go.microsoft.com/fwlink/?linkid=2144512)
2. Utilize o MDATP [manualmente](https://go.microsoft.com/fwlink/?linkid=2144809) ou utilizando uma ferramenta de gestão de configuração: [Marioneta](https://go.microsoft.com/fwlink/?linkid=2144715) [ou Ansible](https://go.microsoft.com/fwlink/?linkid=2144716).

Para saber mais, consulte [Como instalar o MDATP para o Linux.](https://go.microsoft.com/fwlink/?linkid=2144717)
