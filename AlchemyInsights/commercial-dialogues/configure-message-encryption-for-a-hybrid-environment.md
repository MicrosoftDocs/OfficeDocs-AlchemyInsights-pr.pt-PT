# <a name="configure-message-encryption-for-a-hybrid-environment"></a>Configurar encriptação de mensagens para um ambiente híbrido

Para ambientes de Troca híbrida, os utilizadores no local só podem enviar e-mails encriptados usando encriptação de mensagens do Office Message (OME) se o e-mail for encaminhado através do Exchange Online.

Para encriptar e-mails usando OME, execute os seguintes passos:

1. Utilize o [assistente de configuração híbrido](https://docs.microsoft.com/Exchange/hybrid-configuration-wizard) para configurar o seu ambiente híbrido. Não são necessários passos especiais para a configuração da encriptação.
2. [Crimes as tuas regras de fluxo de correio para encriptação](https://docs.microsoft.com/microsoft-365/compliance/define-mail-flow-rules-to-encrypt-email) como normalmente farias.
