# <a name="buy-a-new-domain-from-microsoft"></a>Comprar um novo domínio da Microsoft

Para comprar um novo domínio da Microsoft, execute os seguintes passos:

1. Navegue para **configurar > domínios** e clique em **Comprar domínio**. 
2. Siga as instruções do assistente.
