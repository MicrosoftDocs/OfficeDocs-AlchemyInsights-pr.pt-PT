# <a name="dkim-domainkeys-identified-mail"></a>DKIM (Correio Identificado de DomainKeys)

DomainKeys Identified Mail (DKIM) é um protocolo que permite que uma organização assuma a responsabilidade de transmitir uma mensagem de uma forma que possa ser verificada pelos fornecedores de caixas de correio. Esta verificação é possível através da autenticação criptográfica.
