# <a name="set-up-safe-attachment-policies-in-microsoft-defender-for-office-365"></a>Configurar políticas de anexo seguro no Microsoft Defender para o Office 365

1. Vá ao [Centro de Segurança & Compliance do Office 365](https://go.microsoft.com/fwlink/p/?linkid=2077143) e selecione Anexos Seguros de **Política de Gestão de Ameaças**  >    >  .
2. Para ativar esta funcionalidade para outras cargas de trabalho, selecione **Ligue o Microsoft Defender para o Office 365 para SharePoint Online, OneDrive e Microsoft Teams**.
3. Para criar uma nova política, selecione o ícone **Novo** **+** ()
4. Especifique o nome, descrição e configurações para a apólice.
5. Seleccione **Guardar**.
