# <a name="onboard-a-windows-server-to-microsoft-defender-for-endpoint"></a>A bordo de um servidor do Windows para o Microsoft Defender para o Endpoint

Para o sistema operativo Windows Server, o Microsoft Defender for Endpoint fornece capacidades avançadas para detetar e investigar ataques. Estas capacidades são fornecidas sem problemas através da consola Do Microsoft Defender Security Center.

O Defender for Endpoint suporta as seguintes versões do Windows Server:

- Windows Server 2008 R2 SP1
- Windows Server 2012 R2
- Windows Server 2016
- Versão 1803 e posterior do Windows Server (SAC) e posterior
- Windows Server 2019 e mais tarde
- Edição core do Windows Server 2019

Para saber mais, consulte [os servidores do Windows no Microsoft Defender para o serviço Endpoint](https://go.microsoft.com/fwlink/?linkid=2143627).
