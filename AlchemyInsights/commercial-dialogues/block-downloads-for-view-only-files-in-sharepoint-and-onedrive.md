# <a name="block-downloads-for-view-only-files-in-sharepoint-and-onedrive"></a>Bloquear transferências para ficheiros apenas para visualização no SharePoint e no OneDrive

Quando partilha itens com outros no OneDrive ou No SharePoint, pode dar-lhes acesso à visualização, mas não baixar. Para obter mais informações, consulte [as transferências do Bloco para ficheiros apenas visualizadores no SharePoint e no OneDrive](https://support.microsoft.com/office/block-downloads-for-view-only-files-in-sharepoint-and-onedrive-6051184b-62ac-4149-b874-13dcd40ef91e).
