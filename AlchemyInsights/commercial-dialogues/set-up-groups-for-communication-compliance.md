# <a name="set-up-groups-for-communication-compliance"></a>Criar grupos para a conformidade com a comunicação

Este passo é opcional. Se quiser começar com apenas alguns utilizadores para ver como funciona a conformidade com a comunicação, pode saltar este passo.  
  
Quando cria uma política de conformidade de comunicação, define quem tem as suas comunicações revistas e quem realiza comentários. Você usará endereços de e-mail para identificar indivíduos ou grupos de pessoas. Para simplificar a sua configuração, pode criar grupos para pessoas que tenham a sua comunicação revista e grupos para pessoas que revejam essas comunicações.  
  
Para saber como, consulte [Configurar grupos para a conformidade com a comunicação](https://go.microsoft.com/fwlink/?linkid=2129594).
