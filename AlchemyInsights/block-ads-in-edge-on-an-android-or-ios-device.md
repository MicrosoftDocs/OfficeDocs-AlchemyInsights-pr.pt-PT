# <a name="block-ads-in-the-microsoft-edge-browser-on-an-android-or-ios-device"></a>Bloqueie anúncios no navegador Microsoft Edge num dispositivo Android ou iOS

Para bloquear todos os anúncios, exceto os considerados aceitáveis pelo AdBlock Plus:
1. Selecione **...** > **(Configurações e muito mais)**  >  **Configurações**  >  **Bloqueadores de conteúdos**.
2. Ligue **os anúncios do Bloco**.

Para bloquear todos os anúncios:
1. Selecione **...** > **(Configurações e muito mais)**  >  **Configurações**  >  **Bloqueadores de conteúdos**.
2. Ligue **os anúncios do Bloco**.
3. Selecione **definições avançadas**.
4. Desligue **Deixe os anúncios aceitáveis**.
