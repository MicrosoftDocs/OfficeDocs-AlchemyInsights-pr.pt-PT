# <a name="onedrive-puid-mismatch"></a>Incompatibilidade OneDrive PUID
Parece que você está experimentando uma [incompatibilidade PUID](https://docs.microsoft.com/sharepoint/support/administration/access-denied-or-need-permission-error-sharepoint-online-or-onedrive-for-business#when-accessing-a-onedrive-site) em OneDrive for Business. Por favor, siga os passos abaixo para que possamos ajudá-lo o mais rapidamente possível:

1. Clique no suporte de **contato**e, em seguida, **nova solicitação de serviço.**
2. Deixe o título e a descrição como **incompatibilidade PUID no site da OneDrive.**
3. Uma vez que o bilhete foi enviado, atualize-o com as seguintes informações:
    - Url do site OneDrive que é afetado.
    - Nome principal do usuário (UPN) do usuário afetado.
