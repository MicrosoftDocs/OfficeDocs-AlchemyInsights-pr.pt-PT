# <a name="create-mailboxes-that-dont-require-licenses"></a>Criar caixas de correio que não necessitem de licenças

- Se necessitar de um endereço de correio electrónico de grupo para várias pessoas, consulte [criar uma caixa de correio partilhada](https://support.office.com/article/871a246d-3acd-4bba-948e-5de8be0544c9).

- Para permitir que uma equipa de pessoas trabalham em conjunto e partilhar ficheiros e correio electrónico, consulte para [aprender sobre grupos do Office 365](https://support.office.com/article/b565caa1-5c40-40ef-9915-60fdb2d97fa2).
