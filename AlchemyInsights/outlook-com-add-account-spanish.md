# <a name="agregar-sus-otras-cuentas-de-correo-electrnico"></a>Agregar sus otras cuentas de correo e-mail

Cuando agregue otras cuentas de correo e-mail um Outlook.com, puede usar Outlook.com to enviar y anteporto mensajes de correo e-mail de these cuentas sin salir de Outlook.com. Each número de identificação bancária that se agrega uma Outlook.com se denomina medico número de identificação bancária conectada. También puede [crear un alias de correo messages](https://support.office.com/es-es/article/agregar-o-quitar-un-alias-de-correo-electrónico-en-outlook-com-459b1989-356d-40fa-a689-8f285b13f1f2) that uses la misma Bandeja de entrada, enviar, elements y lista de contactos como su número de identificação bancária de Outlook.com can.
