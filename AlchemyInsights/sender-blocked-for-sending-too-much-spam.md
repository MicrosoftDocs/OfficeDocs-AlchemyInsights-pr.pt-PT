# <a name="account-is-blocked-for-sending-too-much-spam"></a>Bloqueio da conta para o envio de muito spam

A conta de envio poderá ser comprometida. Será necessário:
  
1. [Determinar se a conta for comprometida](https://support.microsoft.com/help/2551603/how-to-determine-whether-your-office-365-account-has-been-compromised). Se comprometida, siga os passos para [responder a uma conta de correio electrónico comprometido no Office 365](https://docs.microsoft.com/office365/securitycompliance/responding-to-a-compromised-email-account).
    
2. Vá para os utilizadores restritos para **[desbloquear a conta](https://protection.office.com/?hash=/restrictedusers)** de segurança &amp; Centro de conformidade. Uma vez concluído, deverá conseguir retome o envio de mensagens, *no prazo de 1 hora* . 
    
3. Para ajudar a evitar futuras conta compromete, siga as recomendações de [procedimentos recomendados de segurança para Office 365](https://support.office.com/article/9295e396-e53d-49b9-ae9b-0b5828cdedc3.aspx).
