# <a name="cannot-add-or-create-a-new-signature-in-outlook"></a>Não é possível adicionar ou criar uma nova assinatura no Outlook

Esta é uma questão conhecida. Estamos a trabalhar para o corrigir e o estado será atualizado no [Outlook congela clicando no botão Signature](https://support.office.com/article/c70b36c2-66ca-401c-ab45-f29a46495d02).

Para contornar este problema, tente primeiro os passos para [desinstalar](https://support.office.com/article/c70b36c2-66ca-401c-ab45-f29a46495d02) as aplicações pré-instaladas do Microsoft Office Desktop Apps. 

Se isso não funcionar para a máquina afetada, por favor experimente a [solução baseada no registo.](https://support.office.com/article/c70b36c2-66ca-401c-ab45-f29a46495d02)
