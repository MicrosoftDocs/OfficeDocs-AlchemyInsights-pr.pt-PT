# <a name="delete-items-or-empty-the-recycle-bin-of-a-sharepoint-site"></a>Eliminar itens ou esvaziar o caixote do reciclo de um site do SharePoint 

Quando elimina um item de uma biblioteca ou lista de documentos do SharePoint, este não é imediatamente removido do SharePoint. Os itens apagados entram no site Reciclar bin por um período de tempo. Durante esse tempo, pode restaurar os itens que apagou à sua localização original, ou removê-los para libertar espaço de armazenamento.

[Eliminar itens ou esvaziar o caixote do reciclo de um site do SharePoint](https://support.office.com/article/2e713599-d13e-40d6-96dc-66f0a366f74e)
