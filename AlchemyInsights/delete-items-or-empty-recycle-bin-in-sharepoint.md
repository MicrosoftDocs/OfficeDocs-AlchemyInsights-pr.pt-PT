# <a name="delete-items-or-empty-the-recycle-bin-of-a-sharepoint-site"></a>Eliminar itens ou esvaziar o Caixote do Lixo de Um Site sharePoint 

Quando elimina um item de uma biblioteca ou lista de documentos SharePoint, este não é imediatamente removido do SharePoint. Os itens eliminados vão para o local Reciclar Bin por um período de tempo. Durante esse tempo, pode restaurar os itens que apagou para a sua localização original, ou removê-los para libertar o espaço de armazenamento.

[Eliminar itens ou esvaziar o Caixote do Lixo de Um Site sharePoint](https://support.office.com/article/2e713599-d13e-40d6-96dc-66f0a366f74e)
