# <a name="get-started-with-microsoft-search"></a>Começar com a Microsoft Search

Utilize o Microsoft Search para encontrar pessoas, ficheiros, gráficos organizacionais, sites, respostas a questões comuns e outros conteúdos que possam ajudá-lo a completar tarefas na aplicação que está a usar.

Para mais informações, consulte [a visão geral do Microsoft Search](https://docs.microsoft.com/microsoftsearch/overview-microsoft-search).
