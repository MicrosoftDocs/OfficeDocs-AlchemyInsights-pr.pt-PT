# <a name="use-microsoft-edge-extensions"></a>Utilize extensões microsoft edge

[As extensões](https://go.microsoft.com/fwlink/?linkid=2135619) são addons que pode instalar no Microsoft Edge para bloquear anúncios, gerir palavras-passe, traduzir texto e personalizar a sua experiência de navegação de muitas outras formas.
