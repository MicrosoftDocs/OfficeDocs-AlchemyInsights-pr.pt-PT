# <a name="your-users-mailbox-is-being-created"></a>A caixa de correio do seu utilizador está a ser criada

Pode demorar até uma hora até que uma nova caixa de correio esteja pronta a ser utilizada.
  
Se a caixa de correio do utilizador estiver a "preparar-se" há mais de uma hora, pode tentar eliminar e recriar o utilizador, ou utilizar a opção "Deixe-nos ligar-lhe" para falar com um engenheiro de suporte.
