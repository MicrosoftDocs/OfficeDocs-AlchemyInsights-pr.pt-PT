# <a name="your-users-mailbox-is-being-created"></a>A caixa de correio do seu utilizador está a ser criada

Pode levar até uma hora até que uma nova caixa de correio esteja pronta a ser usada.
  
Se a caixa de correio do utilizador estiver a "preparar" durante mais de uma hora, pode tentar eliminar e recriar o utilizador, ou utilizar a opção "Deixe-nos ligar-lhe" para falar com um engenheiro de suporte.
