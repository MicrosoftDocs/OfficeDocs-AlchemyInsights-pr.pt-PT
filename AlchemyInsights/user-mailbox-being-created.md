# <a name="your-users-mailbox-is-being-created"></a>Caixa de correio do utilizador está a ser criada

Pode demorar até uma hora antes de uma nova caixa de correio está pronta a utilizar.
  
Se a caixa de correio do utilizador tiver sido "Preparar" para mais de uma hora, pode tentar eliminar e recriar o utilizador ou utilize a opção "diga-nos chamada" para falar com um engenheiro de suporte.
