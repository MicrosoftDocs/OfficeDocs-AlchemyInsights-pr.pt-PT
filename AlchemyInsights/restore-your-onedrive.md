# <a name="restore-your-onedrive"></a>Restaurar o OneDrive

Se muitos dos ficheiros de OneDrive obter eliminados, overwritted, danificados ou infectados por software malicioso, pode restaurar o OneDrive todo para uma hora anterior nos últimos 30 dias.
  
1. Vá para o OneDrive no web browser.
    
2. Clique no ícone de definições no canto superior direito e, em seguida, seleccione **Restaurar o OneDrive**.
    
3. Seleccione uma data a partir da lista ou escolha **personalizado data e hora**.
    
4. Clique em **Restaurar** para anular todas as actividades após a data seleccionada. 
    
[Obter ajuda através do restauro de ficheiros](https://go.microsoft.com/fwlink/?linkid=872874)
