# <a name="onedrive-troubleshoot-processing-changes"></a>OneDrive: Resolução de problemas 'Alterações de processamento'

Se vir "Processing changes" por um longo período de tempo, pode ser devido a ter um ficheiro online aberto, uma série de ficheiros presos numa fila, ou uma sincronização de ficheiros muito grande, ver [OneDrive: Troubleshoot 'Processing changes'](https://support.office.com/article/onedrive-is-stuck-on-processing-changes-b386b813-9b66-4e47-8c4c-2b45533edccd) para ajudar a resolver este problema.

Também verá esta mensagem se tiver um ficheiro que muda a toda a hora (como um Outlook . Ficheiro PST ou base de dados). Saiba como [mover os ficheiros de dados](https://support.office.com/article/how-to-remove-an-outlook-pst-data-file-from-onedrive-b6b9e522-59bd-40f7-949f-168d0aa9b38e) do Outlook que está a utilizar regularmente a partir do OneDrive.
