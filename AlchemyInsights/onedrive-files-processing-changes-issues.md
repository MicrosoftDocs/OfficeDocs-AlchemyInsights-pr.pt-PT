# <a name="onedrive-troubleshoot-processing-changes"></a>OneDrive: solucionar problemas de ' alterações de processamento '

Se você vir "Processando alterações" por um longo período de tempo, pode ser porque você tem um arquivo on-line aberto, muitos arquivos presos em uma fila ou uma sincronização de arquivo muito grande, consulte [onedrive bloqueado no processamento](https://support.office.com/article/onedrive-is-stuck-on-processing-changes-b386b813-9b66-4e47-8c4c-2b45533edccd).

Leia este artigo para solucionar problemas de "processamento de alterações". Você também verá esta mensagem se você tiver um arquivo que muda o tempo todo (como um Outlook. Arquivo PST ou um banco de dados). Saiba como [mover os arquivos de dados do Outlook](https://support.office.com/article/how-to-remove-an-outlook-pst-data-file-from-onedrive-b6b9e522-59bd-40f7-949f-168d0aa9b38e) que você está usando regularmente fora do onedrive.
