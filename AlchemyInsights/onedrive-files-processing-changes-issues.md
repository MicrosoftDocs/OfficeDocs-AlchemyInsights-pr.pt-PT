# <a name="onedrive-troubleshoot-processing-changes"></a>OneDrive: Troubleshoot 'Mudanças de processamento'

Se você ver "Alterações de processamento" por um longo período de tempo, pode ser porque você tem um arquivo on-line aberto, muitos arquivos presos em uma fila ou uma sincronização de arquivos muito grande, veja [o OneDrive preso no processamento.](https://support.office.com/article/onedrive-is-stuck-on-processing-changes-b386b813-9b66-4e47-8c4c-2b45533edccd)

Leia este artigo para solucionar problemas "Alterações de processamento". Você também verá essa mensagem se tiver um arquivo que mude o tempo todo (como um Outlook. Arquivo PST ou um banco de dados). Saiba como mover seus arquivos de [dados do Outlook](https://support.office.com/article/how-to-remove-an-outlook-pst-data-file-from-onedrive-b6b9e522-59bd-40f7-949f-168d0aa9b38e) que você está usando regularmente do OneDrive.
