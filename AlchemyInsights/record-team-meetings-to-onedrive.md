# <a name="teams-meeting-recordings-to-onedrive"></a>Equipas Reunindo Gravações para OneDrive

A Microsoft Teams tem um novo método para guardar as gravações de reuniões. Como primeira fase de uma transição do clássico Microsoft Stream para o [novo Stream](https://docs.microsoft.com/stream/streamnew/new-stream), este método armazena gravações no Microsoft OneDrive para Business e SharePoint na Microsoft 365 e oferece muitos benefícios.  

- [Configurar a opção de gravação de reunião para OneDrive para Business and SharePoint](https://docs.microsoft.com/MicrosoftTeams/tmr-meeting-recording-change#set-up-the-meeting-recording-option-for-onedrive-for-business-and-sharepoint)
- [Opte por sair do OneDrive para negócios e SharePoint para continuar a usar o Stream](https://docs.microsoft.com/MicrosoftTeams/tmr-meeting-recording-change#opt-out-of-onedrive-for-business-and-sharepoint-to-continue-using-stream)  
- [Perguntas frequentes](https://docs.microsoft.com/MicrosoftTeams/tmr-meeting-recording-change#frequently-asked-questions)
