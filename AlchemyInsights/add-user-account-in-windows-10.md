# <a name="add-a-user-account-in-windows-10"></a>Adicionar uma conta de utilizador no Windows 10

Eis como adicionar uma conta no Windows 10:

1. Vá para **iniciar**  >  **definições** de contas Família &  >    >  **outras pessoas** (ou **Outras pessoas**, se estiver a utilizar o Windows 10 Enterprise) > **Adicionar outra pessoa a este PC**.
2. Siga as instruções para adicionar uma conta Microsoft ou conta local.

[Saiba mais sobre a criação de contas](https://support.microsoft.com/help/17197/)
