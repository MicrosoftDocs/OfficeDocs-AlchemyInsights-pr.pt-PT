# <a name="add-a-user-account-in-windows-10"></a>Adicionar uma conta de usuário no Windows 10

Veja como adicionar uma conta no Windows 10:

1. Vá para **iniciar** > **configurações** > **de família de contas** > **& outras pessoas** (ou **outras pessoas,** se você estiver usando o Windows 10 Enterprise) > **Adicione outra pessoa a este PC**.
2. Siga as instruções para adicionar uma conta da Microsoft ou conta local.

[Saiba mais sobre a criação de contas](https://support.microsoft.com/help/17197/)
