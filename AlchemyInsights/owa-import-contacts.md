# <a name="import-contacts"></a>Importar contactos

Antes de importar contactos para o Outlook na web, terá de exportá-los a partir de outro fornecedor de correio electrónico como um ficheiro. csv. Em seguida, siga as instruções abaixo.

1. Na parte inferior da página, seleccione o ícone de pessoas para ir para a página de pessoas.
2. Na barra de ferramentas, seleccione **Gerir** > **Importar contactos**.
3. Seleccione **Procurar**, escolher o ficheiro CSV e, em seguida, clique em **Abrir**.
4. Seleccione **Importar**.

Para obter mais informações, consulte [Importar contactos do Outlook](https://support.office.com/article/bb796340-b58a-46c1-90c7-b549b8f3c5f8#ID0EAACAAA=Outlook_on_the_web).
