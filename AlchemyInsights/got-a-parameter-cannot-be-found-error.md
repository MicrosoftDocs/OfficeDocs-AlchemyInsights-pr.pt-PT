# <a name="migrating-email-from-imap-email-system-to-office-365"></a>Migrar mensagem de correio electrónico do sistema de correio electrónico IMAP Office 365

Para migrar o correio electrónico de um sistema de correio IMAP, siga este processo: [migração de IMAP no Centro de administração de Microsoft 365](https://support.office.com/article/4682f2e4-f720-4868-91ab-207f5b0c325d)
  
Muitas das grandes organizações terá de planear cuidadosamente gerir expectativas dos utilizadores. O artigo de [que necessita de saber sobre a migração das caixas de correio IMAP para o Office 365](https://docs.microsoft.com/Exchange/mailbox-migration/migrating-imap-mailboxes/migrating-imap-mailboxes) ajudará a gerir a alteração.

Verifique também se existem a página de [adoptar o Office 365 com FastTrack](https://www.microsoft.com/fasttrack/microsoft-365/office-365) para Ajuda Assistida para migração, se elegíveis.
