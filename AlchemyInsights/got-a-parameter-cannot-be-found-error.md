# <a name="migrating-email-from-imap-email-system-to-office-365"></a>E-mail migratório do sistema de e-mail imap para o Office 365

Para migrar e-mail de um sistema de correio IMAP, siga este processo: [migração do IMAP no centro de administração do Microsoft 365](https://docs.microsoft.com/Exchange/mailbox-migration/migrating-imap-mailboxes/imap-migration-in-the-admin-center)
  
Muitas grandes organizações precisarão planejar cuidadosamente para gerenciar as expectativas do usuário. O que você precisa saber sobre a migração de suas caixas de correio do IMAP para o artigo [do Office 365](https://docs.microsoft.com/Exchange/mailbox-migration/migrating-imap-mailboxes/migrating-imap-mailboxes) irá ajudá-lo a gerenciar a mudança.

Verifique também o [Adopt Office 365 com](https://www.microsoft.com/fasttrack/microsoft-365/office-365) a página FastTrack para obter ajuda assistida com a migração, se for elegível.
