# <a name="create-a-shared-mailbox"></a>Criar uma caixa de correio partilhada

Uma caixa de correio partilhada permite mais de uma pessoa enviar e receber correio electrónico para um alias, tal como sales@contoso.com, sem atribuir outra licença. 

Consulte o artigo [criar uma caixa de correio partilhada](https://docs.microsoft.com/office365/admin/email/create-a-shared-mailbox) das sugestões para obter mais informações sobre como utilizar uma caixa de correio partilhada.
