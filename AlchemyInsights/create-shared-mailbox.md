# <a name="create-a-shared-mailbox"></a>Criar uma caixa de correio partilhada

Uma caixa de correio partilhada permite mais de uma pessoa enviar e receber correio electrónico para um alias, tal como sales@contoso.com, sem atribuir outra licença. Para criar uma caixa de correio partilhada, vá para o [Guia de correio electrónico partilhado](https://portal.office.com/adminportal/home).
  
Consulte o artigo [criar uma caixa de correio partilhada](https://support.office.com/article/Create-a-shared-mailbox-871a246d-3acd-4bba-948e-5de8be0544c9) das sugestões para obter mais informações sobre como utilizar uma caixa de correio partilhada.
