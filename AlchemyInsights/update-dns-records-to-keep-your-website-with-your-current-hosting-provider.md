# <a name="update-dns-records-to-keep-your-website-with-your-current-hosting-provider"></a>Atualizar os registos DNS para manter o seu Web site junto do seu fornecedor de alojamento atual

Como parte do processo de mudança do seu Website Público Microsoft 365 para outro fornecedor de hospedagem, provavelmente terá de fazer alterações nos seus registos de domínio.
  
1. No centro de administração, vá ao **Setup** \> **Domains** e escolha o nome de domínio que pretende alterar.

2. Informe as informações que se alinham com o seu website com o seu novo fornecedor de hospedagem.

Para saber mais, consulte [os registos do Update DNS para manter o seu website com o seu fornecedor de hospedagem atual.](https://docs.microsoft.com/office365/admin/dns/update-dns-records-to-retain-current-hosting-provider)
  
Para saber mais sobre a descontinuação do site público, consulte os [Sites Públicos Online sharePoint para serem descontinuados](https://support.office.com/article/sharepoint-online-public-websites-to-be-discontinued-e86bfd2f-5c7d-446f-a430-7cfcc0130916).
