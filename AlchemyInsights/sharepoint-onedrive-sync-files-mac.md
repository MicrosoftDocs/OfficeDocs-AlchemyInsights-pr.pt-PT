# <a name="install-setup-and-sync-onedrive-or-sharepoint-files-on-mac"></a>Instalar, configurar e sincronizar arquivos do OneDrive ou do SharePoint no Mac 

Instale, configure e sincronize arquivos do OneDrive ou do SharePoint em um Mac com as instruções dos [arquivos Sync com o cliente de sincronização do onedrive no Mac os X](https://support.office.com/article/sync-files-with-the-onedrive-sync-client-on-mac-os-x-d11b9f29-00bb-4172-be39-997da46f913f).

Se o aplicativo OneDrive estiver instalado e configurado em um Mac, mas os usuários estiverem enfrentando problemas de sincronização, consulte [corrigir problemas de sincronização com o onedrive (Mac)](https://support.office.com/article/fix-onedrive-sync-problems-on-a-mac-af3012d7-13ec-4ac9-bbb1-ebcd2a0cd756).

> [!NOTE]
> A funcionalidade mover pasta conhecida na sincronização do OneDrive para o Windows não está disponível para Mac.
