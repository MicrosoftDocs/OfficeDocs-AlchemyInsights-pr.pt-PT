# <a name="install-setup-and-sync-onedrive-or-sharepoint-files-on-mac"></a>Instalar, configurar e sincronizar ficheiros OneDrive ou SharePoint no Mac 

Instale, configurar e sincronizar ficheiros OneDrive ou SharePoint num Mac com as instruções dos [ficheiros Sync com o cliente sincronizado OneDrive no Mac OS X](https://support.office.com/article/sync-files-with-the-onedrive-sync-client-on-mac-os-x-d11b9f29-00bb-4172-be39-997da46f913f).

Se a aplicação OneDrive for instalada e configurada num Mac, mas os utilizadores estiverem a ter problemas de sincronização, consulte problemas de sincronização fix com o [OneDrive (Mac)](https://support.office.com/article/fix-onedrive-sync-problems-on-a-mac-af3012d7-13ec-4ac9-bbb1-ebcd2a0cd756).

> [!NOTE]
> A função De movimento de pasta conhecida na sincronização OneDrive para Windows não está disponível para Mac.
