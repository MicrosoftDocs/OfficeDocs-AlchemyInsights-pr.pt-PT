# <a name="add-phone-number"></a>Adicionar número de telefone

- Antes de poder configurar utilizadores na sua organização para fazer e receber chamadas telefónicas, tem de obter [números de telefone](https://docs.microsoft.com/MicrosoftTeams/manage-phone-numbers-for-your-organization/) para os mesmos.

- Não tem a certeza sobre os serviços de voz que a Microsoft oferece e quais as funcionalidades que poderá necessitar? Ver [voz cloud nas Equipas da Microsoft](https://docs.microsoft.com/MicrosoftTeams/cloud-voice-landing-page).

- Precisa adicionar [um número de telefone a um convite para uma reunião?](https://docs.microsoft.com/MicrosoftTeams/set-the-phone-numbers-included-on-invites-in-teams)
