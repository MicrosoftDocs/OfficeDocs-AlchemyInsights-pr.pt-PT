# <a name="add-phone-number"></a>Adicionar número de telefone

Antes de poder configurar utilizadores na sua organização para fazer e receber chamadas telefónicas, tem de obter [números de telefone](https://docs.microsoft.com/MicrosoftTeams/phone-number-calling-plans/port-order-overview) para os mesmos. Fornecemos um processo passo a passo para completar a configuração do sistema telefónico. Consulte [Configuração do Sistema Telefónico na sua organização](https://docs.microsoft.com/MicrosoftTeams/phone-number-calling-plans/port-order-overview).
