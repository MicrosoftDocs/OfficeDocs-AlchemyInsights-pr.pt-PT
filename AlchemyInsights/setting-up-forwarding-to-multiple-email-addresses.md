# <a name="setting-up-forwarding-to-multiple-email-addresses"></a>Configuração do encaminhamento para vários endereços de e-mail

Se precisar de configurar o encaminhamento de e-mails para um grupo de utilizadores, criar uma lista de distribuição, adicionar vários utilizadores à lista e, em seguida, configurar o encaminhamento para essa lista. Primeiro, no centro de administração, navegue para **Groups**  >  [grupos de grupos.](https://portal.office.com/adminportal/home#/groups)
  
1. Criar um grupo.

2. Adicione vários utilizadores.

3. Configure a caixa de correio reencaminhando para aquele grupo.
