# <a name="setting-up-forwarding-to-multiple-email-addresses"></a>Configuração de encaminhamento para vários endereços de e-mail

Se precisar de configurar o reencaminhamento de e-mail para um grupo de utilizadores, criar uma lista de distribuição, adicionar vários utilizadores e, em seguida, configurar o reencaminhamento para essa lista. Primeiro, no centro de administração, navegar para **grupos** > [de grupos.](https://portal.office.com/adminportal/home#/groups)
  
1. Criar um grupo.

2. Adicione vários utilizadores.

3. Configure a caixa de correio reencaminhamento para aquele grupo.
