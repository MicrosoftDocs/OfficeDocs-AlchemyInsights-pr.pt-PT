# <a name="how-to-create-a-rule-in-outlook-for-mac"></a>Como criar uma regra no Outlook for Mac

1. Para criar uma pasta para conter as mensagens de correio eletrónico enviadas pela regra, selecione a caixa de correio raiz, no separador **Organizar,** clique em **Nova Pasta,** digite o nome da pasta e prima Enter.

2. Na lista de mensagens, clique numa mensagem de e-mail para a qual deseja criar uma regra.

3. No separador **'Casa',** clique em **Regras**e, em seguida, clique em **Criar Regra**.

4. Em **Quando uma nova mensagem chega** edita os critérios para atender às suas necessidades. 

Para obter instruções detalhadas sobre a criação de regras personalizadas, consulte [Criar uma regra no Outlook for Mac](https://aka.ms/AA1uy0v)
