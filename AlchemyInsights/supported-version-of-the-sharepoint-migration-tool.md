# <a name="supported-version-of-the-sharepoint-migration-tool"></a>Versão suportada da ferramenta de migração sharepoint



Atualmente, a Ferramenta de Migração SharePoint só suporta migrações do SharePoint 2013. As versões 2010 e 2016 não são suportadas neste momento.
  
Para mais informações, [consulte a introdução da ferramenta de migração sharepoint.](https://go.microsoft.com/fwlink/?linkid=2044765&amp;clcid=0x409)
