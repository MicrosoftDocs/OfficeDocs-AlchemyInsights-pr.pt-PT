# <a name="supported-version-of-the-sharepoint-migration-tool"></a>Versão suportada da Ferramenta de Migração SharePoint



Atualmente, a Ferramenta de Migração SharePoint apenas suporta migrações do SharePoint 2013. As versões 2010 e 2016 não são suportadas neste momento.
  
Para obter mais informações, consulte [a introdução da Ferramenta de Migração SharePoint](https://go.microsoft.com/fwlink/?linkid=2044765&amp;clcid=0x409).
