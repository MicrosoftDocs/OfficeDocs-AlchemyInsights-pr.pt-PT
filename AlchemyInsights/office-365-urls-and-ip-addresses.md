# <a name="microsoft-365-urls-and-ip-address-ranges"></a>Gamas de endereços Microsoft 365 URLs e IP

O Microsoft 365 requer conectividade com a Internet através de endereços IP específicos chamados *pontos finais*.
Para obter uma lista de pontos finais que precisam de ser alcançáveis, consulte [os intervalos de URLs e endereços IP da Microsoft 365](https://docs.microsoft.com/office365/enterprise/urls-and-ip-address-ranges) 

O serviço web Microsoft 365 IP Address e URL ajuda-o a identificar e diferenciar melhor o tráfego de rede microsoft 365, facilitando a avaliação, configuração e permanência atualizado com as alterações. Consulte [o Microsoft 365 IP Address e o serviço web URL](https://docs.microsoft.com/office365/enterprise/office-365-ip-web-service).
