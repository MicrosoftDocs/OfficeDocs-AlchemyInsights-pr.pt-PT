# <a name="microsoft-365-urls-and-ip-address-ranges"></a>Microsoft 365 URLs e gamas de endereços IP

O Microsoft 365 requer conectividade com a Internet através de endereços IP específicos *chamados pontos finais*.
Para obter uma lista de pontos finais que precisam de ser acessíveis, consulte os [intervalos de endereços Microsoft 365 URLs e IP](https://docs.microsoft.com/office365/enterprise/urls-and-ip-address-ranges) 

O serviço web Microsoft 365 IP Address e URL ajuda-o a identificar e diferenciar melhor o tráfego de rede do Microsoft 365, facilitando a avaliação, configuração e permanência a par das alterações. Consulte o [Microsoft 365 IP Address e o serviço web URL](https://docs.microsoft.com/office365/enterprise/office-365-ip-web-service).
