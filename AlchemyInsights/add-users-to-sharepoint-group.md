# <a name="add-users-to-a-sharepoint-group"></a>Adicione utilizadores a um grupo SharePoint

Um grupo SharePoint é uma coleção de utilizadores que todos têm o mesmo conjunto de permissões para sites e conteúdos. Em vez de atribuir permissões uma pessoa de cada vez, pode usar grupos para atribuir convenientemente o mesmo nível de permissão a muitas pessoas ao mesmo tempo.

- [Adicionar utilizadores a um grupo](https://docs.microsoft.com/sharepoint/customize-sharepoint-site-permissions#add-users-to-a-group)
- [Adicione ou remova membros dos Grupos Microsoft 365 utilizando o centro de administração](https://docs.microsoft.com/office365/admin/create-groups/add-or-remove-members-from-groups?view=o365-worldwide)
