# <a name="unable-to-print"></a>Incapaz de imprimir

Estamos cientes de um problema em que alguns utilizadores não podem imprimir quando usam navegadores Safari ou Edge. Enquanto trabalhamos numa correção para isso, por favor use outro navegador. Pedimos desculpa pelo inconveniente.
