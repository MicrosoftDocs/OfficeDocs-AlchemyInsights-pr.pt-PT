Como administrador, tem a capacidade de bloquear o acesso ao Exchange Online PowerShell para os utilizadores. Para mais informações, consulte os seguintes tópicos:

- [Activar ou desactivar o acesso ao Exchange Online PowerShell](https://docs.microsoft.com/powershell/exchange/exchange-online/disable-access-to-exchange-online-powershell)

- [Acesso de cliente regras no Exchange Online](https://technet.microsoft.com/library/mt842508.aspx) (procure o protocolo de **RemotePowerShell** ) 

**Nota**: tal como descrito no tópico, tenha cuidado para não bloquear o acesso ao Exchange Online PowerShell para admins.
