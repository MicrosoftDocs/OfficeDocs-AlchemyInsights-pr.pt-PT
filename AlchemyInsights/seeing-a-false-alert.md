# <a name="seeing-a-false-alert"></a>Ver um alerta falso

- Envie um ficheiro à Microsoft para análise. Para obter mais informações, consulte [o que fazer com falsos positivos/negativos no Microsoft Defender Antivirus](https://docs.microsoft.com/windows/security/threat-protection/microsoft-defender-antivirus/antivirus-false-positives-negatives#submit-a-file-to-microsoft-for-analysis).
- Crie um indicador "Permitir" para evitar um alerta falso positivo. Para obter mais informações, consulte [criar indicadores para ficheiros](https://docs.microsoft.com/windows/security/threat-protection/microsoft-defender-atp/indicator-file).  
- Configure as exclusões no dispositivo para evitar que um item seja digitalizado. Para obter mais informações, consulte [Definir uma exclusão num dispositivo Windows individual para evitar que um item seja digitalizado.](https://docs.microsoft.com/windows/security/threat-protection/microsoft-defender-antivirus/antivirus-false-positives-negatives#define-an-exclusion-on-an-individual-windows-device-to-prevent-an-item-from-being-scanned)
