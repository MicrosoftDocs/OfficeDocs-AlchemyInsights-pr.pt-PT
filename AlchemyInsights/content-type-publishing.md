# <a name="content-type-publishing"></a>Publicação do tipo de conteúdo

Se quiser que os tipos de conteúdo estejam disponíveis em toda a sua organização para todos os utilizadores, é um bom candidato publicar em todas as coleções do site. Para publicar um tipo de conteúdo, consulte [publicar um tipo de conteúdo a partir de um centro de publicação de conteúdos](https://support.office.com/article/publish-a-content-type-from-a-content-publishing-hub-58081155-118d-4e7a-9cc5-d43b5dbb7d02).
