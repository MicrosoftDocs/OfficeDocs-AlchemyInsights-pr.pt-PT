# <a name="troubleshooting-refresh-scenarios---tile-errors"></a>Cenários de atualização de resolução de problemas - Erros de azulejos

Se sentir erros de atualização de azulejos dentro do serviço Power BI, aguarde cerca de 10-15 minutos para que a atualização seja refletida nos azulejos do painel de instrumentos. Se ainda não estiver a aparecer, re-fixar a visualização no painel de instrumentos.

Para mais informações: [https://docs.microsoft.com/power-bi/refresh-troubleshooting-refresh-scenarios](https://docs.microsoft.com/power-bi/refresh-troubleshooting-refresh-scenarios)
