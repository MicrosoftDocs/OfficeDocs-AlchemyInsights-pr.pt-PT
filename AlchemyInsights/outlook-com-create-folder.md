# <a name="create-a-folder-in-outlookcom"></a>Criar uma pasta em Outlook.com

1. Na parte inferior do painel da pasta, selecione **Nova pasta**. Pode ser necessário deslocar-se para baixo no painel de pastas para ver o botão **new folder.**
2. Na caixa de texto, introduza um nome para a pasta e, em seguida, prima Enter.

Para saber mais, consulte [Criar uma pasta em Outlook.com](https://support.office.com/article/6bb0723a-f39f-4a8d-bb3f-fab5dcc2510a?wt.mc_id=Office_Outlook_com_Alchemy).
