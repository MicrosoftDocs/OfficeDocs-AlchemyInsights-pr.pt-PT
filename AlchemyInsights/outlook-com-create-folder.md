# <a name="create-a-folder-in-outlookcom"></a>Criar uma pasta Outlook.com

1. Na parte inferior do painel de pastas, seleccione **nova pasta**. Poderá ter de deslocar para baixo no painel de pastas para ver o botão ' **nova pasta** .
2. Na caixa de texto, introduza um nome para a pasta e, em seguida, prima Enter.

Para obter mais informações, consulte [criar uma pasta no Outlook.com](https://support.office.com/article/6bb0723a-f39f-4a8d-bb3f-fab5dcc2510a?wt.mc_id=Office_Outlook_com_Alchemy).
