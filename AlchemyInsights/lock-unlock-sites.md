# <a name="lock-and-unlock-sites"></a>Bloquear e desbloquear sites

Enquanto administrador Microsoft 365 ou administrador do SharePoint, pode bloquear o acesso a um site ou tornar um site só de leitura ao utilizar o Microsoft PowerShell para alterar o estado de bloqueio do site. Para obter mais informações, consulte [Bloquear e desbloquear sites.](/sharepoint/manage-lock-status)
