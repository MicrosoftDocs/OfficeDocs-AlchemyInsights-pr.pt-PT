# <a name="access-services"></a>Serviços do Access

A funcionalidade De Serviços de Acesso no SharePoint Online está a ser reformada. As novas aplicações web baseadas no Acesso e bases de dados web já não podem ser criadas, e as aplicações web existentes e bases de dados da Web serão encerradas até abril de 2018. As bases de dados de acesso desktop (.accdb) não são impactadas por esta alteração. Recomendamos o Microsoft PowerApps como uma alternativa ao Access web apps e bases de dados web. 
  
[Conheça os Microsoft PowerApps](https://powerapps.microsoft.com/)
