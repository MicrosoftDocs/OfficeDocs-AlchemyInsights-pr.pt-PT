# <a name="getting-an-account-disabled-error-in-outlook-on-the-web"></a>Obter um erro de desativado de conta no Outlook na web?

Se tiver o erro A sua `X-OWA-Error: Microsoft.Exchange.Data.Storage.AccountDisabledException` conta foi **desativada** quando tenta abrir o Outlook na web (anteriormente conhecida como Outlook Web App), a sua administração da organização pode ter desativado o seu acesso ao Outlook na web. A sua administração pode corrigir este erro seguindo os passos do tópico [Enable ou desativar](https://technet.microsoft.com/library/bb124124%28v=exchg.150%29.aspx)o Outlook Web App para uma caixa de correio .
