# <a name="submit-microsoft-cloud-for-healthcare-support-requests"></a>Submeta o Microsoft Cloud para pedidos de suporte a cuidados de saúde

O cliente Microsoft Cloud for Healthcare submete o seu bilhete através do novo [Centro de Administração](https://admin.powerplatform.microsoft.com/support?newTicket&product=Flow) da plataforma Power.

O novo portal fornece um caminho mais direto para o suporte assistido que está otimizado para o Microsoft Cloud for Healthcare.
