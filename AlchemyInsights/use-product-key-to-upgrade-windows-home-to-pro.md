# <a name="use-a-product-key-to-upgrade-windows-10-home-to-windows-10-pro"></a>Utilize uma chave de produto para atualizar o Windows 10 Home para o Windows 10 Pro

Upgrade do Windows 10 Home para o Windows 10 Pro utilizando uma chave de produto:

1. Selecione o botão **Iniciar** e, em seguida, selecione **Atualização de Definições**  >  **&**  >  **ativação de** segurança .

1. Selecione **Alterar a tecla do produto** e, em seguida, introduza a tecla de produto Windows 10 Pro de 25 caracteres.

1. Selecione **Next** para iniciar a atualização para o Windows 10 Pro.

Se já instalou e ativou o Windows 10 Pro no seu dispositivo, tem uma licença digital que lhe permite fazer o upgrade do Windows 10 Home para o Windows 10 Pro sem uma chave de produto:

1. Para atualizar, selecione o botão **Iniciar** e, em seguida, selecione **'Atualização de Definições'**  >  **&**  >  **ativação de** segurança :

    - Se tiver uma licença digital para o Windows 10 Pro e o Windows 10 Home estiver atualmente ativado no seu dispositivo, selecione **Ir para a Microsoft Store** e será solicitado para fazer o upgrade para o Windows 10 Pro gratuitamente.
    - Se tiver uma licença digital para o Windows 10 Pro e o Windows 10 Home não estiver ativado no seu dispositivo, selecione **Troubleshoot**– e siga os passos para atualizar para o Windows 10 Pro.

**Nota:** Para saber mais sobre a ativação no Windows 10, consulte [Ativar o Windows 10](https://support.microsoft.com/windows/activate-windows-10-c39005d4-95ee-b91e-b399-2820fda32227).
