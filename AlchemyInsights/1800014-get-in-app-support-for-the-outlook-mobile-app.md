# <a name="get-in-app-support-for-the-outlook-mobile-app"></a>Obter suporte na aplicação para a aplicação móvel do Outlook

1. Na aplicação móvel do Outlook, abra o **Menu**![de Menu do botão](media/265b9089-9630-42dd-a244-d9a412d8fe47.png).
    
2. Seleccione o ponto de interrogação![botão ajuda a](media/3b8cbf5a-6ced-4d79-b53c-fa82045c3e25.png) para abrir o **Ajuda &amp; comentários** ecrã. 
    
3. Toque em **Contactar o suporte** e explique o problema. 
    
4. Toque **Enviar** e vai receber uma notificação em-app quando contacta um agente de suporte. 
    
Para obter instruções detalhadas, consulte [obter ajuda na aplicação para o Outlook para iOS e Android](https://support.office.com/article/https://support.office.com/article/218a22d1-9fa5-4889-b689-de1c63493243.aspx#ID0EAABAAA=Contact_Support).
