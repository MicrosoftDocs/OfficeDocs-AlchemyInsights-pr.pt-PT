# <a name="read-receipts"></a>Recibos de leitura

Não é possível pedir um recibo de leitura no Outlook.com, mas podem pedir recibos de leitura para mensagens de Outlook.com enviadas utilizando o Outlook para Windows. Para obter informações sobre como proceder, consulte [Adicionar e ler recibos e notificações de entrega do pedido](https://support.office.com/article/a34bf70a-4c2c-4461-b2a1-12e4a7a92141?wt.mc_id=Office_Outlook_com_Alchemy).
  
Para escolher a forma como o Outlook.com responde a pedidos de recibo de leitura, vá para [definições de recibo de leitura](https://outlook.live.com/mail/options/mail/handling/readReceipts).
