# <a name="error-the-rules-on-this-computer-do-not-match"></a>Erro: As regras deste computador não correspondem

Este é um problema conhecido com o Outlook Desktop. O Outlook Team implementou uma correção inicial na versão 12228.20332 que foi lançada em 12/3/2019. Esta correção não abordou todos os relatórios que recebemos.

Para ver o estado atualizado deste problema conhecido, consulte [as regras deste computador não corresponderem às regras da Microsoft Exchange](https://support.office.com/article/d032e037-b224-429e-b325-633afde9b5f0)
