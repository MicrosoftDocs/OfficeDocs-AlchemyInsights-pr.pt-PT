# <a name="moving-a-folder-in-outlookcom"></a>Mover uma pasta na Outlook.com

Pode arrastar uma pasta Outlook.com para uma nova mancha na hierarquia de Favoritos, ou pode utilizar a opção **Mover** no classic Outlook.com menu de atalho. 
  
Pode reordenar as pastas de **Favoritos**, mas não é possível reordenar ou criar uma pasta de nível superior de uma subpasta na versão beta do Outlook.com. Se acidentalmente criar uma subpasta, ou mover uma pasta de nível superior para que se torne uma subpasta, crie uma nova pasta e copie o correio electrónico da subpasta para a nova pasta de nível superior. 
  
Mais informações sobre como [mover ou mudar o nome de uma pasta no Outlook.com](https://support.office.com/article/c9c66fed-8a7c-426a-afc6-0d46a72080fb).
