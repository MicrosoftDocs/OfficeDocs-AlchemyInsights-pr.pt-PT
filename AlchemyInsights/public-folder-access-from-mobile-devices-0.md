# <a name="public-folder-access-from-mobile-devices"></a>Acesso a pastas públicas de dispositivos móveis

As pastas públicas não são acessíveis a partir de dispositivos móveis.
  
Se pretender ter funcionalidades semelhantes às pastas públicas e acedê-lo a partir de dispositivos móveis, consulte para [aprender sobre grupos do Office 365](https://support.office.com/article/learn-about-office-365-groups-b565caa1-5c40-40ef-9915-60fdb2d97fa2) para uma melhor alternativa.
