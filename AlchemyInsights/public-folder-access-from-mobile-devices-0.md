# <a name="public-folder-access-from-mobile-devices"></a>Acesso a pasta pública a partir de dispositivos móveis

As pastas públicas não são acessíveis a partir de dispositivos móveis.
  
Se pretender ter funcionalidades semelhantes às pastas públicas e acessá-la a partir de dispositivos móveis, consulte o Learn sobre o [Microsoft 365 Groups](https://support.office.com/article/learn-about-office-365-groups-b565caa1-5c40-40ef-9915-60fdb2d97fa2) para obter uma melhor alternativa.
