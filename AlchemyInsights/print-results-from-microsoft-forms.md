# <a name="print-results-in-a-summary-of-responses"></a>Resultados de impressão de um resumo das respostas

Pode [Imprimir os resultados de formulários do Microsoft](https://support.office.com/article/print-a-form-22100b98-ba3c-41c1-9513-f76caca664fc) como um resumo das respostas ou para um indivíduo. 

Não é possível imprimir formulários sem respostas, mas pode pedir ou votar esta funcionalidade, visitando o [site da Microsoft UserVoice de formulários](https://microsoftforms.uservoice.com/forums/386451-welcome-to-microsoft-forms-suggestion-box).
