# <a name="onedrive-and-pst-files"></a>Ficheiros OneDrive e .pst 

O OneDrive suporta ficheiros Outlook .pst, no entanto, são sincronizados com menos frequência em comparação com outros tipos de ficheiros para reduzir o tráfego de rede. Para evitar que a aplicação de sincronização OneDrive carrene ficheiros .pst, consulte [a sincronização do Bloco de tipos de ficheiros específicos](https://docs.microsoft.com/onedrive/block-file-types). 

Para retirar ficheiros .pst do OneDrive, consulte [como remover um ficheiro de dados Outlook .pst do OneDrive](https://support.microsoft.com/office/how-to-remove-an-outlook-pst-data-file-from-onedrive-b6b9e522-59bd-40f7-949f-168d0aa9b38e). 

Se os utilizadores ativarem a cópia de segurança manual da pasta pc sem a política de grupo, ocorre um erro se tiverem um ficheiro .pst numa das pastas conhecidas.

Se utilizar o OneDrive para trabalho ou escola, alguns tipos de ficheiros podem ser bloqueados pela administração do site SharePoint da organização.
