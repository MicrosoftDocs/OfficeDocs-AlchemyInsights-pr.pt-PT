# <a name="workflows-in-sharepoint"></a>Fluxos de trabalho no SharePoint

Os fluxos de trabalho do SharePoint são mini-aplicações pré-programadas que simplificam e automatizam uma grande variedade de processos de negócios. Os fluxos de trabalho podem variar desde a coleta de assinaturas, feedback ou aprovações para um plano ou documento até o rastreamento do status atual de um procedimento de rotina.

**Começar:** Se você está apenas começando com fluxos de trabalho sharepoint, consulte [introdução para fluxos de trabalho SharePoint para obter mais informações](https://support.office.com/article/introduction-to-sharepoint-workflow-07982276-54e8-4e17-8699-5056eff4d9e3).

**Microsoft Flow:** Considere usar o Microsoft Flow em vez das soluções clássicas de fluxo de trabalho sharepoint. Para mais informações, [consulte criar um fluxo para uma lista ou biblioteca no SharePoint Online ou no OneDrive](https://support.office.com/article/create-a-flow-for-a-list-or-library-in-sharepoint-online-or-onedrive-for-business-a9c3e03b-0654-46af-a254-20252e580d01)

Fluxo de **trabalho solução de problemas e depuração:**

Os fluxos de trabalho de depuração assemelham-se aos das versões anteriores. Para mais informações, consulte os fluxos de [trabalho do Debugging SharePoint.](https://docs.microsoft.com/sharepoint/dev/general-development/debugging-sharepoint-server-workflows) Para mais informações sobre solução de problemas, consulte erros de validação de fluxo de trabalho do [Servidor do Troubleshoot SharePoint.](https://docs.microsoft.com/sharepoint/dev/general-development/troubleshooting-sharepoint-server-workflow-validation-errors-in-visio)
