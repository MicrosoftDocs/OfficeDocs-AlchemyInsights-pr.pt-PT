# <a name="create-users-in-hybrid-deployments"></a>Criar utilizadores em implementações híbridas

Quando a sua organização é uma implantação híbrida entre um ambiente no local e o Microsoft 365, precisa de criar todos os objetos de utilizador no seu Diretório Ativo no local. Cria os objetos de utilizador na unidade organizacional apropriada (OU) para que sejam sincronizados na sua organização.
