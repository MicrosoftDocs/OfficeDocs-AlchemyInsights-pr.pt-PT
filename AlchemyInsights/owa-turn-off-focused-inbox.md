# <a name="turn-off-focused-inbox"></a>Desactivar direccionadas para a pasta a receber

1. Na parte superior da página, seleccione o ícone de engrenagem para abrir o painel de definições.
2. No painel Definições, mova a alternar **Direccionadas para a pasta a receber** como desactivado.

Para mais informações, consulte [Desactivar direccionadas para a pasta a receber](https://support.office.com/article/f714d94d-9e63-4217-9ccb-6cb2986aa1b2#bkmk_outlookonweb).
