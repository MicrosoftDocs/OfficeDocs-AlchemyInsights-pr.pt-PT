# <a name="change-display-language-settings-in-sharepoint-or-onedrive"></a>Alterar as configurações de linguagem de exibição no SharePoint ou no OneDrive 

Você não pode alterar as configurações de idioma padrão para SharePoint ou OneDrive porque essas configurações foram definidas quando você criou sua assinatura do Office 365. No entanto, pode alterar as configurações do idioma de exibição.

Para alterar suas configurações pessoais, veja [alterar suas configurações pessoais](https://support.office.com/article/Change-your-personal-language-and-region-settings-caa1fccc-bcdb-42f3-9e5b-45957647ffd7)de idioma s.

Para alterar a interface do usuário do site, veja [escolha os idiomas que deseja disponibilizar para](https://support.office.com/article/choose-the-languages-you-want-to-make-available-for-a-site-s-user-interface-16d3a83c-05ab-4b50-8fbb-ff576a3351e8)a interface do usuário de um site.

Para obter mais informações sobre as configurações de linguagem de solução de problemas, consulte que não pode alterar o idioma padrão para uma coleção de [sites on-line do SharePoint.](https://docs.microsoft.com/sharepoint/support/sites/change-default-language-site-collection)
