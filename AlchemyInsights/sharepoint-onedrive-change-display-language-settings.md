# <a name="change-display-language-settings-in-sharepoint-or-onedrive"></a>Alterar as definições de idioma de exibição no SharePoint ou no OneDrive 

Não é possível alterar as definições de idioma padrão para SharePoint ou OneDrive porque essas definições foram definidas quando criou a subscrição do Microsoft 365. No entanto, pode alterar as definições do seu idioma de exibição.

Para alterar as suas definições pessoais, consulte  [alterar as definições de idioma pessoal e região](https://support.office.com/article/Change-your-personal-language-and-region-settings-caa1fccc-bcdb-42f3-9e5b-45957647ffd7).

Para alterar a interface de utilizador do site, consulte [Escolha os idiomas que pretende disponibilizar para a interface de utilizador de um site.](https://support.office.com/article/choose-the-languages-you-want-to-make-available-for-a-site-s-user-interface-16d3a83c-05ab-4b50-8fbb-ff576a3351e8)

Para obter mais informações sobre definições de linguagem de resolução de problemas, consulte [Não pode alterar o idioma predefinido para uma coleção de sites SharePoint Online](https://docs.microsoft.com/sharepoint/support/sites/change-default-language-site-collection).
