# <a name="change-display-language-settings-in-sharepoint-or-onedrive"></a>Alterar definições de idioma de visualização no SharePoint ou OneDrive 

<p>Não é possível alterar as<strong> definições de idioma predefinidas</strong> para SharePoint ou OneDrive, porque essas definições foram definidas quando criou a subscrição do Office 365. &nbsp;No entanto, pode alterar as definições de idioma a <strong>apresentar</strong> .&nbsp;</p> <ol> <li><span style="display: inline !important; float: none; cursor: text; font-family: Verdana,Arial,Helvetica,sans-serif; font-size: 12px; font-style: normal; font-variant: normal; font-weight: 400; letter-spacing: normal; orphans: 2; text-align: left; text-decoration: none; text-indent: 0px; text-transform: none; -webkit-text-stroke-width: 0px; white-space: normal; word-spacing: 0px;">Para alterar as suas definições pessoais</span>, consulte <a title="alterar as definições de idioma e região pessoais" href="https://support.office.com/en-us/article/Change-your-personal-language-and-region-settings-caa1fccc-bcdb-42f3-9e5b-45957647ffd7">Alterar as definições de idioma e região pessoais</a>.&nbsp;<br /><br /></li> <li>Para alterar a interface de utilizador do site, consulte <a title="escolha os idiomas que pretende disponibilizar para interface de utilizador de um site" href="https://support.office.com/en-us/article/choose-the-languages-you-want-to-make-available-for-a-site-s-user-interface-16d3a83c-05ab-4b50-8fbb-ff576a3351e8">Escolha os idiomas que pretende disponibilizar para interface de utilizador de um site</a>. Para mais informações sobre resolução de problemas de definições de idioma, consulte <a title="não é possível alterar o idioma predefinido para uma colecção de sites SharePoint Online" href="https://support.office.com/en-us/article/you-can-t-change-the-default-language-for-a-sharepoint-online-site-collection-40dda07e-6b41-49e9-9828-8805dcb92964">Não é possível alterar o idioma predefinido para uma colecção de sites SharePoint Online</a>.</li> </ol> <p>&nbsp;</p>
