# <a name="make-an-email-alias-the-primary-address-for-a-user"></a>Faça um e-mail pseudónimo o endereço principal para um utilizador

Para obter instruções, consulte [Adicionar outro pseudónimo de e-mail para um utilizador](https://docs.microsoft.com/office365/admin/email/add-another-email-alias-for-a-user).

**Nota:** Se a sua organização estiver em modo Híbrido, terá de alterar o endereço de e-mail primário na organização de intercâmbio no local. Para mais informações, consulte Adicionar ou remover endereços de [e-mail para uma caixa de correio](https://technet.microsoft.com/library/bb123794.aspx).
