# <a name="make-an-email-alias-the-primary-address-for-a-user"></a>Faça um pseudónimo de e-mail o endereço principal de um utilizador

Para obter instruções, consulte [Adicionar outro pseudónimo de e-mail para um utilizador](https://docs.microsoft.com/microsoft-365/admin/email/add-another-email-alias-for-a-user).

**Nota:** Se a sua organização estiver no modo Híbrido, tem de alterar o endereço de e-mail primário na sua organização de troca no local. Para mais informações, consulte [Adicionar ou remover endereços de e-mail para uma caixa de correio.](https://technet.microsoft.com/library/bb123794.aspx)
