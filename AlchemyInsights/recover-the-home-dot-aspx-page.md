# <a name="recover-the-homeaspx-page"></a>Recuperar a página Home.aspx

Se a página inicial do site for eliminada, pode ser recuperada do Recipiente de Reciclagem acedendo ao URL do Reciclo.

1. Navegue para https:// nome de \< inquilino>.sharepoint.com/sites/ \< sitename>/_layouts/15/RecycleBin.aspx?, substituindo <nome de **inquilino**> com o nome do seu inquilino e <nome de **site**> com o nome do seu site.

2. Selecione a página para restaurar.

Se a página inicial ainda não puder ser encontrada, pode criar uma nova página do site e [defini-la como página inicial do seu site](https://support.microsoft.com/en-gb/office/use-a-different-page-for-your-sharepoint-site-home-page-35a5022c-f84a-455d-985e-c691ab5dfa17?ui=en-us&rs=en-gb&ad=gb). O erro não encontrado 404 não deve mais aparecer quando tentar aceder ao site.
