# <a name="change-your-profile-name"></a>Alterar o nome do perfil

1. Vá para [o Microsoft informações de conta](https://go.microsoft.com/fwlink/p/?linkid=860841).
    
2. Seleccione **Editar nome**. 
    
3. Introduza um nome próprio e apelido e seleccione **Guardar**. 
    
Cerca de 24 horas para que as alterações para o nome do perfil sejam implementadas.
  
Para obter mais informações, consulte [alterar o nome de Outlook.com ou fotografias de perfil](https://go.microsoft.com/fwlink/?linkid=873110).
