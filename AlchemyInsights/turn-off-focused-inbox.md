# <a name="turn-off-focused-inbox"></a>Desactivar direccionadas para a pasta a receber

1. Seleccione **as definições de** \> **direccionadas para a pasta a receber**.  

2. Em **quando é recebida a mensagem de correio electrónico**, seleccione **não ordenar as mensagens**.

3. Seleccione **' OK '**.

Para obter mais informações, consulte [Direccionadas para a pasta a receber do Outlook](https://go.microsoft.com/fwlink/p/?linkid=873108).
