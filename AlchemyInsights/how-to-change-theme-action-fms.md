# <a name="how-to-change-your-theme"></a>Como alterar o seu tema

Para alterar o seu tema em Outlook.com ou Outlook na web, vá a **Definições** e escolha um tema sob **Tema**. Para ver todos os temas disponíveis, **selecione Ver tudo** no **Tema**.
