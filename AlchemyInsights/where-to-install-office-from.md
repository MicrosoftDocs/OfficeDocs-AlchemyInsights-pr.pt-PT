# <a name="install-office"></a>Instalar o Office

Para instalar o Office, os utilizadores devem ter um [Microsoft 365 para produto empresarial](https://support.office.com/article/f8ab5e25-bf3f-4a47-b264-174b1ee925fd?wt.mc_id=Alchemy_ClientDIA) que inclua as aplicações para desktop e uma [licença atribuída.](https://docs.microsoft.com/microsoft-365/admin/add-users/add-users)
  
O Office pode então ser instalado seguindo os passos abaixo:
  
1. Vá para [portal.office.com/OLS/MySoftware.aspx](https://portal.office.com/OLS/MySoftware.aspx)

2. Na página inicial do Office, selecione o idioma que deseja e, em seguida, selecione **Advanced** para escolher entre a versão de 64 bits ou 32 bits.

3. Selecione **o Gabinete de Instalação**.

Para obter mais detalhes, consulte [Download e instale ou reinstale o Office num PC ou Mac.](https://support.office.com/article/4414eaaf-0478-48be-9c42-23adc4716658?wt.mc_id=Alchemy_ClientDIA)
  
Para obter instruções específicas sobre Visio ou Project, consulte [o Visio](https://support.office.com/article/f98f21e3-aa02-4827-9167-ddab5b025710) ou [o Projeto de Instalação.](https://support.office.com/article/7059249b-d9fe-4d61-ab96-5c5bf435f281)
