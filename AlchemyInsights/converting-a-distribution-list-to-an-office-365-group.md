# <a name="convert-a-distribution-group-to-a-microsoft-365-group"></a>Converter um grupo de distribuição para um Grupo Microsoft 365

Para converter um grupo ou resolver problemas de conversão, consulte [as listas de distribuição de upgrade para o Microsoft 365 Groups in Outlook](https://docs.microsoft.com/microsoft-365/admin/manage/upgrade-distribution-lists).
