# <a name="convert-a-distribution-group-to-a-microsoft-365-group"></a>Converter um grupo de distribuição num Grupo do Microsoft 365

Pode converter grupos de distribuição em Grupos do Microsoft 365 se satisfazerem determinados critérios. Para converter um grupo ou remoção de problemas de conversão, consulte Atualizar listas de distribuição para grupos do [Microsoft 365 no Outlook.](https://docs.microsoft.com/microsoft-365/admin/manage/upgrade-distribution-lists)

Também temos um script que pode executar para ajudá-lo com a validação da elegibilidade do Grupo de Distribuição para atualizar para o [Grupo do Microsoft 365.](https://aka.ms/DLToM365Group)
