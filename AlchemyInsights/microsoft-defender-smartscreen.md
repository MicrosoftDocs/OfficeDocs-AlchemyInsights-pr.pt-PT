# <a name="microsoft-defender-smartscreen"></a>Microsoft Defender Smartscreen

Quando utiliza a nova Microsoft Edge, o Microsoft Defender SmartScreen ajuda-o a identificar sites de phishing e software malictado reportados e a tomar decisões informadas sobre transferências.

Para saber mais, consulte [O que é o SmartScreen e como pode ajudar a proteger-me?](https://support.microsoft.com/microsoft-edge/what-is-smartscreen-and-how-can-it-help-protect-me-1c9a874a-6826-be5e-45b1-67fa445a74c8).
