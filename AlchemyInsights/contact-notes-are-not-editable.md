# <a name="cant-edit-the-notes-field-for-a-contact-in-outlook"></a>Não é possível editar o campo de notas para um contacto no Outlook

Esta é uma questão conhecida. Estamos a trabalhar para o corrigir e podes ver o estado mais recente no [Can't editar o campo](https://support.office.com/article/fb8394ce-04ce-48b5-bae4-be46f77f10fe)de notas para um contacto.

**Sinuoso**: Se estiver a utilizar uma conta Microsoft 365 ou uma conta Outlook.com, pode editar os seus contactos utilizando a web. Se estiver a utilizar uma conta IMAP ou POP, não existe qualquer suver neste momento.
