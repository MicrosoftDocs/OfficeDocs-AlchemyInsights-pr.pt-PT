# <a name="error-message-when-sharing-files-from-windows-explorer"></a>Mensagem de erro ao partilhar ficheiros do Windows Explorer

Ao tentar partilhar do Windows Explorer, poderá receber a seguinte mensagem de erro:
  
"Desculpe, não conseguimos chegar ao servidor agora. Por favor, tente novamente mais tarde"
  
Para resolver a mensagem de erro, do Internet Explorer aceda às **Ferramentas** \> **Opções** \> de Internet clique no separador **Advanced** e, em seguida, limpe a caixa de verificação intitulada **Não guarde páginas encriptadas para o disco**.
  
 **Nota:** Poderá ter de reiniciar o seu navegador para que a definição faça efeito. Para obter informações adicionais, consulte a mensagem de erro "Incapaz de [contactar o servidor agora" ao partilhar ficheiros](https://go.microsoft.com/fwlink/?linkid=2022914).
