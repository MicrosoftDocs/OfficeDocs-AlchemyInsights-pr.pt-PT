# <a name="error-message-when-sharing-files-from-windows-explorer"></a>Mensagem de erro ao compartilhar arquivos do Windows Explorer

Ao tentar compartilhar do Windows Explorer, você pode receber a seguinte mensagem de erro:
  
"Desculpe, não conseguimos chegar ao servidor agora. Por favor, tente novamente mais tarde"
  
Para resolver a mensagem de erro, do Internet Explorer acesse **ferramentas** \> **opções** \> de internet clique na guia **avançada** e, em seguida, limpar a caixa de seleção intitulada Não salve **páginas criptografadas para o disco**.
  
 **Nota:** Você pode precisar reiniciar seu navegador para que a configuração entre em vigor. Para obter informações adicionais, veja ["Incapaz de alcançar o servidor agora" mensagem de erro ao compartilhar arquivos](https://go.microsoft.com/fwlink/?linkid=2022914).
