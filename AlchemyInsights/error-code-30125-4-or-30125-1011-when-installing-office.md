# <a name="error-code-30125-4-or-30125-1011-when-installing-office"></a>Código de erro 30125-4 ou 30125-1011 ao instalar o Office

Pode ver este erro se o seu software antivírus, firewall, configurações de procuração ou ligação o impedir de instalar o Office. Aqui estão algumas coisas que pode tentar resolver o problema:
  
- Reinicie o computador e reinstale o Office

- Instalação Do Office utilizando o [instalador offline do Office](https://support.office.com/article/f0a85fe7-118f-41cb-a791-d59cef96ad1c?wt.mc_id=Alchemy_ClientDIA)

- Desligue temporariamente as definições de procuração

- Desligue temporariamente o software antivírus

- Desligue temporariamente a firewall

Para mais informações consulte [error code 30125-4 ou 30125-1011 ao instalar o Office](https://support.office.com/article/7bfabec6-76be-4cde-880e-819a9c569612?wt.mc_id=Alchemy_ClientDIA)
