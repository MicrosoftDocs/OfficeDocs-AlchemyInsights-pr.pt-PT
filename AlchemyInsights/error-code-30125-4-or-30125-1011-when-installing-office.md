# <a name="error-code-30125-4-or-30125-1011-when-installing-office"></a>Código de erro 30125 4 ou 30125. doc-1011 ao instalar o Office

Poderá ver este erro se o software antivírus, firewall, definições de proxy ou ligação impedirem a instalação do Office. Eis algumas coisas que pode tentar corrigir o problema:
  
- Reinicie o computador e reinstalar o Office

- Instalar o Office utilizando o [programa de instalação offline do Office](https://support.office.com/article/f0a85fe7-118f-41cb-a791-d59cef96ad1c?wt.mc_id=Alchemy_ClientDIA)

- Desactivar temporariamente as definições de proxy

- Desactivar temporariamente o software antivírus

- Desactivar temporariamente o firewall

Para mais informações consulte o [código de erro 30125 4 ou 30125. doc-1011 ao instalar o Office](https://support.office.com/article/7bfabec6-76be-4cde-880e-819a9c569612?wt.mc_id=Alchemy_ClientDIA)
