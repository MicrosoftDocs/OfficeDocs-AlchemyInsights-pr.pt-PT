# <a name="troubleshoot-error-code-aadsts650056"></a>Código de erro de resolução de problemas AADSTS650056

Para resolver o erro AADSTS650056, execute o passo abaixo recomendado.

**AADSTS65005**: Aplicação de recursos mal configurada - A lista de acesso a recursos exigida pela aplicação não contém aplicações detetáveis pelo recurso ou a aplicação do cliente solicitou acesso a recursos, o que não foi especificado na sua lista de acesso a recursos ou serviço de gráfico devolveu mau pedido ou recurso não encontrado. Se a aplicação suportar o SAML, poderá ter configurado a aplicação com o identificador errado (Entidade).

Para obter mais informações, consulte o artigo de resolução de problemas por erro [AADSTS650056](https://docs.microsoft.com/troubleshoot/azure/active-directory/error-code-aadsts650056-misconfigured-app).
