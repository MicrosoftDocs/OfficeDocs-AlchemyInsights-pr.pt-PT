# <a name="block-pop-up-windows-in-microsoft-edge"></a>Bloqueie janelas pop-up no Microsoft Edge

1. Ir **para... (Configurações e muito mais)**  >  **Configurações**  >  **Permissões do site**.
2. Selecione **Pop-ups e redirecionamentos**.
3. Ligue **o Bloco.**
