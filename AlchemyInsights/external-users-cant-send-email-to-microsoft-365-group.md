# <a name="external-users-cant-send-email-to-microsoft-365-group"></a>Utilizadores externos não podem enviar e-mail para o grupo Microsoft 365

Utilize estes passos para permitir que utilizadores externos enviem um e-mail ao seu grupo Microsoft 365:

1. Inicie sessão no [centro](https://admin.microsoft.com/)de  >  **Groups**  >  **administração Grupos grupos**.
2. Escolha o grupo que pretende conseguir abrir o painel de Definições.
3. No **separador Definições,** **selecione Deixe os remetentes externos enviarem um e-mail a este grupo**.
4. Selecione **Guardar**.</br>
    Deixe refletir até uma hora até uma hora para que as definições sejam refletidas.
