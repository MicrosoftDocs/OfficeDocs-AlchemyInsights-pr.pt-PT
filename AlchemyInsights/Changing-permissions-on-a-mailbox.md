# <a name="changing-permissions-on-a-mailbox"></a>Alterar as permissões numa caixa de correio

Se os utilizadores necessitam de aceder a caixas de correio do Office 365 que não os próprios, terão de ser concedidas permissões para trabalhar com essas caixas de correio. Como atribuir essas permissões depende do tipo de caixa de correio. Se os utilizadores necessitam de aceder a:
  
- Caixas de correio de outro utilizador - Consulte a secção [atribuir permissões de caixa de correio para outro utilizador no Office 365](https://docs.microsoft.com//office365/admin/add-users/give-mailbox-permissions-to-another-user).
    
- Uma caixa de correio partilhada - [Adicionar ou remover membros de uma caixa de correio partilhada](https://support.office.com/article/add-or-remove-members-from-a-shared-mailbox-a1cd0ae0-216c-4dc1-8171-bfacfbd4c1a7), consulte.
    
Se as permissões não funcionarão correctamente depois de efectuar os passos acima indicados, vá para o Centro de administração do Exchange e tente definir as permissões a partir daí (consulte [Gerir permissões para os destinatários](https://technet.microsoft.com/library/jj919240%28v=exchg.150%29.aspx)).
