# <a name="credit-request-due-to-a-service-outage"></a>Pedido de crédito devido a uma paragem de serviço

Para solicitar um pedido de crédito devido a uma falha de serviço, por favor abra um pedido de serviço.

Por favor, certifique-se de ter o ID incidente, encontrado na página de [saúde](https://docs.microsoft.com/office365/enterprise/view-service-health) do serviço no centro de administração Microsoft 365.

Informamos que, como o seu pedido é processado, será submetido a validações para observar o impacto da paralisação.
