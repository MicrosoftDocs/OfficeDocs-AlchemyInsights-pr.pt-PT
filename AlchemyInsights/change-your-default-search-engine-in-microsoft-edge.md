# <a name="change-your-default-search-engine-in-microsoft-edge"></a>Mude o seu motor de busca padrão no Microsoft Edge

Para alterar o seu motor de busca predefinido, siga estas instruções:
1. Na barra de endereços, efetue uma pesquisa utilizando o motor de busca que pretende definir como o seu padrão.
2. Selecione **... (Configurações e muito mais) > Configurações > Privacidade, pesquisa e serviços**.
3. Percorra todo o caminho até a secção **Serviços** e, em seguida, selecione **a barra de endereços e procure**.
4. A partir do motor de busca utilizado no menu **da barra de endereços,** selecione o seu motor de busca preferido.
