# <a name="turn-on-newsguard-in-microsoft-edge-on-an-android-or-ios-device"></a>Ligue o NewsGuard no Microsoft Edge num dispositivo Android ou iOS

NewsGuard é uma extensão de terceiros que avalia e classifica websites sobre padrões jornalísticos de credibilidade e transparência nos EUA. Esta extensão está disponível no Microsoft Edge mas está desligada por padrão. Quando ligada, a extensão não impedirá o acesso aos sites e não interromperá a navegação.

Para ligar o NewsGuard, complete os passos abaixo:
1. Selecione ... > **Configurações**  >  **Notação noticiosa.**
2. Ligue **a classificação de visualização na barra de endereços**.
