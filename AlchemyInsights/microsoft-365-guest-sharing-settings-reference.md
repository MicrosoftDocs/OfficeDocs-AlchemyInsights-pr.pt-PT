# <a name="microsoft-365-guest-sharing-settings-reference"></a>Referência de definições de partilha do Microsoft 365 para convidados

O artigo seguinte fornece uma referência para as diversas definições que podem afetar a partilha com pessoas externas à sua organização nas cargas de trabalho do Microsoft 365: Teams, Grupos do Office 365, SharePoint e OneDrive. Estas definições encontram-se nos centros de administração do Azure Active Directory, Microsoft 365, Teams e SharePoint.

- [Referência de definições de partilha do Microsoft 365 para convidados](https://docs.microsoft.com/microsoft-365/solutions/microsoft-365-guest-settings?view=o365-worldwide)
