# <a name="installation-hangs-or-gets-stuck"></a>Instalação deixa de responder ou obtém bloqueada

Poderá ver este comportamento devido a conectividade de rede, o software antivírus ou definições do Windows Update no computador. Eis algumas coisas que pode tentar corrigir o problema.
  
- Reinicie o computador e verificar se o Office instalado
    
- Utilizar uma ligação com fios
    
- Se tiver uma ligação lenta à internet, instalar o Office [utilizando o programa de instalação offline do Office](https://support.office.com/article/f0a85fe7-118f-41cb-a791-d59cef96ad1c?wt.mc_id=Alchemy_ClientDIA.aspx)
    
- Desactivar temporariamente o software antivírus
    
- Executar reparação online a partir do painel de controlo
    
- Remover e reinstalar o Office
    
[Office está a demorar tempo a instalar](https://support.office.com/article/0f09f357-3fef-42a6-b8aa-cef4c6c44bdf?wt.mc_id=Alchemy_ClientDIA.aspx)
