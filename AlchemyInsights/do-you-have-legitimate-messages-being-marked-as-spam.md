# <a name="do-you-have-legitimate-messages-being-marked-as-spam"></a>Tem mensagens legítimas que está a ser marcadas como spam?


Experimente o diagnóstico de acima ou este [guia passo a passo para resolução de problemas de mensagens válidas que está a ser marcadas como spam](https://docs.microsoft.com/office365/securitycompliance/prevent-email-from-being-marked-as-spam-0).
  
A melhor forma de comunicar estes tipos de problemas consiste em:
  
1. [Activar o relato suplemento](https://docs.microsoft.com/office365/securitycompliance/enable-the-report-message-add-in) para o utilizador. 
    
2. Ter o utilizador [a mensagem como não solicitado do relatório](https://support.office.com/article/use-the-report-message-add-in-b5caa9f1-cdf3-4443-af8c-ff724ea719d2?ui=en-US&amp;rs=en-US&amp;ad=US).
    
Se pretender abrir uma permissão de suporte, anexe os cabeçalhos de correio electrónico completo de, pelo menos, **uma mensagem original completa** .
