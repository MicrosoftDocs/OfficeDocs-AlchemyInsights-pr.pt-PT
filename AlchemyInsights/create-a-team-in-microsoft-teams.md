# <a name="create-a-team-in-microsoft-teams"></a>Criar uma equipa em equipas da Microsoft

Para criar uma equipa em Microsoft Teams, execute os seguintes passos:

1. No lado esquerdo das Equipas, clique em **Equipas.**
2. Na parte inferior da lista de equipas, clique em **Juntar ou** criar **uma equipa.**
3. Clique **em Criar uma nova equipa.**

Se está com dificuldades em criar uma equipa, pode haver uma política aplicada. Para obter mais informações, consulte [Manage quem pode criar grupos Microsoft 365](https://docs.microsoft.com/microsoft-365/solutions/manage-creation-of-groups).
