# <a name="one-drive-announcement"></a>Anúncio de Uma Unidade

Faça o back up de ficheiros e fotografias automaticamente, para que tenha sempre acesso aos mesmos, mesmo que o seu PC esteja perdido, danificado ou roubado. Para obter mais informações, consulte [configurar a pasta para PC OneDrive gratuita.](https://www.microsoft.com/microsoft-365/onedrive/pc-cloud-backup)
