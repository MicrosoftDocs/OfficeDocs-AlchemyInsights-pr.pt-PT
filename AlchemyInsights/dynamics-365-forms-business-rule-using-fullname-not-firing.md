# <a name="full-name-field-not-supported-with-unified-inteface-apps"></a>Campo nome completo não suportado com aplicativos inteface unificados

Ações ou condições que usam um campo **nome completo** (FullName) não são suportadas em aplicativos com base na interface unificada. Como alternativa, você pode usar ações ou condições **com os campos nome (** FirstName) e **sobrenome** (sobrenome).
