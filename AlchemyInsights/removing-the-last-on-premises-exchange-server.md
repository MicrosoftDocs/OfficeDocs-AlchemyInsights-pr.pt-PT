# <a name="removing-the-last-on-premises-exchange-server"></a>Remoção do último servidor de intercâmbio no local

Antes de remover o último servidor exchange da sua organização no local, deve certificar-se de que compreende as implicações de o fazer para o ajudar a decidir se precisa desse último servidor para qualquer outro fim. Para mais informações, consulte [Como e quando desativar os seus servidores](https://technet.microsoft.com/library/dn931280%28v=exchg.150%29.aspx)de intercâmbio no local numa implementação híbrida .
