# <a name="how-to-move-a-onenote-notebook"></a>Como mover um bloco de notas do OneNote

* Para mover um bloco de notas do OneNote para OneDrive, vá para o **ficheiro** > **partilha de** > **Mover blocos de notas** no OneNote de 2016. Para mais informações, consulte a [Mover um bloco de notas do OneNote para OneDrive](https://support.office.com/article/Move-a-OneNote-notebook-to-OneDrive-0af0a141-0bdf-49ab-9e50-45dbcca44082) .

* Para obter informações adicionais sobre as melhores práticas como preparar para mover blocos de notas, consulte [procedimentos recomendados a sincronizar o OneNote](https://support.microsoft.com/help/2819334/onenote-syncing-best-practices).

* [Mover um bloco de notas do OneNote para o Windows que tenha partilhado com outros utilizadores](https://support.office.com/article/Move-a-OneNote-for-Windows-notebook-that-you-ve-shared-with-others-56c7659e-1850-49a6-8874-e2db6b440cd4) fornece mais informações sobre como mover blocos de notas partilhados.
