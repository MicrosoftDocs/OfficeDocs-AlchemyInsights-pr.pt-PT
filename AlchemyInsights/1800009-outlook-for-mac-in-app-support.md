# <a name="in-app-support-in-outlook-for-mac"></a>Na aplicação suporte do Outlook para Mac

Para receber suporte em-app para Outlook para problemas de Mac, abra o Outlook e clique em Ajuda \> contactar suporte. Escreva a pergunta e um membro do Outlook para Mac equipa de suporte irá obter contactá-lo. [Contacte o suporte do Outlook para Mac](https://go.microsoft.com/fwlink/?linkid=2002400&amp;clcid=0x409)
