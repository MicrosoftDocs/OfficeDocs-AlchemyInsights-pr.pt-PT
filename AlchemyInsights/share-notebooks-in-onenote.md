# <a name="share-notebooks-in-onenote-for-the-web-or-onenote-2016"></a>Partilhar Cadernos no OneNote para a web ou OneNote 2016

No OneNote para a web, pode partilhar notas convidando outros a visualizar e fazer alterações no seu caderno, ou criando um link para uma página de notas que pode colar num e-mail, blog ou página web. Para mais informações, consulte [Use OneNote para a web partilhar notas com outras pessoas](https://support.office.com/article/D3481FBE-E06C-4883-B7E9-B2EE9F38AED3)

No OneNote 2016, certifique-se de que o seu notebook OneNote 2016 está no OneDrive para que outros possam chegar ao mesmo. Para partilhar, pode convidar pessoas por correio ou obter um link de partilha. Para mais informações, consulte [Partilhar um OneNote 2016 para o windows notebook com outras pessoas](https://support.office.com/article/d14b6033-7a95-4536-9216-bb0a5e0f8285)
