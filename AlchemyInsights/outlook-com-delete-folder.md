# <a name="delete-a-folder"></a>Eliminar uma pasta

1. No painel esquerdo, direito do rato na pasta ou subpasta que pretende eliminar e seleccione **Eliminar pasta**.
2. Seleccione **OK** para confirmar.

Para obter mais informações, consulte [trabalhar com pastas de mensagens em Outlook.com](https://support.office.com/article/6bb0723a-f39f-4a8d-bb3f-fab5dcc2510a?wt.mc_id=Office_Outlook_com_Alchemy).
