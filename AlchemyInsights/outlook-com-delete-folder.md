# <a name="delete-a-folder-in-outlookcom"></a>Apagar uma pasta em Outlook.com

1. No painel esquerdo, clique na pasta ou subpasta que pretende eliminar e selecionar apagar a **pasta**.
2. Selecione **OK** para confirmar.

Para saber mais, consulte Trabalhar com pastas de [mensagens em Outlook.com](https://support.office.com/article/6bb0723a-f39f-4a8d-bb3f-fab5dcc2510a?wt.mc_id=Office_Outlook_com_Alchemy).
