# <a name="microsoft-stream-licensing-overview"></a>Visão geral do licenciamento do Microsoft Stream

O Microsoft Stream é um serviço na nuvem licenciado por subscrição por utilizador. Está disponível como parte de subscrições elegíveis da Microsoft 365. Consulte os preços do Microsoft 365 para obter informações sobre cada tipo de subscrição.

- [Visão geral do licenciamento do Microsoft Stream](https://docs.microsoft.com/stream/license-overview)
