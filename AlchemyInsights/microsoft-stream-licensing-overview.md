# <a name="microsoft-stream-licensing-overview"></a>Visão geral do licenciamento do Microsoft Stream

O Microsoft Stream é um serviço de nuvem licenciado por subscrição de utilizadores. Está disponível como parte de subscrições elegíveis da Microsoft 365. Consulte os preços da Microsoft 365 para obter informações sobre cada tipo de subscrição.

- [Visão geral do licenciamento do Microsoft Stream](https://docs.microsoft.com/stream/license-overview)
