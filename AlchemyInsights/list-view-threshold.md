# <a name="list-view-threshold"></a>Limiar da vista de lista

As listas e bibliotecas do SharePoint podem conter até 30 milhões de itens, mas quando têm mais de 5000 itens, poderá ver um erro limiar de vista de lista quando tenta trabalhar com as mesmas. Este limiar está em vigor para manter o desempenho do serviço. Não pode ser alterado. Também pode receber este erro ao tentar partilhar ou interromper a herança de uma pasta.

Para evitar atingir este limiar, visite:

- [Gerir listas e bibliotecas de grandes dimensões no SharePoint](https://support.office.com/article/manage-large-lists-and-libraries-in-sharepoint-b8588dae-9387-48c2-9248-c24122f07c59)

- [Erro ao Partilhar/Interromper a Herança](https://docs.microsoft.com/SharePoint/troubleshoot/lists-and-libraries/error-share-break-inheritance)
