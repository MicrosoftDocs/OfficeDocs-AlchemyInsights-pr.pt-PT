# <a name="list-view-threshold"></a>Limiar da vista de lista

O limiar de visualização da lista está em vigor para manter o desempenho do serviço. O limiar não pode ser alterado no SharePoint Online. Para evitar atingir este limiar, consulte [Gerir grandes listas e bibliotecas no SharePoint](https://support.office.com/article/manage-large-lists-and-libraries-in-sharepoint-b8588dae-9387-48c2-9248-c24122f07c59).

Também pode receber este erro ao tentar partilhar ou interromper a herança de uma pasta. Ver [Error Share/Break Herança](https://docs.microsoft.com/SharePoint/troubleshoot/lists-and-libraries/error-share-break-inheritance).
