# <a name="add-update-or-delete-payment-method-in-azure"></a>Adicionar, atualizar ou eliminar o método de pagamento em Azure

Um Administrador de Conta pode adicionar um novo cartão de crédito, atualizar um cartão de crédito existente ou apagar um cartão de crédito que não utiliza. Siga estes passos para adicionar ou substituir o método de pagamento:

1. Inscreva-se no [portal Azure](https://portal.azure.com/) como administrador de [conta](https://docs.microsoft.com/azure/billing/billing-subscription-transfer?WT.mc_id=Portal-Microsoft_Azure_Support#whoisaa). **Gestão de Custos de** Pesquisa + Faturação
2. Selecione uma Subscrição a que pretende adicionar o cartão de crédito ou débito.
3. Selecione **métodos de pagamento**
4. No canto superior esquerdo, selecione + para adicionar uma carta. Um formulário de cartão de crédito aparecerá à direita. Insira detalhes do cartão de crédito ou de débito
5. Para tornar este cartão o seu método de pagamento ativo, consulte a caixa ao lado faça deste o meu método de pagamento ativo acima do formulário. Este cartão tornar-se-á o instrumento de pagamento ativo para todas as subscrições utilizando o mesmo cartão que a subscrição selecionada. Selecione **Seguinte**
6. Para **utilizar um cartão de crédito diferente,** verifique a caixa ao lado do cartão que pretende fazer o método de pagamento ativo.
7. Clique **em Definir ativo**
