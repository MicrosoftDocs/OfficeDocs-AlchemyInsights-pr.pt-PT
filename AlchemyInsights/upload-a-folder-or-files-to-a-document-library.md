# <a name="upload-a-folder-or-files-to-a-sharepoint-document-library"></a>Faça o upload de uma pasta ou ficheiros para uma biblioteca de documentos SharePoint

Para fazer o upload de uma pasta, tem de utilizar o Microsoft Edge, o Google Chrome ou o Mozilla FireFox. Não é possível carregar pastas na experiência clássica da biblioteca, nem utilizar o Internet Explorer 10 ou 11.
  
1. Abra a biblioteca de documentos onde pretende fazer o upload de uma pasta ou ficheiros.
    
2. Abra o Explorador de Ficheiros no seu computador e navegue na pasta ou nos ficheiros que pretende carregar.
    
3. Arraste a pasta ou os ficheiros para a biblioteca de documentos. Se não vir os itens que fez o upload aparecerem na biblioteca, refresque a página. 
    
Se não quiser fazer o upload arrastando itens entre janelas, também pode utilizar o botão **Upload** na biblioteca de documentos para selecionar a pasta ou ficheiros que pretende carregar.
