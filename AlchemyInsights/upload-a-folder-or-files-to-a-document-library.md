# <a name="upload-a-folder-or-files-to-a-sharepoint-document-library"></a>Envie uma pasta ou arquivos para uma biblioteca de documentos do SharePoint

Para carregar uma pasta, você precisa usar o Microsoft Edge, o Google Chrome ou o Mozilla FireFox. Você não pode carregar pastas na experiência clássica da biblioteca ou usar o Internet Explorer 10 ou 11.
  
1. Abra a biblioteca de documentos onde você deseja carregar uma pasta ou arquivos.
    
2. Abra o File Explorer no seu computador e navegue na pasta ou arquivos que deseja carregar.
    
3. Arraste a pasta ou arquivos para a biblioteca de documentos. Se você não ver os itens que você carregou aparecer na biblioteca, atualize a página. 
    
Se você não quiser fazer upload arrastando itens entre as janelas, você também pode usar o botão **Upload** na biblioteca de documentos para selecionar a pasta ou arquivos que deseja carregar.
