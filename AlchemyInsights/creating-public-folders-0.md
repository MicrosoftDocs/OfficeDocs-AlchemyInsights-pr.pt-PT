# <a name="creating-public-folders"></a>Criação de pastas públicas

Antes de os seus utilizadores poderem utilizar pastas públicas, terá de criar pelo menos uma caixa de correio de pastas públicas. Para criar:
  
1. No centro de administração Microsoft 365, vá ao **Admin Centers** \> **Exchange**.

2. Navegue para **pastas** \> **públicas Caixas de correio públicas**e clique em **Adicionar**.

3. Na **Caixa de Correio da Pasta Pública,** forneça um nome para a caixa de correio da pasta pública.

4. Clique em **Guardar**.

Uma vez criada uma caixa de correio, pode começar a criar pastas individuais ou dizer aos seus utilizadores para começarem a [criar e utilizarem pastas públicas no Outlook](https://support.office.com/article/Create-and-share-a-public-folder-in-Outlook-a2835011-d524-4a5c-a207-05c159bb2a97).
