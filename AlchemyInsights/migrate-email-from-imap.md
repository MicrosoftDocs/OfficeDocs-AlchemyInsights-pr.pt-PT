# <a name="when-you-get-a-not-an-accepted-domain-error"></a>Quando chegar um "Não um domínio aceite" erro

Quando tentar adicionar um endereço de correio electrónico, o utilizador ou um grupo ou durante a migração de caixas de correio, poderá obter o seguinte erro: **não é possível utilizar o domínio porque não é um domínio aceite para o sistema de intercâmbio ainda.**
  
Na maioria dos casos, este erro é um resultado do nome do domínio * (por exemplo, contoso.com) * não ser totalmente configurado para a sua organização. 
  
Certifique-se de que o domínio é completamente o programa de configuração, seguindo as instruções de [Adicionar um domínio para o Office 365](https://support.office.com/article/6383f56d-3d09-4dcb-9b41-b5f5a5efd611).
