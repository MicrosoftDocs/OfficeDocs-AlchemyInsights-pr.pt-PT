# <a name="powerpoint-slides-not-showing-in-skype-for-business-on-android-devices"></a>Slides do PowerPoint não aparecem no Skype para negócios em dispositivos Android

Se tiver algum problema com os slides do PowerPoint que não sejam exibidos em dispositivos Android, isso pode dever-se a um bug no navegador Chrome.

Tente desativar e, em seguida, desinstalar o Chrome. Depois de desinstalado o Chrome, reativar o Chrome para obter a versão de fábrica (74.0.3729.136) onde o problema está resolvido.
