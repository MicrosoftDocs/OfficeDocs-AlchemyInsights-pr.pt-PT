# <a name="microsoft-365-groups-not-synching-in-outlook"></a>Microsoft 365 Os grupos não são sincronizados Outlook

Para resolver Microsoft 365 de sincronização de e-mails de Grupo, tem de ativar a OAuth (Autenticação Moderna). 

Para ver os passos para ativar a OAuth, consulte Ativar ou desativar a [autenticação moderna Outlook no Exchange Online](/exchange/clients-and-mobile-in-exchange-online/enable-or-disable-modern-authentication-in-exchange-online).
