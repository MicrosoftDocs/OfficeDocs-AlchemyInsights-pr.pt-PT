# <a name="how-to-buy-a-copy-of-windows-10"></a>Como comprar uma cópia do Windows 10

Para preços actualizados e como comprar e transferir uma versão completa do Windows 10, vá para o [Microsoft Store.](https://www.microsoft.com/store/b/windows)

**Nota:** A Windows 10 atualização gratuita através da Obter o Windows 10 (GWX) terminou a 29 de julho de 2016.

**Importante:** Se estiver a utilizar o Windows 7, a Microsoft recomenda que mude para um novo PC a executar o Windows 10, uma vez que o Windows 7 chegou ao fim do seu ciclo de vida de suporte a 14 de janeiro de 2020.
