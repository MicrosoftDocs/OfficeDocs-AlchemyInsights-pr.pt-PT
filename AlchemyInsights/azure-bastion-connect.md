# <a name="azure-bastion-connect"></a>Azure Bastion Connect

Tente utilizar o Azure Bastion para ligar à sua máquina virtual (VM). Proporciona conectividade RDP segura e sem emenda ao seu VM diretamente no portal Azure sobre TLS. Quando se conecta através do Azure Bastion, a sua máquina virtual não necessita de um endereço IP público ou de uma regra NSG de entrada que permite o tráfego de RDP. Para saber mais sobre o que é O Bastião Azure, consulte [Azure Bastion | Microsoft Docs](https://docs.microsoft.com/azure/bastion/bastion-overview).
