# <a name="solutions-for-error-30180-28"></a>Soluções para o erro 30180-28

Poderá ver [erro 30180-28](https://support.office.com/article/47ae453b-677c-412f-9a21-6766555ff4de?wt.mc_id=Alchemy_ClientDIA) durante uma actualização do Office, se tiver uma má ligação de internet. Eis algumas coisas que pode tentar corrigir o problema.
  
- Reinicie o computador e, em seguida, [actualizar o Office](https://support.office.com/article/2ab296f3-7f03-43a2-8e50-46de917611c5?wt.mc_id=Alchemy_ClientDIA).

- Siga os passos acima, mas utilizar uma ligação com fios durante a actualização do Office.
