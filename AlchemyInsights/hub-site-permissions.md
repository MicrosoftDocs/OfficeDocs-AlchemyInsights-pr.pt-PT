# <a name="hub-site-permissions"></a>Permissões do site do hub

A gestão das permissões de um hub depende do tipo de site subjacente. Se o site for um site de equipa ligado a grupos, então deve gerir permissões através do grupo Microsoft 365. Se for um site de comunicação, então deve gerir permissões através dos grupos SharePoint.

- [Permissões do site do hub](https://docs.microsoft.com/sharepoint/modern-experience-sharing-permissions#hub-site-permissions)  
- [Sites de Hub de Planeamento](https://docs.microsoft.com/sharepoint/planning-hub-sites)
