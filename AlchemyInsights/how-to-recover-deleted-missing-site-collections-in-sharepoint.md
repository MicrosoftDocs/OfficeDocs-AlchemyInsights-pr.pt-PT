# <a name="recover-missing-or-deleted-site-collections"></a>Recuperar recolhas de sites desaparecidas ou eliminadas

Consulte os seguintes artigos para obter informações sobre como recuperar as recolhas de sites desaparecidas ou eliminadas no SharePoint.

[Restaurar sites eliminados](https://docs.microsoft.com/sharepoint/restore-deleted-site-collection)

[Restaurar usando o módulo SPO PowerShell](https://support.office.com/article/Introduction-to-the-SharePoint-Online-Management-Shell-C16941C3-19B4-4710-8056-34C034493429)

Se não conseguir encontrar o site, poderá querer [pesquisar o registo de auditoria no Microsoft 365 Security Compliance &amp; Center](https://docs.microsoft.com/microsoft-365/compliance/search-the-audit-log-in-security-and-compliance).

**Nota:** Sites de equipas "modernos" não são visíveis na **UI de** administração Do Clássico SharePoint. Pode aceder à lista de sites "modernos" a partir do [centro de administração New Sharepoint.](https://docs.microsoft.com/sharepoint/get-started-new-admin-center)
