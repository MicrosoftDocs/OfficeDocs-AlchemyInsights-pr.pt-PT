# <a name="service-principal-key"></a>Chave principal do serviço

Para obter orientações sobre a gestão de credenciais chave para um diretor de serviço, consulte os seguintes artigos:

- [Get-AzureADServicePrincipalKeyCredential](https://docs.microsoft.com/powershell/module/azuread/get-azureadserviceprincipalkeycredential) - Para obter credenciais-chave para um chefe de serviço usando o PowerShell.

- [New-AzureADServicePrincipalKeyCredential](https://docs.microsoft.com/powershell/module/azuread/new-azureadserviceprincipalkeycredential) - Para criar uma nova credencial chave para um chefe de serviço usando o PowerShell.

- [Remove-AzureADServicePrincipalKeyCredential](https://docs.microsoft.com/powershell/module/azuread/remove-azureadserviceprincipalkeycredential) - Para remover uma credencial chave de um principal de serviço utilizando o PowerShell.
