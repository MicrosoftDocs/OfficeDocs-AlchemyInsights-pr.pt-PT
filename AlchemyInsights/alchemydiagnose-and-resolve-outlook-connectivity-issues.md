# <a name="diagnose-and-resolve-outlook-connectivity-issues"></a>Diagnosticar e resolver problemas de conectividade do Outlook

Para diagnosticar e corrigir os problemas de conectividade do Outlook, pode executar a [ferramenta de conectividade do Outlook](https://aka.ms/SaRA-OutlookDisconnect-Alchemy) no computador afectado. 
  
Esta ferramenta irá efectuar verificações de diagnóstico e sugere possíveis soluções para conseguir resolver o problema de conectividade do Outlook.
