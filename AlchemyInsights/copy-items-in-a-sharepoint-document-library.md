# <a name="copy-items-in-a-sharepoint-document-library"></a>Copiar itens numa biblioteca de documentos SharePoint

1. Num navegador web, navegue nos ficheiros, pastas ou links que pretende copiar e, em seguida, clique em **Copiar para**.

    > [!NOTE]
    > **A cópia para** não está disponível se estiver a usar a experiência clássica do SharePoint Online.
  
2. Em **Escolher um destino,** selecione o local para onde pretende copiar os itens. Se estiver a copiar para um site ou subsite diferente e não o vir listado, clique em **sites browse** para ver a lista completa de sites para os quais pode copiar.

    > [!NOTE]
    > Se não vir outros sites listados quando copia itens, copiar através de sites não foi configurado. Para o ativar, vá à página de definições do centro de administração do SharePoint e clique em **OK**.
  
    Para criar uma nova pasta para copiar os ficheiros, selecione uma localização na hierarquia da pasta, clique em **Nova pasta,** introduza um nome para a pasta e clique na marca de verificação para guardar o nome.

3. Clique em **Copiar aqui**.

> [!NOTE]
> Pode copiar até 500 MB de ficheiros e pastas de uma só vez. > Quando utiliza o **Copy** com documentos que têm histórico de versão, apenas a versão mais recente é copiada.
