# <a name="add-or-remove-an-email-address-for-a-user"></a>Adicione ou remova um endereço de e-mail para um utilizador

[Siga estes passos](https://portal.office.com/AdminPortal/Home#/AssistedGuide/addemailoptions) no centro de administração da Microsoft 365 para começar.

 **Nota:** Pode ter mais do que um endereço de e-mail para um utilizador, mas os pseudónimos adicionais de *e-mail* só podem ser utilizados para receber e-mail. Se precisar enviar e-mail com um pseudónimo, crie uma caixa de [correio partilhada](https://docs.microsoft.com/office365/admin/email/create-a-shared-mailbox). Desta forma, o pseudónimo enviará e receberá correio.
