# <a name="create-custom-microsoft-surface-operating-system-images"></a>Crie imagens personalizadas do sistema operativo Microsoft Surface

Utilize o Microsoft Surface Deployment Accelerator (SDA) para simplificar e automatizar a implementação de um novo sistema operativo (OS) que seja adaptado ao seu org e que satisfaça as recomendações da Microsoft.

1. Em GitHub, vá ao SurfaceDeploymentAccelerator.

1. Selecione **Código** e, em seguida, selecione **Abrir com o GitHub Desktop** ou baixar **ZIP**.

1. Reveja o ficheiro **README.md** para saber como funciona o guião.

1. Edite o script com as variáveis apropriadas para o seu ambiente, conforme documentado no ficheiro Readme e, em seguida, reveja-o antes de o executar no seu ambiente de teste.
