# <a name="there-isnt-enough-memory-to-complete-this-action"></a>Não há memória suficiente para completar esta ação.

Quando trabalha com um ficheiro Microsoft Excel, recebe uma das seguintes mensagens:

- O Excel não pode completar esta tarefa com recursos disponíveis. Escolha menos dados ou feche outras aplicações.
- Fora da Memória
- Não há recursos de sistema suficientes para exibir completamente
- Não há memória suficiente para completar esta ação. Tente utilizar menos dados ou fechar outras aplicações. Para aumentar a disponibilidade de memória, considere: 
    - Utilizando uma versão de 64 bits do Microsoft Excel.
    - Adicionando memória ao seu dispositivo.

Para resolver estes erros, consulte [Como resolver os erros de "recursos disponíveis" no Excel](https://docs.microsoft.com/office/troubleshoot/excel/available-resources-errors).
