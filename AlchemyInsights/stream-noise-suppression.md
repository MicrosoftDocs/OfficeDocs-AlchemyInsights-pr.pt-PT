# <a name="stream-noise-suppression"></a>Supressão de ruído sonoro de fluxo

A supressão de ruído, quando ligada pelos proprietários de vídeo, permite que os espectadores isolem o discurso de outros sons e música durante a reprodução. A supressão de ruído pode ser ligada ou desligada para cada vídeo recém-carregado. Para obter mais informações, consulte [a supressão de ruído.](https://docs.microsoft.com/stream/noise-suppression)
