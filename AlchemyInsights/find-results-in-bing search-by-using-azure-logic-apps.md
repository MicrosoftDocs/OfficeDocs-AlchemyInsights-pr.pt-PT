# <a name="find-results-in-bing-search-by-using-azure-logic-apps"></a>Encontre resultados em Bing Search utilizando aplicações Azure Logic

Ao utilizar o Bing Search a partir de uma aplicação lógica, pode automatizar o processamento de resultados de pesquisa e disponibilizar esses resultados para outras ações. Por exemplo, pode encontrar notícias e depois ter o Twitter a publicá-los como tweets no seu feed.

Para saber mais, consulte [os resultados da Bing Search utilizando aplicações Azure Logic.](https://go.microsoft.com/fwlink/?linkid=2151928)
