# <a name="error-when-a-microsoft-365-trial-user-sends-email-client-host-blocked-using-spamhaus"></a>Erro quando um utilizador de teste microsoft 365 envia e-mail: Anfitrião do cliente bloqueado usando Spamhaus

Parece que está a enviar e-mails de uma nova organização de testes microsoft 365. Para ajudar a prevenir abusos, bloqueamos endereços IP que se sabe terem sido previamente comprometidos. Este cheque específico já não é aplicado depois de começar a usar licenças pagas. Poderá obter alívio temporário pedindo um novo endereço IP do seu fornecedor de serviços de internet (ISP), ou removendo o bloco do website do [Spamhaus.](https://go.microsoft.com/fwlink/p/?linkid=123245)
