# <a name="microsoft-threat-experts"></a>Especialistas em ameaças da Microsoft

Microsoft Threat Experts é um serviço de caça a ameaças gerido no Microsoft Defender para Endpoint.  Associamo-nos ao seu Centro de Operações de Segurança para monitorizar, analisar e apoiar para identificar e responder a ameaças críticas no seu ambiente único.

Este serviço gerido de caça a ameaças fornece insights e dados orientados por especialistas através destas duas capacidades: notificação de ataque direcionada e acesso a especialistas a pedido. Consulte a documentação [dos Peritos em Ameaças](https://docs.microsoft.com/windows/security/threat-protection/microsoft-defender-atp/microsoft-threat-experts) da Microsoft para obter mais detalhes.
