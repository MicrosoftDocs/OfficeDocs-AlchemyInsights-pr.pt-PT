# <a name="powerapps-and-microsoft-flow-plan-extension"></a>Extensão do plano PowerApps e Microsoft Flow

Se pretender solicitar uma extensão de plano para **o Microsoft Flow** ou **PowerApps** e o seu inquilino já tiver utilizadores ativos de uma das seguintes funcionalidades, não há necessidade de apresentar um pedido de suporte.

- Criação e publicação de conectores personalizados em PowerApps e Microsoft Flow
- Ações personalizadas HTTP integradas no Flow fora do SharePoint e do OneDrive
- Integração de dados no local através do portal de dados no local

Os clientes com utilizadores ativos destas funcionalidades terão uma extensão automática até 31 de janeiro de 2020, ou a expiração do seu atual prazo de subscrição Microsoft 365, o que for mais longo.
