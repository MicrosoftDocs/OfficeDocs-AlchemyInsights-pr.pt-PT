# <a name="unable-to-access-files-in-a-team"></a>Não é possível aceder a ficheiros numa equipa

O acesso de ficheiros nas Equipas microsoft baseia-se nas definições configuradas no SharePoint ou no OneDrive para partilha de ficheiros.

- Se os novos membros da equipa sofrerem um atraso no acesso aos ficheiros, façam-nos esperar pelo menos 4 horas.

- Se o utilizador tiver um problema de acesso ao separador **Ficheiros,** este poderá estar relacionado com [o erro "O objeto especificado não pertence a uma lista" ao selecionar o separador Ficheiros num canal de Equipas.](https://docs.microsoft.com/microsoftteams/troubleshoot/files/object-specified-not-belong-to-list)
