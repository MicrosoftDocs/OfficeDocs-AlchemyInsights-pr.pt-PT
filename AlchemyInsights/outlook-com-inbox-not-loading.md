# <a name="inbox-not-loading-with-firefox"></a>Pasta a receber não carregado com o Firefox

Determinados suplementos no Firefox browser poderão impedir a pasta a receber de Outlook.com a carga.
  
Visite o suporte Firefox para obter informações sobre como [desactivar ou remover suplementos](https://support.mozilla.org/kb/disable-or-remove-add-ons).
