# <a name="how-to-change-your-theme"></a>Como alterar o tema

Para alterar o tema em Outlook.com ou o Outlook na web, vá para **Definições** e escolha um tema em **tema**. Para ver todos os temas disponíveis, seleccione **Ver todos** em **tema**.
