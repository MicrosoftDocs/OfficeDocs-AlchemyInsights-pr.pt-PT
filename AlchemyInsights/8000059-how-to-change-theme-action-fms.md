# <a name="how-to-change-your-theme"></a>Como alterar o tema

Para alterar o tema em Outlook.com ou o Outlook na web, vá para **Definições**![definições de](media/f4b2e798-fff1-4a14-931f-5677a4543b58.png) e escolher um tema em **tema**. Para ver todos os temas disponíveis, seleccione **Ver todos** em **tema**.
