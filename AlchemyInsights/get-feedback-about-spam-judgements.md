# <a name="new-feature-get-feedback-about-spam-judgments"></a>Nova funcionalidade: obter respostas sobre decisões de spam

Como administrador, pode agora facilmente comunicar mensagens de correio electrónico suspeitas, URLs e conteúdo para a Microsoft no [Explorador de submissões](https://protection.office.com/reportsubmission). Até pode utilizar para ajudar a compreender porque é que uma mensagem legítima foi sinalizada incorrectamente. Para mais informações, consulte este [anúncio](https://techcommunity.microsoft.com/t5/Security-Privacy-and-Compliance/Empower-security-teams-to-easily-report-suspicious-emails-amp/ba-p/752622).

Para que easer, pode permitir que os utilizadores comunicar directamente à Microsoft através do [suplemento de mensagem de relatório](https://appsource.microsoft.com/product/office/WA104381180?src=office&tab=Overview).
