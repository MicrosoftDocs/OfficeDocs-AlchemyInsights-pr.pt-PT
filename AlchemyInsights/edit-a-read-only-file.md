# <a name="edit-a-read-only-file"></a>Editar um ficheiro só de leitura

Em bibliotecas de documentos, quando clica no nome de um documento, o programa do Office poderá inesperadamente abri-lo como um ficheiro só de leitura. Para abri-lo para edição, clique em **Abrir** (três pontos) e, em seguida, clique em **Editar**
