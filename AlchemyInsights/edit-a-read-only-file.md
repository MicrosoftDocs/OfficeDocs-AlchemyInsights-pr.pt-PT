# <a name="edit-a-read-only-file"></a>Eite um arquivo somente para leitura

Nas bibliotecas de documentos, quando você clica no nome de um documento, o programa do Office pode abri-lo inesperadamente como um arquivo somente de leitura. Para abri-lo para edição, clique **no Open** (os três pontões) e, em seguida, clique em **Editar.**
