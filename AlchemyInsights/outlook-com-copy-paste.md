# <a name="copy-and-paste-in-outlookcom"></a>Copiar e colar em Outlook.com

Não há botões de cópia ou pasta em Outlook.com, mas pode usar atalhos de teclado:

**Num PC:** CTRL+C para copiar, CTRL+X para cortar e CTRL+V para colar.

**Num Mac:** CMD+C para copiar, CMD+X para cortar, e CMD+V para colar.
