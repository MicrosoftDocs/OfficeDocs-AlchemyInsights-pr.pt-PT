# <a name="workflows-in-sharepoint"></a>Fluxos de trabalho no SharePoint

Fluxos de trabalho no SharePoint permitem que você modelar e automatizar processos de negócios. O SharePoint oferece suporte a dois ambientes de desenvolvimento de fluxo de trabalho primários para criar fluxos de trabalho: SharePoint Designer e Visual Studio. 

Consulte [desenvolver fluxos de trabalho do SharePoint no SharePoint Designer e Visio](https://docs.microsoft.com/sharepoint/dev/general-development/develop-sharepoint-workflows-using-visual-studio) ou [desenvolver fluxos de trabalho do SharePoint usando o Visual Studio](https://docs.microsoft.com/sharepoint/dev/general-development/develop-sharepoint-workflows-using-visual-studio) para avaliar as vantagens e desvantagens de cada um. 

Se você estiver tentando determinar se os fluxos de trabalho são uma solução para usar em seu ambiente do SharePoint ou está procurando informações gerais sobre como começar, consulte [visão geral dos fluxos de trabalho no SharePoint](https://docs.microsoft.com/sharepoint/dev/general-development/get-started-with-workflows-in-sharepoint#overview-of-workflows-in-sharepoint).
