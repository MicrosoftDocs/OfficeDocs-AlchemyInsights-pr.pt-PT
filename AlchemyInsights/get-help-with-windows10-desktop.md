# <a name="get-help-with-windows-10-desktop"></a>Obtenha ajuda com o Windows 10 Desktop

- Para criar um atalho para desktop, consulte [aplicações pin e pastas para o ambiente de trabalho ou barra de tarefas](https://support.microsoft.com/windows/pin-apps-and-folders-to-the-desktop-or-taskbar-f3c749fb-e298-4cf1-adda-7fd635df6bb0).

- Para tornar os ícones ou texto maiores no Windows 10, consulte [alterar o tamanho dos seus ícones de ambiente de trabalho](https://support.microsoft.com/windows/change-the-size-of-your-desktop-icons-85a9d341-2a4f-3d96-c796-ae116a187211).

- Para mostrar ícones de ambiente de trabalho no Windows 10, consulte [os ícones do ambiente de trabalho do Windows 10](https://support.microsoft.com/windows/show-desktop-icons-in-windows-10-c13270f0-3812-c71d-f27e-29aa32588b20).
