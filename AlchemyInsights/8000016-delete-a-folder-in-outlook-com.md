# <a name="delete-a-folder"></a>Eliminar uma pasta

1. No painel esquerdo, direito do rato na pasta ou subpasta que pretende eliminar e seleccione **Eliminar**. 
    
2. Seleccione **OK** para confirmar. 
    
Para obter mais informações, consulte [Eliminar uma pasta no Outlook.com](https://go.microsoft.com/fwlink/p/?linkid=873134).
