# <a name="action-confirmations"></a>Confirmações de acção

Se acidentalmente movida ou eliminada uma mensagem, pode anular essa acção. A caixa de diálogo de cópia de segurança é apresentado na parte inferior do ecrã para cerca de 5 segundos. Seleccione o **Anular** para cancelar a mover ou eliminar a acção.

Pode desactivar esta funcionalidade em [confirmações de acção](https://outlook.live.com/mail/options/general/notifications) (**Definições** > **Ver todas as definições do Outlook** > **Geral** > **notificações**).

Nota: Depois de ter sido enviada uma mensagem, é possível anular essa acção. Para mais informações, consulte [pretende resgatar uma mensagem no Outlook.com](https://support.office.com/article/c069ddde-5282-4085-8f4c-d7b133324f8a).
