# <a name="install-and-configure-onedrive"></a>Instalar e configurar OneDrive

O cliente de sincronização OneDrive tiver sido pré-instalado com o Windows 10. Para instalar o OneDrive para o Windows 7 ou Mac ou para obter a aplicação móvel para o seu telemóvel ou um tablet, vá para [http://onedrive.com/download](http://onedrive.com/download).
  
Para configurar o cliente de sincronização de OneDrive no Windows, consulte:
  
- [Configurar o computador para sincronizar o OneDrive para ficheiros de negócio no Office 365](https://go.microsoft.com/fwlink/?linkid=533375)

- [Sincronizar ficheiros do SharePoint com o novo cliente de sincronização do OneDrive](https://go.microsoft.com/fwlink/?linkid=871666)
