# <a name="error-onedrive-cannot-connect-to-windows"></a>Erro: OneDrive não pode ligar-se ao Windows

Para resolver a mensagem de erro, procure o ficheiro cldflt.sys para ver se está a funcionar, e se não estiver, ative-o. 
  
Para obter informações adicionais sobre como ativar o ficheiro cldflt.sys, consulte o [Erro OneDrive não pode ligar-se ao erro do Windows ao aceder aos ficheiros no OneDrive.](https://go.microsoft.com/fwlink/?Linkid=2031032)
