# <a name="error-onedrive-cannot-connect-to-windows"></a>Erro: O OneDrive não pode ligar-se ao Windows

Para resolver a mensagem de erro, procure o ficheiro cldflt.sys para ver se está em execução e se não está, ative-a. 
  
Para obter informações adicionais sobre como ativar o ficheiro cldflt.sys, consulte [o OneDrive não pode ligar-se ao erro do Windows ao aceder a ficheiros no OneDrive.](https://go.microsoft.com/fwlink/?Linkid=2031032)
