# <a name="error-onedrive-cannot-connect-to-windows"></a>Erro: OneDrive não é possível ligar ao Windows

Para resolver a mensagem de erro, procure o ficheiro cldflt.sys ver se está a ser executado e se não estiver, activá-la. 
  
Para obter informações adicionais sobre como activar o ficheiro de cldflt.sys, consulte [OneDrive não é possível ligar ao Windows "Erro ao aceder a ficheiros em OneDrive.](https://go.microsoft.com/fwlink/?Linkid=2031032)
