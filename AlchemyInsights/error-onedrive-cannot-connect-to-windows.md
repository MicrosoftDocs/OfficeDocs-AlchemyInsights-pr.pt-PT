# <a name="error-onedrive-cannot-connect-to-windows"></a>Erro: O OneDrive não pode se conectar ao Windows

Para resolver a mensagem de erro, pesquise o arquivo cldflt.sys para ver se ele está sendo executado e, se não estiver, habilite. 
  
Para obter informações adicionais sobre como ativar o arquivo cldflt.sys, consulte o OneDrive não pode se conectar ao erro do [Windows ao acessar arquivos no OneDrive.](https://go.microsoft.com/fwlink/?Linkid=2031032)
