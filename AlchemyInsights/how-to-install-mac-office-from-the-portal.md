# <a name="how-to-install-mac-office-from-the-portal"></a>Como instalar o Mac Office a partir do portal

1. [Desinstalar o Office 2011 para Mac do computador](https://support.office.com/article/4bfcd230-0ea1-4656-bf30-dbfa44d358fa?wt.mc_id=Alchemy_ClientDIA).

2. [Transferir o software de gerir as definições no Office 365](https://docs.microsoft.com/DeployOffice/manage-software-download-settings-office-365).

3. [Atribuir licenças aos utilizadores](https://docs.microsoft.com//office365/admin/subscriptions-and-billing/assign-licenses-to-users).

4. Os utilizadores podem, em seguida, instalar o software do Office próprios ou pode utilizar a [Implementar o Office para os utilizadores de Mac](https://docs.microsoft.com/DeployOffice/mac/deployment-guide-for-office-for-mac).
