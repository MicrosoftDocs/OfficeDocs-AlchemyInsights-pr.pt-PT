# <a name="how-to-install-mac-office-from-the-portal"></a>Como instalar o Mac Office a partir do portal

1. [Desinstalar o Office 2011 para o Mac a partir dos seus computadores](https://support.office.com/article/4bfcd230-0ea1-4656-bf30-dbfa44d358fa?wt.mc_id=Alchemy_ClientDIA).

2. [Gerir as definições de descarregamento de software](https://docs.microsoft.com/DeployOffice/manage-software-download-settings-office-365).

3. [Atribuir licenças aos utilizadores.](https://docs.microsoft.com/microsoft-365/admin/manage/assign-licenses-to-users)

4. Os utilizadores podem então instalar o software do Office em si, ou pode utilizar o [Office de implementação para os seus utilizadores Mac.](https://docs.microsoft.com/DeployOffice/mac/deployment-guide-for-office-for-mac)
