# <a name="how-to-type-special-characters-on-a-mac"></a>Como escrever caracteres especiais num Mac

Para escrever caracteres especiais no seu Mac, abra o menu **de caracteres Especiais** com o atalho de teclado ⌘ Espaço de Controlo de **Comando**  +  **Control**  +  **Space**.

Um atalho rápido para @ também é **ALT**  +  **64**.
