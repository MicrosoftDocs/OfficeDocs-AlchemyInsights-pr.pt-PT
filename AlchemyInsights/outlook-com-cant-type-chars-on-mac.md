# <a name="how-to-type-special-characters-on-a-mac"></a>Como escrever caracteres especiais num Mac

Para escrever caracteres especiais no seu Mac, abra o menu **de caracteres especiais** com o atalho do teclado /**Espaço**de **Comando** + **Control** + .

Um atalho rápido para @ também é **ALT** + **64**.
