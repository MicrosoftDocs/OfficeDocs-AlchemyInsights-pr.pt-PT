# <a name="how-to-type-special-characters-on-a-mac"></a>Como escrever caracteres especiais numa Mac

Para escrever caracteres especiais no computador Mac, abrir o menu de **caracteres especiais** com o atalho de teclado ⌘ **comando** + **controlo** + **espaço**.

Um atalho rápido para @ também é **ALT** + **64**.
