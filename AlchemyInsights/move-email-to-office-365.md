# <a name="move-email-to-microsoft-365"></a>Mude o e-mail para o Microsoft 365

Tem várias opções para migrar o e-mail para o Microsoft 365. O [consultor de migração de correio da Microsoft 365](https://aka.ms/alchemyinsight-mailmigrationadvisor) ajuda-o a escolher o melhor para o seu ambiente e também o guia através da migração por correio.
  
Para ler sobre as suas opções sem iniciar a migração, consulte [decidir sobre um caminho de migração.](https://docs.microsoft.com/Exchange/mailbox-migration/decide-on-a-migration-path)

Verifique também a página [Adopt Microsoft 365 com fastTrack](https://www.microsoft.com/fasttrack/microsoft-365/office-365) para obter ajuda assistida na migração, se for elegível.
