# <a name="proxyaddress-incorrect"></a>ProxyAddress incorreto

Quando um objeto é sincronizado para AD Azure, os valores especificados no atributo proxyAddresses em Diretório Ativo são comparados com as regras da AD Azure, e então o atributo proxyAddresses é povoado em Azure AD. Por isso, os valores dos proxyAddresss atributos para o objeto em Diretório Ativo podem não ser os mesmos que os valores do atributo proxyAddresses em Azure AD.
  
Para saber mais sobre como o proxyaddress é povoado, veja [como o atributo proxyAddress é povoado em Azure AD](https://support.microsoft.com/help/3190357/how-the-proxyaddresses-attribute-is-populated-in-azure-ad).
