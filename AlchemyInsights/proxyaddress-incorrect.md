# <a name="proxyaddress-incorrect"></a>ProxyAddress incorreto

Quando um objeto é sincronizado com o Azure AD, os valores que são especificados no atributo proxyAddresses no Active Directory são comparados com as regras do Azure AD e, em seguida, o atributo proxyAddresses é preenchido no Azure AD. Por conseguinte, os valores do atributo proxyAddresses para o objecto no Active Directory poderão não ser iguais aos valores do atributo proxyAddresses no Azure AD.
  
Para saber mais sobre como o proxyAddress é preenchido, consulte [como o atributo proxyAddress é preenchido no Azure ad](https://support.microsoft.com/help/3190357/how-the-proxyaddresses-attribute-is-populated-in-azure-ad).
