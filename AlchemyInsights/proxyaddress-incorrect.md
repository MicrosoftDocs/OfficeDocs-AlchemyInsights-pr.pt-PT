# <a name="proxyaddress-incorrect"></a>ProxyAddress incorrecto

Quando um objecto é sincronizado para Azure AD, os valores que são especificados no atributo proxyAddresses no Active Directory são comparados com as regras de Azure AD e, em seguida, o atributo proxyAddresses for preenchido no Azure AD. Por conseguinte, os valores do atributo proxyAddresses para o objecto no Active Directory podem não ser o mesmo que os valores do atributo proxyAddresses em Azure AD.
  
Para mais informações sobre como o proxyaddress é preenchida, consulte a [forma como o atributo proxyAddress for preenchido no Azure AD](https://support.microsoft.com/en-us/help/3190357/how-the-proxyaddresses-attribute-is-populated-in-azure-ad).
