# <a name="proxyaddress-incorrect"></a>ProxyAddress incorreto

Quando um objeto é sincronizado com o AD do Azure, os valores especificados no atributo proxyAddressno Diretório Ativo são comparados com as regras do AD do Azure e, em seguida, o atributo proxyAddressé preenchido em AD azure. Portanto, os valores do atributo proxyAddresss para o objeto no Diretório Ativo pode não ser o mesmo que os valores do atributo proxyAddresss em AD Azure.
  
Para saber mais sobre como o endereço proxy é preenchido, veja [como o atributo proxyAddress é preenchido em azure ad](https://support.microsoft.com/help/3190357/how-the-proxyaddresses-attribute-is-populated-in-azure-ad).
