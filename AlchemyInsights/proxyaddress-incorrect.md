# <a name="proxyaddress-incorrect"></a>ProxyAddress incorreto

Quando um objeto é sincronizado com Azure AD, os valores especificados no atributo proxyAddresses no Ative Directory são comparados com as regras Azure AD, e então o atributo proxyAddresses é povoado em Azure AD. Portanto, os valores do atributo proxyAddresses para o objeto no Ative Directory podem não ser os mesmos que os valores do proxyAddresses atribuem em Azure AD.
  
Para saber mais sobre como o proxyaddress é povoado, veja [como o atributo proxyAddress é povoado em Azure AD](https://support.microsoft.com/help/3190357/how-the-proxyaddresses-attribute-is-populated-in-azure-ad).
