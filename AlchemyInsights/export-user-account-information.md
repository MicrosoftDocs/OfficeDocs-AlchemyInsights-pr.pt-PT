# <a name="export-user-account-information"></a>Exportar informações da conta de utilizador

Para exportar informações da conta de utilizador no Microsoft 365 admin centro, incluindo informações sobre licenças, política de palavra-passe, contas bloqueadas, o estado da sincronização das e mais, vá para os **utilizadores** > página de[utilizadores do Active](https://go.microsoft.com/fwlink/p/?linkid=834822) e, em seguida, seleccione **exportação Os utilizadores**.
