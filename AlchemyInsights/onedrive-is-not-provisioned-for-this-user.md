# <a name="onedrive-is-not-provisioned-for-this-user"></a>O OneDrive não está previsto para este utilizador

Para contas recém-criadas, os utilizadores precisam de iniciar sômposições no centro de administração microsoft 365 com a sua própria conta e clicar no OneDrive para iniciar manualmente a oferta do site OneDrive.
  
No entanto, se o utilizador já tiver clicado no OneDrive e o seu site pessoal não for provisionado dentro de 24 horas, por favor, regisuente um bilhete de apoio para mais assistência.
