# <a name="onedrive-is-not-provisioned-for-this-user"></a>OneDrive não está aprovisionado para este utilizador

Para contas recentemente criadas, os utilizadores necessitam iniciar sessão no portal Office 365 com a sua própria conta e clique em OneDrive para iniciar manualmente o fornecimento do OneDrive site.
  
No entanto, se o utilizador já tenha clicado em OneDrive e respectivos sites pessoais não está aprovisionado no prazo de 24 horas, inicie uma permissão de suporte para obter mais assistência.
