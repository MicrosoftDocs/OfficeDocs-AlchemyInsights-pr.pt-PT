# <a name="onedrive-is-not-provisioned-for-this-user"></a>OneDrive não é provisionado para este utilizador

Para contas recém-criadas, os utilizadores precisam de iniciar sessão no centro de administração da Microsoft 365 com a sua própria conta e clicar no OneDrive para iniciar manualmente a oferta do site OneDrive.
  
No entanto, se o utilizador já clicou no OneDrive e o seu site pessoal não for provisionado dentro de 24 horas, por favor, faça login num bilhete de apoio para mais assistência.
