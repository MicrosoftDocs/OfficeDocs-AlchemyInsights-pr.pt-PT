# <a name="onedrive-is-not-provisioned-for-this-user"></a>OneDrive não é provisionado para este usuário

Para contas recém-criadas, os usuários precisam entrar no centro de administração do Microsoft 365 com sua própria conta e clicar no OneDrive para iniciar manualmente o fornecimento do site OneDrive.
  
No entanto, se o usuário já clicou OneDrive e seu site pessoal não é provisionado dentro de 24 horas, por favor, registrar um bilhete de suporte para assistência adicional.
