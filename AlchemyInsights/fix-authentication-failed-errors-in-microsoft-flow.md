# <a name="fix-authentication-failed-errors-in-microsoft-flow"></a>Corrigir erros de "Autenticação Falhada" no Microsoft Flow

Se receber uma mensagem de erro "Autenticação Falhada" quando tenta iniciar sação no Microsoft Flow, é porque o serviço Flow está desativado no Azure Ative Directory. Para obter detalhes e passos para corrigir o erro, consulte [Não pode iniciar sôm no Microsoft Flow e recebe um erro de "Autenticação falhada".](https://support.microsoft.com/help/4316891)
