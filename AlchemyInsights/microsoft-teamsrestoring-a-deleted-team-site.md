# <a name="restoring-a-deleted-team-in-microsoft-teams"></a>Restaurando uma equipe excluída em equipes da Microsoft

Você acidentalmente excluiu uma equipe? Se já se passaram menos de 30 dias desde que você apagou a equipe, você está com sorte; você pode restaurá-lo. Para fazer isso, siga os passos aqui: [Restaurar uma equipe excluída](https://docs.microsoft.com/microsoftteams/archive-or-delete-a-team#restore-a-deleted-team).
