# <a name="microsoft-365-language-settings"></a>Microsoft 365 Definições de idioma

Determine se deseja alterar o idioma de exibição e outras configurações para os utilizadores ou para todo o inquilino.

- Para os utilizadores, consulte [Alterar o seu idioma de exibição e fuso horário na Microsoft 365 para Negócios para todas as aplicações.](https://support.microsoft.com/office/6f238bff-5252-441e-b32b-655d5d85d15b)
- Para o inquilino, consulte  [definição de idioma e regional para Microsoft 365](https://docs.microsoft.com/office365/troubleshoot/access-management/set-language-and-region).
