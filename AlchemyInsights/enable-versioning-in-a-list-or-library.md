# <a name="enable-versioning-for-a-list-or-library"></a>Ativar a versão para uma lista ou biblioteca

1. Vá à lista ou biblioteca.
    
2. Clique em **Definições**e, em seguida, clique nas **definições de Biblioteca** ou **Lista**.
    
3. Clique **nas definições de versão**.
    
4. Selecione as definições desejada e, em seguida, clique **em OK**.
