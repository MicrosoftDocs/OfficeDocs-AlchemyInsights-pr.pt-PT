# <a name="gatewaynotreachable-when-setting-credentials"></a>GatewayNotReachable ao definir credenciais

Poderá encontrar GatewayNotReachable ao tentar definir credenciais para uma origem de dados. Isto pode ser o resultado de um gateway desactualizado. Instalar o gateway mais recente e tente novamente.

Para mais informações:[https://docs.microsoft.com/power-bi/refresh-troubleshooting-refresh-scenarios](https://docs.microsoft.com/power-bi/refresh-troubleshooting-refresh-scenarios)
