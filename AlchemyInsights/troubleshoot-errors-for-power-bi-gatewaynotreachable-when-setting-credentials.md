# <a name="gatewaynotreachable-when-setting-credentials"></a>Gatewaynotreacessível ao definir credenciais

Você pode encontrar Gatewaynotreável ao tentar definir credenciais para uma fonte de dados dentro do serviço do Power BI. Isso pode ser o resultado de um gateway desatualizado. Instale o gateway mais recente e tente novamente.

Para obter mais informações:[https://docs.microsoft.com/power-bi/refresh-troubleshooting-refresh-scenarios](https://docs.microsoft.com/power-bi/refresh-troubleshooting-refresh-scenarios)
