# <a name="add-your-outlook-on-the-web-account-to-outlook"></a>Adicionar o Outlook na conta de web para o Outlook

Estes passos são iguais, se estiver a adicionar a primeira conta de correio electrónico ou contas de correio electrónico adicionais para o Outlook.

1. Seleccione o **ficheiro** > **Adicionar conta**.
1. Em seguida o que vê depende da versão do Outlook:
    - Para o **Outlook para o Office 365** e **de 2016 Outlook**, introduza o seu endereço de correio electrónico e clique em **Ligar**.
    - Para **Outlook 2013** e o **Outlook 2010**, introduza o nome, endereço de correio electrónico e palavra-passe e clique em **seguinte**.
1. Se lhe for pedido, introduza novamente a palavra-passe, em seguida, seleccione **' OK '** > **Concluir** para começar a utilizar a conta de correio electrónico no Outlook.

Para mais informações, consulte [Adicionar uma conta de correio electrónico para o Outlook](https://support.office.com/article/6e27792a-9267-4aa4-8bb6-c84ef146101b)
