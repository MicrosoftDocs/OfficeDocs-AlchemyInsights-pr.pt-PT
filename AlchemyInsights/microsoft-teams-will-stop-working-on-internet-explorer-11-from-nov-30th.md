# <a name="microsoft-teams-will-stop-working-on-internet-explorer-11-from-nov-30th"></a>Microsoft Teams vai deixar de trabalhar no Internet Explorer 11 a partir de 30 de novembro

Conforme comunicado no MC225570, a partir de 30 de novembro de 2020, a aplicação web Microsoft Teams deixará de suportar o Internet Explorer (IE) 11. 

Recomendamos que atualize o novo [Microsoft Edge](https://www.microsoft.com/edge) para uma experiência online mais segura e inovadora. 

Para mais informações, consulte [as aplicações da Microsoft 365, diga adeus ao Internet Explorer 11 e ao Windows 10 sunsets Microsoft Edge Legacy - Microsoft Tech Community](https://techcommunity.microsoft.com/t5/microsoft-365-blog/microsoft-365-apps-say-farewell-to-internet-explorer-11-and/ba-p/1591666)
