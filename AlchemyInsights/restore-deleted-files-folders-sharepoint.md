# <a name="restore-deleted-files-or-folders-in-sharepoint"></a>Restaurar ficheiros ou pastas eliminadas no SharePoint

Se eliminar acidentalmente um ficheiro ou pasta no SharePoint, poderá recuperá-lo mais tarde do caixote do lixo.

* [Restaurar ficheiros ou pastas eliminadas no SharePoint](https://support.office.com/article/restore-deleted-items-from-the-site-collection-recycle-bin-5fa924ee-16d7-487b-9a0a-021b9062d14b)

Se muitos dos seus ficheiros SharePoint forem eliminados, substituídos, corrompidos ou infetados por malware, pode restaurar toda a sua biblioteca SharePoint para uma hora anterior.

* [Restaurar uma biblioteca de documentos](https://support.office.com/article/restore-a-document-library-317791c3-8bd0-4dfd-8254-3ca90883d39a)
