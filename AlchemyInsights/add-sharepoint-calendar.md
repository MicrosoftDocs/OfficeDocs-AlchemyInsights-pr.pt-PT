# <a name="add-a-sharepoint-calendar"></a>Adicione um calendário de SharePoint

Os calendários do SharePoint são tipos clássicos da lista que às vezes são confundidos com calendários de grupo do Office 365.
 
Para criar um Calendário SharePoint, siga estes passos:
 
1.  Selecione o ícone de engrenagem **Definições** e, em seguida, selecione **Adicionar uma App**.
2.  Na caixa de pesquisa de **aplicativos,** digite **calendário** e pressione a entrada.
3.  Selecione a aplicação intitulada **Calendário**e, em seguida, introduza um nome para o seu calendário.

Para mais informações sobre o Office 365 Groups, consulte [Saiba mais sobre o Office 365 Groups](https://support.office.com/article/Learn-about-Office-365-groups-b565caa1-5c40-40ef-9915-60fdb2d97fa2).
