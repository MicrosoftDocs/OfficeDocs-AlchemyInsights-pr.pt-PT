# <a name="use-jamf-to-configure-microsoft-edge-policy-settings-on-macos"></a>Utilize o Jamf para configurar as definições de política do Microsoft Edge no macOS

Para configurar o Microsoft Edge (canal estável 81 ou mais tarde) no macOS, utilize um ficheiro manifesto de política do Microsoft Edge no Jamf Pro 10.19.

Para obter mais informações, consulte [configurar as definições de política do Microsoft Edge no macOS com o Jamf](https://go.microsoft.com/fwlink/?linkid=2134761).
