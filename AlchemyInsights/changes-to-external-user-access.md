# <a name="changes-to-external-user-access"></a>Alterações ao acesso de utilizadores externos

Iniciar 23 de Março de 2018, os utilizadores externos já não será possível aceder ao conteúdo que foi partilhado com todos os utilizadores, todos os utilizadores autenticados ou todos os utilizadores de formulários, mesmo que tenham sido convidados ou adicionados aos grupos autorizados. Os utilizadores externos só poderá aceder ao conteúdo que tenha sido partilhado com o utilizador ou grupos aos quais o utilizador pertence. Se pretender manter as definições existentes, consulte:https://support.microsoft.com/help/4089534/how-to-grant-the-everyone-claim-to-external-users-in-office-365
