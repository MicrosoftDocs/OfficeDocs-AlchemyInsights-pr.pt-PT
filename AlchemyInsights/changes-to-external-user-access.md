# <a name="changes-to-external-user-access"></a>Alterações no acesso externo ao utilizador

A partir de 23 de março de 2018, os utilizadores externos deixarão de poder aceder a conteúdos partilhados com todos, todos os utilizadores autenticados ou utilizadores de todas as formas, mesmo que tenham sido convidados ou adicionados a grupos autorizados. Os utilizadores externos apenas poderão aceder a conteúdos partilhados com o utilizador ou com grupos a que o utilizador pertence. Se quiser manter as definições existentes, consulte: https://support.microsoft.com/help/4089534/how-to-grant-the-everyone-claim-to-external-users-in-office-365
