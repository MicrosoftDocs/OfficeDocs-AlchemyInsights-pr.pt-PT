# <a name="sharepoint-permissions-errors"></a>Erros de permissões sharePoint

Há muitos cenários que podem levar a uma destas mensagens. Mais frequentemente, as permissões para o utilizador ou administrador são configuradas incorretamente ou não configuradas de todo. 

Para mais informações, consulte "[Access Denied", "You need permission to access this site", ou "User not found in the directy" erros no SharePoint Online e OneDrive for Business](https://docs.microsoft.com/sharepoint/support/administration/access-denied-or-need-permission-error-sharepoint-online-or-onedrive-for-business).
