# <a name="upgrade-to-the-full-version-of-microsoft-teams"></a>Upgrade para a versão completa do Microsoft Teams

Precisa de atualizar de graça para a versão completa do Microsoft Teams? Consulte [a versão gratuita da Microsoft Teams para a versão de subscrição do Microsoft 365](https://docs.microsoft.com/microsoftteams/upgrade-freemium)

Terá de cumprir estes requisitos:

- És a pessoa que se inscreveu na versão gratuita das Equipas.
- Se tem o seu próprio domínio, já não pode estar associado ao Azure Ative Directory.
- Para transferir dados ao fazer a atualização, tem de subscrever durante o processo de atualização. Se comprou uma subscrição fora do processo de atualização, não pode transferir dados.
