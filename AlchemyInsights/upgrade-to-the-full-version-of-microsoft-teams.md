# <a name="upgrade-to-the-full-version-of-microsoft-teams"></a>Upgrade para a versão completa das Equipas microsoft

Precisa de atualizar do free para a versão completa do Microsoft Teams? Ver [Upgrade Microsoft Teams grátis para a versão de subscrição do Microsoft 365](https://docs.microsoft.com/microsoftteams/upgrade-freemium)

Terá de cumprir estes requisitos:

- És a pessoa que se inscreveu na versão gratuita atual das Equipas.
- Se tem o seu próprio domínio, já não pode estar associado ao Azure Ative Directory.
- Para transferir dados quando faz a atualização, deve subscrever durante o processo de atualização. Se comprou uma subscrição fora do processo de upgrade, não pode transferir dados.
