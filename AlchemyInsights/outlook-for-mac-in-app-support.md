# <a name="in-app-support-in-outlook-for-mac"></a>Na aplicação suporte do Outlook para Mac

Para receber suporte em-app para Outlook para problemas de Mac, abra o Outlook e clique em Ajuda \> contactar suporte. Escreva a pergunta e um membro do Outlook para Mac equipa de suporte irá obter contactá-lo. [Contacte o suporte do Outlook para Mac](https://answers.microsoft.com/msoffice/forum/msoffice_outlook-mso_mac/new-contact-support-feature-in-outlook-2016-for/d4fc21c4-25e2-4e10-b943-1fba6542b517)
