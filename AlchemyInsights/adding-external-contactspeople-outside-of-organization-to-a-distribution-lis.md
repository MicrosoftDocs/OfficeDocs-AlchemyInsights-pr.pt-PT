# <a name="add-external-email-addresses-to-a-distribution-group"></a>Adicionar endereços de correio electrónico externo a um grupo de distribuição

Em primeiro lugar, tem de criar um contacto na sua organização para o endereço de correio electrónico externo. Em seguida, pode adicionar o contacto para o grupo de distribuição. Para mais informações, consulte [Adicionar utilizadores externos a um grupo de distribuição?](https://support.office.com/client/caa0f310-0bb7-48e3-8ad2-cb358b53bbba).
  
Se pretender que os utilizadores externos para conseguir enviar mensagens para o grupo de distribuição, verifique se que o grupo é configurado para aceitar correio de remetentes externos. Para mais informações, consulte [Gestão de entrega](https://technet.microsoft.com/library/bb124513.aspx#deliverymanagement).
