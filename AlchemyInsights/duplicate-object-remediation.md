# <a name="duplicate-object-synchronization-errors"></a>Erros de sincronização de objetos duplicados

Se receber um dos seguintes erros durante uma sincronização:

- Um objeto sincronizado com o mesmo endereço proxy já existe no seu diretório de Serviços Online da Microsoft.

- Uma mensagem semelhante para UserPrincipalName.

Utilize o novo diagnóstico AADConnect Health para resolver erros duplicados de objetos. Para obter mais informações, consulte [diagnosticar e remediar erros de sincronização de atributos duplicados](https://docs.microsoft.com/azure/active-directory/hybrid/how-to-connect-health-diagnose-sync-errors).
