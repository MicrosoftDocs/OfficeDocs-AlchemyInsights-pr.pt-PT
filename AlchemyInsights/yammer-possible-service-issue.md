# <a name="yammer---possible-service-issue"></a>Yammer - possível problema de serviço

Pode haver um problema temporário em que os utilizadores possam experimentar erros ao aceder ao Yammer. Consulte o [Painel de Saúde do Serviço](https://admin.microsoft.com/AdminPortal/Home#/servicehealth) para ver se isto afeta a sua rede.
