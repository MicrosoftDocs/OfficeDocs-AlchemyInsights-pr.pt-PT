# <a name="workflow-troubleshooting"></a>Resolução de problemas de fluxo de trabalho

Para obter informações sobre a depuração dos fluxos de trabalho do SharePoint, reveja o seguinte:
- Depurar fluxos de trabalho assemelha-se ao das versões anteriores.  Para obter mais informações, consulte [os fluxos de trabalho do Debugging SharePoint](https://docs.microsoft.com/sharepoint/dev/general-development/debugging-sharepoint-server-workflows).
- Para obter mais informações sobre a resolução de problemas, consulte [os erros de validação do fluxo de trabalho do Troubleshoot SharePoint Server](https://docs.microsoft.com/sharepoint/dev/general-development/troubleshooting-sharepoint-server-workflow-validation-errors-in-visio).
