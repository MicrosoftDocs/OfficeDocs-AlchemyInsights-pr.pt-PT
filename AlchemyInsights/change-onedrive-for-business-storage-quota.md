# <a name="change-the-default-onedrive-storage-space-for-your-users"></a>Alterar o espaço de armazenamento onedrive padrão para seus usuários

1. Vá para a [página de armazenamento do centro de administração OneDrive](https://admin.onedrive.com/?v=StorageSettings).
    
2. Insira o valor de armazenamento padrão (em GB) na caixa de **armazenamento padrão** e, em seguida, clique em **Salvar**.
    
Esta configuração de espaço de armazenamento se aplica a todos os usuários para os quais você não definiu limites de armazenamento específicos. Para alterar o espaço de armazenamento para usuários específicos, você precisa usar o Microsoft PowerShell. Para obter informações sobre como fazer isso, veja o espaço de [armazenamento OneDrive dos usuários do Change usando o PowerShell.](https://go.microsoft.com/fwlink/?linkid=866402)
