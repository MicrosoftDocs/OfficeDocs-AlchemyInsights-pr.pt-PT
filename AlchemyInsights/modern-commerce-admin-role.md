# <a name="modern-commerce-administrator-role"></a>Papel de administrador de comércio moderno

Esta função é automaticamente atribuída aos utilizadores que fazem uma compra para lhes dar acesso limitado ao portal de administração para gerir essa compra. Estes utilizadores não são administradores e não têm privilégios elevados.
