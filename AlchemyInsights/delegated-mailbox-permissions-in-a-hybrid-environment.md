# <a name="delegated-mailbox-permissions-in-a-hybrid-environment"></a>Caixa de correio delegadas permissões num ambiente de híbridos

Dependendo da versão do Exchange tiver instalado na sua organização no local, pode querer tomar algumas medidas adicionais para configurar permissões de caixa de correio delegadas entre a organização no local e Exchange Online no Office 365. Para mais informações sobre como configurar permissões delegadas, observe [Configurar o Exchange para suportar a caixa de correio delegadas permissões numa implementação híbrida](https://technet.microsoft.com/library/mt784505%28v=exchg.150%29.aspx).
