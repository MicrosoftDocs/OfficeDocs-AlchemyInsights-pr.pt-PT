# <a name="report-on-file-and-folder-sharing-in-sharepoint-sites"></a>Relatório sobre partilha de ficheiros e pastas em sites sharePoint

Pode criar um ficheiro CSV que detalhe todos os ficheiros, utilizador, permissão e link únicos num determinado site do SharePoint ou na biblioteca OneDrive. Isto pode ajudá-lo a entender como a partilha está a ser usada e se quaisquer ficheiros ou pastas estão a ser partilhados com os hóspedes.

Para obter mais informações, [reporte sobre a partilha de ficheiros e pastas num site do SharePoint](https://docs.microsoft.com/sharepoint/sharing-reports).

> [!NOTE]
> Esta funcionalidade está atualmente implantada para os inquilinos da First Release e irá gradualmente ser lançada para todos os inquilinos da Produção.
