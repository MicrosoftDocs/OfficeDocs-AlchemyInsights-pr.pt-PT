# <a name="about-universal-print"></a>Sobre a Impressão Universal

A Universal Print está incluída nas subscrições comerciais e educativas do Microsoft 365 e do Windows 10, mas também pode ser adquirida como uma subscrição autónoma.

Aprenda a configurar impressoras, dar acesso aos utilizadores e completar o seu primeiro trabalho de impressão na página de Onboarding para a página de Impressão Universal.

Para mais informações, consulte [a Licença Universal Print | Microsoft Docs](https://docs.microsoft.com/universal-print/fundamentals/universal-print-license).
