# <a name="migrate-on-premises-to-sharepoint-online"></a>Migrar no local para o SharePoint Online

Para migrar de uma Fazenda SharePoint no local para o SharePoint Online, pode utilizar a Ferramenta de Migração SharePoint. Para obter informações adicionais sobre a ferramenta, consulte [Migrate para SharePoint Online.](https://go.microsoft.com/fwlink/?linkid=2019574)
