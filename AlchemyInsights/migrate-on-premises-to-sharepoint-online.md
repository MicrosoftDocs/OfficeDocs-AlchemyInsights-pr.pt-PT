# <a name="migrate-on-premises-to-sharepoint-online"></a>Migrar nas instalações para o SharePoint Online

Para migrar de uma SharePoint Farm no local para o SharePoint Online, pode utilizar a Ferramenta de Migração SharePoint. Para obter informações adicionais sobre a ferramenta, consulte [Migrar para SharePoint Online.](https://go.microsoft.com/fwlink/?linkid=2019574)
