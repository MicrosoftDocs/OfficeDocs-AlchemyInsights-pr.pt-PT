# <a name="restore-a-deleted-microsoft-365-group"></a>Restaurar um grupo Microsoft 365 eliminado

Os grupos eliminados são mantidos por 30 dias. Para restaurar um grupo eliminado:
  
1. No [centro de administração Exchange](https://outlook.office365.com/ecp/), selecione **os destinatários** no painel esquerdo e, em seguida, selecione **grupos**. Se um grupo foi eliminado há menos de 30 dias, aparecerá na lista e a coluna STATUS listará a data eliminada.

2. Selecione o grupo e, em seguida, clique em **Restaurar** na barra de comando, ou clique no link "Clique aqui para restaurar" no painel de detalhes.

Para obter mais informações ou para aprender a restaurar grupos usando o PowerShell, consulte [Restaurar um grupo Microsoft 365 eliminado](https://go.microsoft.com/fwlink/?linkid=867802).
