# <a name="diagnostic-to-enable-basic-authentication-for-exchange-online-protocols"></a>Diagnóstico para ativar a autenticação básica Exchange Online protocolos

Ao utilizar este diagnóstico, pode ativar a autenticação Básica para protocolos Exchange Online como POP3, IMAP4, Exchange ActiveSync, Serviços Web Exchange, Livro de Endereços Offline, MAPI, RPC e PowerShell Remoto, que a Microsoft poderá ter desativado recentemente para a sua organização. 

Estamos a enviar comunicações diretas através do Centro de Mensagens para que os inquilinos saibam onde estão prontos para desinstalar a autenticação Básica nos respetivos ambientes devido a não ser utilizada, o que irá ajudar a proteger os respetivos ambientes de riscos de segurança relacionados.
