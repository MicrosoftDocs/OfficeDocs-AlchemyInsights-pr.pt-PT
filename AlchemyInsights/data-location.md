# <a name="data-location"></a>Localização de dados

Mantemos várias cópias dos dados ao longo de centros de dados para redundância. [Ver onde os dados estão localizados](https://office.com/datamaps). Se expandir é a localização dos dados para um novo país ou região, vamos dar-lhe um mês de antecedência.
  
Para obter mais informações sobre segurança, privacidade e conformidade do Office 365, consulte o [Centro de fidedignidade do Office 365](https://products.office.com/business/office-365-trust-center-welcome).
