# <a name="how-to-stop-signing-in-automatically"></a>Como parar, iniciar sessão automaticamente

Para deixar de iniciar sessão automaticamente, desmarque a caixa para **manter a sessão iniciada** na página onde introduz a palavra-passe. 
  
Para obter mais informações, consulte [como iniciar sessão para ou do Outlook.com](https://go.microsoft.com/fwlink/p/?linkid=873113).
