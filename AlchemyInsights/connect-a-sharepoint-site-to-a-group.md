# <a name="connect-a-sharepoint-site-to-a-group"></a>Ligue um Site SharePoint a um Grupo

Utilize o assistente de ligação de grupo para ligar um Site SharePoint a um Grupo Microsoft 365.

1. A partir do site que pretende ligar a um grupo, selecione **Definições (ícone de engrenagem)**.
2. Selecione **Connect para o novo Microsoft 365 Group**.
3. Siga o assistente para completar a ligação.

**Nota:**  Se o Connect to new Microsoft 365 Group não estiver disponível, o site não suportará uma ligação de grupo.

Para mais informações, consulte [Connect to a Office 365 group](https://docs.microsoft.com/sharepoint/dev/transform/modernize-connect-to-office365-group).
