# <a name="manage-recurring-billing"></a>Gerir a faturação periódica

Para gerir periódico de facturação, pode alternar **periódico facturação** ou desactivar.
  
1. No Centro de administração, vá para a **Facturação** \> página de [subscrições](https://go.microsoft.com/fwlink/p/?linkid=842054) .

2. Escolha a subscrição para o qual pretende gerir facturação periódica.

3. Mude a alternância de **Facturação periódico** para **Activar** ou **Desactivar**.

Para mais informações, consulte [Activar periódico facturação ou desactivar](https://docs.microsoft.com/en-us/office365/admin/subscriptions-and-billing/renew-your-subscription?view=o365-worldwide#turn-recurring-billing-off-or-on).
