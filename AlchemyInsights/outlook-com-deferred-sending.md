# <a name="delay-or-schedule-sending-email-messages"></a>Atraso ou agenda enviar mensagens de correio eletrónico

Desculpe, mas não pode adiar ou agendar o envio de mensagens de correio eletrónico numa aplicação móvel ou navegador web. Para tal, utilize o [Outlook para Windows ou Outlook para Mac](https://products.office.com/outlook/email-and-calendar-software-microsoft-outlook).

**Desfazer enviar**

Pode alterar as definições de Outlook.com para atrasar o envio de uma mensagem por um máximo de 10 segundos. Pode cancelar o envio da mensagem durante este período. Para ativar esta funcionalidade, abra as definições de [envio do Desfazer.](https://outlook.live.com/mail/options/mail/messageContent/undoSend)

Saiba mais sobre a recolha de [e-mails em Outlook.com.](https://support.office.com/article/c069ddde-5282-4085-8f4c-d7b133324f8a?wt.mc_id=Office_Outlook_com_Alchemy)
