# <a name="unable-to-delete-photos-from-files"></a>Não é possível eliminar fotografias a partir de ficheiros

Para eliminar os ficheiros ou fotografias:
  
1. Vá para a pasta **a receber** ou **Itens enviados** e eliminar a mensagem de correio electrónico que estão anexadas. 
    
2. Remova o item (NS) de **itens eliminados**. 
    
Para mais informações, leia [como eliminar ficheiros ou fotografias em Outlook.com](https://support.office.com/article/bae0531f-040f-4c42-90b9-786ca718c16d.aspx).
