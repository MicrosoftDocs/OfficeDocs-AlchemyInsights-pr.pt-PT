# <a name="connecting-powershell-scripts-to-exchange-online"></a>Ligar scripts PowerShell a Exchange Online

A Autenticação Básica no Exchange Online será depreciada, e o caminho a seguir é conectar-se utilizando a autenticação baseada em certificados para scripts e tarefas não atadas. Para saber mais, consulte [a autenticação apenas da App para scripts não acompanhados no módulo EXO V2](https://docs.microsoft.com/powershell/exchange/app-only-auth-powershell-v2).
