# <a name="connect-to-exchange-online-powershell-when-mfa-is-enabled"></a>Ligue-se à Exchange Online PowerShell quando o MFA estiver ativado

Se a sua conta tiver a autenticação multi-factor (MFA) ativada, tem de seguir estas instruções para se ligar à Exchange Online PowerShell: Connect to Exchange Online PowerShell utilizando a [autenticação de vários fatores](https://docs.microsoft.com/powershell/exchange/exchange-online/connect-to-exchange-online-powershell/mfa-connect-to-exchange-online-powershell).

**Nota:** Mesmo que tenha ligado ao Exchange Online PowerShell no passado utilizando [as instruções regulares de ligação,](https://docs.microsoft.com/powershell/exchange/exchange-online/connect-to-exchange-online-powershell/connect-to-exchange-online-powershell)tem de utilizar as instruções de ligação MFA depois de o MFA ter sido ativado para a sua conta.
