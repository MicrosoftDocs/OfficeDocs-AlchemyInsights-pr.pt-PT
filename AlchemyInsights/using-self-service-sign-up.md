# <a name="using-self-service-sign-up-in-your-organization"></a>Usando o self-service inscreva-se na sua organização

Com o self-service os utilizadores podem inscrever-se para utilizar os serviços pagos pela sua subscrição.

- Para controlar se os utilizadores podem inscrever-se para subscrições de self-service, utilize o cmdlet PowerShell [set-MsolCompanySettings](https://docs.microsoft.com/powershell/module/msonline/set-msolcompanysettings?view=azureadps-1.0) com o parâmetro **AllowAdHocSubscriptions.** Para mais informações, veja [como controlo as definições de self-service?](https://docs.microsoft.com/microsoft-365/commerce/subscriptions/self-service-purchase-faq?view=o365-worldwide)

- Para ver todos os programas disponíveis, consulte os [programas de self-service disponíveis.](https://docs.microsoft.com/microsoft-365/admin/misc/self-service-sign-up?view=o365-worldwide#available-self-service-programs)

Para obter informações completas, consulte [utilizar a inscrição de self-service na sua organização.](https://docs.microsoft.com/microsoft-365/admin/misc/self-service-sign-up?view=o365-worldwide)
