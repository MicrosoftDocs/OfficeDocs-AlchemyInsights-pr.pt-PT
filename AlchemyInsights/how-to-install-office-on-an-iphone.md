# <a name="how-to-install-office-on-an-iphone"></a>Como instalar o Office num iPhone

- O utilizador toque no link de [descarregamento do](https://support.office.com/article/9df6d10c-7281-4671-8666-6ca8e339b628?wt.mc_id=Alchemy_ClientDIA) seu dispositivo móvel, para Word, Excel, PowerPoint e OneNote.

- Mande o utilizador iniciar a aplicação, tocar **em Iniciar sessão** e inserir as suas credenciais de trabalho ou escola.
