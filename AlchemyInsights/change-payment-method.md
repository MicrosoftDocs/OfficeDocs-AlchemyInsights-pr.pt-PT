# <a name="change-payment-method"></a>Alterar o método de pagamento

Pode alterar o método de pagamento do centro de administração microsoft **365**.
  
1. No centro de administração, vá à página de **Billing** \> **[Your products.](https://go.microsoft.com/fwlink/p/?linkid=842054)**

2. Encontre a subscrição para a qual pretende alterar o método de pagamento. Ao lado do método de pagamento, clique em **Editar**.

3. Siga os passos no ecrã para completar a alteração.

Para saber mais, consulte Alterar o seu método de [pagamento para o Microsoft 365 para negócios](https://docs.microsoft.com/microsoft-365/commerce/billing-and-payments/change-payment-method).
