# <a name="change-payment-method"></a>Alterar o método de pagamento

Pode verificar e alterar o método de pagamento configurado no centro de administração da Microsoft 365. Selecione a subscrição que pretende verificar. O método de pagamento configurado para a subscrição aparece no âmbito **da Faturação.**
  
1. No centro de administração, aceda à página **Faturação** \> [Produtos e Serviços](https://go.microsoft.com/fwlink/p/?linkid=842054).

2. Encontre a subscrição para a qual pretende alterar o método de pagamento. Ao lado do método de pagamento, **selecione Editar**.

3. Siga os passos no ecrã para completar a alteração.

Para saber mais, consulte Alterar o seu método de [pagamento para o Microsoft 365 para negócios](https://docs.microsoft.com/office365/admin/subscriptions-and-billing/change-payment-method)
