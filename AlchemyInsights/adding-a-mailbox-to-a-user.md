# <a name="adding-a-mailbox-to-a-user"></a>Adicionar uma caixa de correio a um utilizador

Para adicionar uma caixa de correio a um utilizador já criado ou a um novo utilizador, o utilizador deve ser atribuído a uma licença para o Exchange Online. Se o utilizador já tiver licenças atribuídas, selecione Exchange Online na secção Apps na página de licença para iniciar o processo para criar uma caixa de correio para o utilizador. Para obter mais informações, consulte [atribuir licenças aos utilizadores.](https://docs.microsoft.com/microsoft-365/admin/manage/assign-licenses-to-users)
