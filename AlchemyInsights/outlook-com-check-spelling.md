# <a name="use-your-browser-to-check-spelling"></a>Utilize o browser para verificar a ortografia

Apesar de Outlook.com não tem uma correcção ortográfica verifique função, muitos web browsers verificar automaticamente a ortografia, incluindo Microsoft Edge, Internet Explorer 10 e versões posteriores e versões actuais do Firefox e Safari cromado. Verifique as opções para o web browser obter mais informações sobre como verificar a ortografia.
