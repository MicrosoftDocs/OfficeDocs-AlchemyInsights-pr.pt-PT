# <a name="use-your-browser-to-check-spelling"></a>Use o seu navegador para verificar a ortografia

Embora Outlook.com não tenha uma função de verificação ortográfica, muitos navegadores da Web verificam automaticamente a sua ortografia, incluindo as versões Microsoft Edge, Internet Explorer 10 e posterior, e as versões atuais do Firefox, Chrome e Safari. Consulte as opções para o seu navegador web para saber mais sobre como verificar a ortografia.
