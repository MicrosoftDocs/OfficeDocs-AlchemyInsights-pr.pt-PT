# <a name="creating-a-team-from-an-existing-team"></a>Criar uma equipa a partir de uma equipa existente

Alguns clientes relataram um problema com a criação de uma equipa de uma equipa existente (ou clonagem de uma equipa) quando optam por copiar os membros da equipa. Para contornar esta questão, não copie os membros como parte do processo de criação e adicione membros após a criação da equipa.

Para mais informações, consulte [Criar uma equipa de uma equipa existente.](https://support.microsoft.com/office/create-a-team-from-an-existing-team-f41a759b-3101-4af6-93bd-6aba0e5d7635)
