# <a name="disable-access-to-yammer"></a>Desabilitar o acesso ao Yammer

A solução para desabilitar o acesso do Yammer é bloquear e cancelar a atribuição de licenças do Yammer para usuários do Office 365. Por favor, consulte o seguinte artigo para obter mais informações.
  
- [Desativar o acesso do Yammer para usuários do Office 365](https://docs.microsoft.com/yammer/manage-yammer-users/turn-off-user-access)
