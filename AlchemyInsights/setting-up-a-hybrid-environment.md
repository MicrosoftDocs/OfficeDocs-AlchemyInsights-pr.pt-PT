# <a name="setting-up-a-hybrid-environment"></a>Criação de um ambiente híbrido

Para permitir que os seus utilizadores trabalhem em conjunto, independentemente de os seus dados forem armazenados em servidores no local ou no Microsoft 365, poderá ter de configurar um ambiente híbrido. Para obter uma visão geral das suas opções, confira a [integração do Microsoft 365 com ambientes no local.](https://docs.microsoft.com/office365/enterprise/office-365-integration)
