# <a name="setting-up-a-hybrid-environment"></a>Configurar um ambiente de híbridos

Para permitir que os utilizadores trabalham facilmente em conjunto independentemente se os dados são armazenados em servidores no local ou no Office 365, poderá pretender configurar um ambiente de híbridos. Para obter uma descrição geral das opções, consulte [integração do Office 365 com ambientes no local](https://docs.microsoft.com/office365/enterprise/office-365-integration).
