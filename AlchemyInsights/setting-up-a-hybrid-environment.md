# <a name="setting-up-a-hybrid-environment"></a>Configurar um ambiente de híbridos

Para permitir que os utilizadores trabalham facilmente em conjunto independentemente se os dados são armazenados em servidores no local ou no Office 365, poderá pretender configurar um ambiente de híbridos. Para obter uma descrição geral das opções, consulte [integração do Office 365 com ambientes no local](https://support.office.com/article/263faf8d-aa21-428b-aed3-2021837a4b65.aspx).
