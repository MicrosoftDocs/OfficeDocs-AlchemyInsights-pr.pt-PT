# <a name="setting-up-a-hybrid-environment"></a>Configurando um ambiente híbrido

Para permitir que seus usuários trabalhem perfeitamente juntos, não importa se seus dados são armazenados em servidores locais ou no Office 365, talvez seja necessário configurar um ambiente híbrido. Para obter uma visão geral das suas opções, verifique a [integração do Office 365 com ambientes locais](https://docs.microsoft.com/office365/enterprise/office-365-integration).
