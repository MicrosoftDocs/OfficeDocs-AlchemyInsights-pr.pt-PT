# <a name="send-email-from-an-alias-or-secondary-address"></a>Envie e-mail de um pseudónimo ou endereço secundário

O Microsoft 365 não suporta atualmente enviar e-mails de um pseudónimo ou endereço de e-mail secundário. Os pseudónimos de e-mail podem ser usados apenas para receber e-mails. Se quiser enviar e-mail de outro endereço de e-mail, tem de criar uma caixa de correio partilhada. A caixa de correio partilhada irá dar-lhe outro endereço de e-mail, sem precisar de outra licença.
  
- [Vá aqui para criar uma caixa de correio partilhada](https://portal.office.com/AdminPortal/Home#/AssistedGuide/addemailoptions)

Pode também ler o artigo da [Create a shared mailbox](https://docs.microsoft.com/office365/admin/email/create-a-shared-mailbox) que lhe dirá tudo o que precisa de saber sobre caixas de correio partilhadas, incluindo como usá-las.
