# <a name="using-a-shared-mailbox-on-a-mobile-device"></a>Utilizar uma caixa de correio partilhada num dispositivo móvel

É possível aceder uma caixa de correio partilhada da aplicação de Outlook que está instalada no dispositivo móvel. Quer esta funcionalidade? Permitir que a equipa de engenharia de Outlook saber. [Direito de voto aqui](https://go.microsoft.com/fwlink/?linked=862116)!
  
Só pode utilizar uma caixa de correio partilhada num dispositivo móvel abrindo o browser, o início de sessão no Office 365 e, em seguida, vai para o Outlook na web. Para obter instruções, consulte [Adicionar uma caixa de correio partilhada no Outlook na web](https://support.office.com/article/add-a-shared-mailbox-to-outlook-on-the-web-98b5a90d-4e38-415d-a030-f09a4cd28207).
  
Se for necessário aceder a caixas de correio partilhadas utilizando a aplicação Outlook num dispositivo móvel, considere a criação de um grupo de 365 do Office em vez de utilizar uma caixa de correio partilhada. Para mais informações, consulte a [Comparar grupos](https://support.office.com/article/758759ad-63ee-4ea9-90a3-39f941897b7d.aspx).
  
Obter mais informações: [obter ajuda na aplicação para o Outlook para iOS e Android](https://support.office.com/article/Get-in-app-help-for-Outlook-for-iOS-and-Android-218a22d1-9fa5-4889-b689-de1c63493243)
