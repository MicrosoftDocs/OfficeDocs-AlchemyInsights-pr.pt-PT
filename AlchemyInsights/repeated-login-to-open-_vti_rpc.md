# <a name="repeated-login-to-open-vtirpc"></a>Início de sessão repetido para abrir _vti_rpc

Quando trabalha com ficheiros de OneDrive, utilizando o Office 2016, poderá visualizar a mensagem: "Indique-na conta que pretende utilizar para abrir _vti_rpc." Este problema foi corrigido na versão 1708 (16.0.8431.2094). Para corrigir este problema, actualize para a versão mais recente do Office. Seleccione o **ficheiro** \> **conta** \> **actualizar as opções da** \> **Actualizar agora**.
