# <a name="get-started-with-using-microsoft-search-usage-reports"></a>Começar a usar relatórios de utilização do Microsoft Search

Os relatórios de utilização do Microsoft Search fornecem informações sobre como a pesquisa está a funcionar na sua organização. Estas informações vão ajudá-lo a encontrar conteúdo e a tornar a pesquisa uma experiência mais útil e deliciosa para os utilizadores das suas organizações.

Para saber mais, consulte [os Relatórios de Utilização da Microsoft Search.](https://go.microsoft.com/fwlink/?linkid=2152048)
