# <a name="open-and-use-a-shared-mailbox-in-outlook-on-the-web"></a>Abrir e utilizar uma caixa de correio partilhada no Outlook na web

Partilhado caixas de correio que tem acesso para não são automaticamente apresentadas no Outlook na web (anteriormente conhecido como o Outlook Web App).

Pode mostrar a caixa de correio partilhada e a caixa de correio principal juntos ou abrir a caixa de correio partilhada numa janela separada do browser. Para receber notificações de novas mensagens para a caixa de correio partilhada, tem de abrir a caixa de correio partilhada numa janela separada do browser. Para mais informações, consulte [Abrir e utilizar uma caixa de correio partilhada no Outlook Web App](https://support.office.com/article/BC127866-42BE-4DE7-92AE-1EF2F787FD5C).
