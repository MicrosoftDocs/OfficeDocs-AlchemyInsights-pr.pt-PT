# <a name="delete-a-contact"></a>Eliminar um contacto

1. Seleccione o ícone de **pessoas** no canto inferior esquerdo do ecrã.

2. Seleccione o contacto que pretende eliminar e seleccione **Eliminar**.

    Se não conseguir seleccionar a **Eliminar**, poderá ser o contacto do Skype ou uma conta de redes sociais ligadas. Para eliminar o contacto, vá para o Skype ou da conta onde se encontra o contacto.

3. Seleccione **Eliminar** para confirmar.
