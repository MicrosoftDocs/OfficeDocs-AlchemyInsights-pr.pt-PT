# <a name="set-up-spf"></a>Configurar SPF

Atualize os seus registos DNS para que possa utilizar um Quadro de Política de Remetente (SPF) com o seu domínio personalizado para validar o e-mail de saída enviado a partir do seu domínio personalizado. Isto ajuda a evitar que outros "falsificam" correio como se fosse enviado do seu domínio.
  
- [Criar SPF para ajudar a prevenir a falsificação](https://docs.microsoft.com/microsoft-365/security/office-365-security/set-up-spf-in-office-365-to-help-prevent-spoofing)

- [Ajude a configurar todos os registos DNS](https://docs.microsoft.com/microsoft-365/admin/get-help-with-domains/create-dns-records-at-any-dns-hosting-provider)
