# <a name="change-billing-country-or-region"></a>Alterar país ou região de faturação

Não é possível alterar o país/região de Contas de Faturação de   >   Faturação no Microsoft 365 administração. Terá de se inscrever para uma nova conta e escolher o país/região pretendido.
