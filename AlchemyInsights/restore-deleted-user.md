# <a name="restore-a-deleted-user"></a>Restaurar um utilizador eliminado

Restaurar um utilizador utilizando estes passos:
  
1. Ir aos [utilizadores \> Eliminar utilizadores](https://admin.microsoft.com/adminportal/home#/deletedusers).

2. Escolha o utilizador e, em seguida, **selecione Restaurar.**

3. Siga as instruções para definir a palavra-passe.

4. Clique **enviar e-mail e fechar,** e está feito!

Não foi fácil? Para obter mais detalhes e passos com imagens de ecrã, confira este artigo: [Restaurar um utilizador](https://docs.microsoft.com/microsoft-365/admin/add-users/restore-user). Se perceber que precisa de restaurar a caixa de correio do utilizador, consulte: [Eliminar ou restaurar caixas de correio do utilizador no Exchange Online](https://docs.microsoft.com/exchange/recipients-in-exchange-online/delete-or-restore-mailboxes).
