# <a name="use-your-browser-to-check-spelling"></a>Utilize o browser para verificar a ortografia

Não existe nenhuma opção de verificação ortográfica em Outlook.com. Para verificar a ortografia, tem de utilizar o web browser. A verificação ortográfica está disponível no Microsoft Edge, Internet Explorer 10 e versões posteriores e versões actuais do Firefox e Safari cromado. Verifique as opções para o web browser obter mais informações sobre como verificar a ortografia.
