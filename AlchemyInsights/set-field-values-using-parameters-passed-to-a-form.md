# <a name="set-field-values-using-parameters-passed-to-a-form---lookup-fields"></a>Definir valores de campo usando parâmetros passados para um formulário - campos de procura

Defina o valor para campos de procuração e exemplo: Use Xrm.Utility.openEntityForm para abrir uma nova janela. Não se pode definir os valores para a lista de festas ou para as procuras.

Para obter mais informações, consulte [os valores definidos para os campos de procuração](https://docs.microsoft.com/previous-versions/dynamicscrm-2016/developers-guide/gg334375(v=crm.8)#set-values-for-lookup-fields).
