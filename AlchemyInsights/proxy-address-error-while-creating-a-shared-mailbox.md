# <a name="proxy-address-error-while-creating-a-shared-mailbox"></a>Erro de endereço proxy ao criar uma caixa de correio partilhada

Se tentou criar uma caixa de correio partilhada e recebeu este erro "O endereço proxy "SMTP:alias@domain.com" já está a ser usado...", o endereço de e-mail que escolheu já é tomado por outra caixa de correio da sua organização.
  
É necessário encontrar o utilizador, grupo ou caixa de correio partilhada que tenha este endereço de e-mail e eliminá-lo ou alterar o seu endereço de e-mail. Em seguida, pode criar uma caixa de correio partilhada com o endereço de e-mail libertado. Use a Pesquisa na página Inicial para encontrá-la.
  
Caso não pretenda eliminar o endereço de e-mail existente, escolha um novo endereço de e-mail para a sua caixa de correio partilhada.
