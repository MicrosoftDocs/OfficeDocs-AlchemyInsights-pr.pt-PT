# <a name="learn-about-microsoft-365-groups"></a>Conheça os Grupos Microsoft 365

O Microsoft 365 Groups é um serviço que funciona com as ferramentas do Office 365 que já utiliza, para que possa colaborar com os seus colegas de equipa ao escrever documentos, criar folhas de cálculo, trabalhar em planos de projeto, agendar reuniões ou enviar e-mails. Para obter mais informações, consulte [Saiba mais sobre os Grupos Microsoft 365.](https://support.office.com/article/b565caa1-5c40-40ef-9915-60fdb2d97fa2)
