# <a name="onedrive-common-issues-and-resolutions"></a>OneDrive questões comuns e resoluções

Aqui estão alguns tópicos frequentemente solicitados sobre questões OneDrive:

- **Configuração OneDrive Sync:** [Sincronizar seus arquivos OneDrive for Business](https://go.microsoft.com/fwlink/?linkid=533375) ou [sincronizar arquivos SharePoint](https://go.microsoft.com/fwlink/?linkid=871666) com o cliente de sincronização OneDrive para Windows.  Usando um Mac? Você pode [sincronizar arquivos com O OneDrive](https://support.office.com/article/Sync-files-with-the-OneDrive-sync-client-on-Mac-OS-X-d11b9f29-00bb-4172-be39-997da46f913f) com ele também.

- **Solucionado de problemas OneDrive Sync:** Se o OneDrive não estiver sincronizando arquivos da maneira que você espera, você pode [solucionar problemas](https://go.microsoft.com/fwlink/?linkid=866431) de sincronização do OneDrive para Windows. Usando um Mac? Você pode [solucionar problemas](https://support.office.com/article/fix-onedrive-sync-problems-on-a-mac-af3012d7-13ec-4ac9-bbb1-ebcd2a0cd756) de sincronização para ele também.
- **Limites onedrive:** saiba mais sobre as [restrições e limitações](https://support.office.com/article/Invalid-file-names-and-file-types-in-OneDrive-OneDrive-for-Business-and-SharePoint-64883a5d-228e-48f5-b3d2-eb39e07630fa) que se aplicam a nomes de arquivos e tipos de arquivos ao sincronizar com o OneDrive.
- **Armazenamento OneDrive:** Veja como alterar o espaço de [armazenamento OneDrive de um usuário específico](https://docs.microsoft.com/onedrive/change-user-storage) ou gerenciar seu armazenamento [OneDrive for Business](https://support.office.com/article/Manage-your-OneDrive-for-Business-storage-31519161-059C-4764-B6F8-F5CD29F7FE68).
- **Registros**de auditoria: Veja como você pode usar a pesquisa de registro de [auditoria](https://docs.microsoft.com/office365/securitycompliance/search-the-audit-log-in-security-and-compliance#search-the-audit-log) para verificar a atividade do arquivo (s), pasta (s), usuário (s), ou permissões.
