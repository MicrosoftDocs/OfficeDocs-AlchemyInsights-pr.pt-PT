# <a name="activate-a-new-subscription-with-a-product-key"></a>Activar uma nova subscrição com uma chave de produto

1. Para activar uma nova subscrição com a chave de produto, vá para o [https://office.com/setupkey](https://office.com/setupkey) página. 
    
2. Introduza a chave de produto do Office 365 e siga as instruções para activar a subscrição. Se tiver problemas com a activação, dar saída desta [lista de erros de chave de produto comuns e soluções](https://support.office.com/article/88d337ab-e7b1-43eb-a25e-7d6204e91099).
