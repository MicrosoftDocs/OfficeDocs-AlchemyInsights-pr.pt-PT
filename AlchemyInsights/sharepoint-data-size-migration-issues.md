# <a name="issues-while-migrating-data-to-sharepoint-online"></a>Problemas durante a migração de dados para o SharePoint Online

Afigura-se que estiver a migrar mais de 100TB de dados online do SharePoint. Siga os passos abaixo, para a Microsoft pode ajudá-lo logo que possível. 

1. Seleccione o **pedido de serviço novo**e, em seguida, **Pedir novo serviço**. 
2. Deixe o título e descrição como **migração do SharePoint mais de 100 TB**.
3. Depois do "ticket" tiver sido apresentado, actualize-as com as seguintes informações: 

    - Tamanho estimado da sua migração.
    - Uma estimativa de quando pretende iniciar e concluir a migração.
    - Descrever a qual está a migrar o conteúdo, tal como o SharePoint Server, caixa, GDrive, partilhas de ficheiros, etc...
