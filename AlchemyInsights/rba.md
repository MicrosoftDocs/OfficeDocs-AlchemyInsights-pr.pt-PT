# <a name="about-room-mailboxes"></a>Sobre caixas de correio de quartos

Uma caixa de correio de sala é uma caixa de correio de recursos que é atribuída a um local físico, como uma sala de conferências, um auditório ou uma sala de treino. Para obter mais informações sobre caixas de correio de salas, consulte [Criar e gerir caixas de correio de quartos.](https://go.microsoft.com/fwlink/p/?linkid=717533)
