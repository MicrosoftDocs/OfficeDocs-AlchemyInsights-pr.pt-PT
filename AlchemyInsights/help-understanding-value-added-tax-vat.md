# <a name="help-understanding-value-added-tax-vat"></a>Ajudar a compreender o imposto sobre o valor acrescentado (IVA)

Se estiver num país que usa o imposto sobre o valor acrescentado (IVA), como o Reino Unido, a Microsoft Invoicing calcula o IVA para si e aplica o valor às suas faturas.
  
Por razões fiscais, deve também especificar o seu número de registo de IVA nas definições para Faturação.
  
Não pode adicionar ou remover as taxas de IVA, mas pode alterar a taxa de IVA padrão em Definições. Por predefinição, a Faturação utiliza um serviço para validar os números de registo do IVA, mas recomendamos que leia a isenção de responsabilidade do [VIES](https://go.microsoft.com/fwlink/?LinkID=841741) publicada pela Comissão Europeia. Se optar por desativar o serviço, deve verificar todos os números de registo do IVA.
