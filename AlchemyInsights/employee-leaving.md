# <a name="block-access-to-a-former-employee"></a>Bloquear o acesso a um ex-funcionário

Tem um funcionário de saída e precisa bloquear o acesso a dados e e-mail? Faça o seguinte.
  
1. No centro de administração, vá à página **utilizadores** \> [ativos utilizadores.](https://go.microsoft.com/fwlink/p/?linkid=834822)

2. Selecione o nome do utilizador que pretende bloquear.

3. Sob o nome do utilizador, selecione **Bloquear este utilizador**.

4. Escolha **bloquear o utilizador de iniciar sessão em** \> **'Guardar' alterações**.

**Nota:** A inscrição para algumas aplicações pode não ser imediata. Acompanhe mais ações neste artigo: [Remova um ex-funcionário](https://docs.microsoft.com/office365/admin/add-users/remove-former-employee)
