# <a name="create-a-new-folder-in-outlookcom"></a>Criar uma nova pasta em Outlook.com

1. **pastas** no painel de navegação, selecione **nova pasta.**

2. Digite um nome para a pasta e pressione Enter.

Para saber mais, veja [criar uma pasta em Outlook.com](https://support.office.com/article/5fa8de74-3562-4729-ac1d-5599f470b25a?wt.mc_id=Office_Outlook_com_Alchemy).
