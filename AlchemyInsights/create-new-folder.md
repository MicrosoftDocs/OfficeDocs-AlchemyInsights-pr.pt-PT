# <a name="create-a-new-folder-in-outlookcom"></a>Criar uma nova pasta na Outlook.com

1. Em **pastas** no painel de navegação, seleccione **nova pasta**.

2. Escreva um nome para a pasta e prima Enter.

Para obter mais informações, consulte [criar uma pasta no Outlook.com](https://support.office.com/article/5fa8de74-3562-4729-ac1d-5599f470b25a?wt.mc_id=Office_Outlook_com_Alchemy).
