# <a name="set-microsoft-edge-as-the-default-browser-on-a-windows-10-device"></a>Desafine o Microsoft Edge como o navegador padrão num dispositivo Windows 10

Faça o seguinte:

1. Selecione o botão **Iniciar** e, em seguida, **introduza aplicações predefinidos**.

1. Nos resultados da pesquisa, selecione **aplicações Predefinidos**.

1. No **navegador Web,** selecione **Microsoft Edge**.
