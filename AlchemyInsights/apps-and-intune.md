# <a name="apps-and-intune"></a>Apps e Intune

Para uma visão geral de como o Intune pode ajudar a gerir apps, comece com [o que é a gestão de aplicações da Microsoft Intune?](https://docs.microsoft.com/mem/intune/apps/app-management)

Para obter mais informações sobre o upload de apps para Intune e implementá-las nos seus dispositivos, reveja [o Quickstart: Add and assigna uma aplicação de cliente.](https://docs.microsoft.com/mem/intune/apps/quickstart-add-assign-app)

Para proteger os dados corporativos dentro das aplicações móveis, consulte o [Quickstart: Criar e atribuir uma política de proteção de aplicações](https://docs.microsoft.com/mem/intune/apps/quickstart-create-assign-app-policy).
