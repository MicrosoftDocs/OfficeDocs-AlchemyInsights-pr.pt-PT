# <a name="how-to-enable-microsoft-to-do"></a>Como habilitar o Microsoft to-do

Para usar o Microsoft to-do, um administrador deve habilitar o aplicativo na conta de cada usuário. Vá para a [página &amp; ](https://portal.office.com/adminportal/home#/Settings/ServicesAndAddIns) de suplementos de serviços da página de configurações de administrador do Office 365 para habilitar a fazer para cada usuário. 
  
O Microsoft to-do usa recursos no Exchange Online para sincronizar e armazenar itens de atividades dos usuários. Para usar o Microsoft to-do, um usuário precisa de uma caixa de correio válida do Exchange Online.
  
Para obter informações adicionais, consulte [Administração do Microsoft to-do preview](https://support.office.com/article/490c1a8c-2333-4952-8125-841afadb9620.aspx).
