# <a name="deploy-microsoft-365"></a>Implementar a Microsoft 365

O [consultor de implementação microsoft 365](https://go.microsoft.com/fwlink/?linkid=2072646) fornece orientação aos clientes comerciais ao configurar ferramentas de produtividade, políticas de segurança e capacidades de gestão de dispositivos. Com um Microsoft 365 Business Premium ou Microsoft 365 para subscrição empresarial, pode utilizar este consultor para configurar e configurar os dispositivos da sua organização.

Receberá orientação e acesso a recursos para ativar os seus serviços na nuvem, atualizar dispositivos para a versão mais recente do Windows 10 e juntar dispositivos ao Azure Ative Directory (Azure AD), tudo num local central.
