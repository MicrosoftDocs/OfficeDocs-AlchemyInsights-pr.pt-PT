# <a name="webresource-visibility-property"></a>Propriedade webresource visibilidade

Tem de selecionar a opção **Visible by predefinição** para que o recurso web apareça no formulário e esteja disponível para os utilizadores. Para obter mais informações, consulte [as propriedades de recursos web para formulários de aplicações orientados por modelos.](https://docs.microsoft.com/powerapps/maker/model-driven-apps/web-resource-properties-legacy)
