# <a name="cant-download-files-in-chrome"></a>Não é possível transferir ficheiros cromado

**Problema** Estamos conhecimento de um problema que impede que alguns clientes conseguem transferir ficheiros quando utilizar o browser do Google Chrome. 

**Solução alternativa** Utilize o [Limite da Microsoft](https://www.microsoft.com/en-gb/windows/microsoft-edge) ou um web browser.
Mais informações sobre como [Transferir ficheiros e anexos de Outlook.com](https://support.office.com/article/8d7c1ea7-4e5f-44ce-bb6e-c5fcc92ba9ab).
