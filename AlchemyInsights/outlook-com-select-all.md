# <a name="how-to-select-all-in-outlookcom"></a>Como "selecionar tudo" em Outlook.com

Só é possível selecionar todas as mensagens se todas as mensagens estiverem visíveis. Isto é para evitar ações acidentais em mensagens que não consegue ver.

**CTRL + clique** seleciona várias mensagens individuais.

**SHIFT + clique** selecione todas as mensagens entre a primeira e a última mensagem selecionada.
