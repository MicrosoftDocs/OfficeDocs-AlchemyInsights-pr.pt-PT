# <a name="sharepoint-online-calendar-overlay"></a>Sobreposição do calendário online do SharePoint

Pode sobrepor-se a outro calendário sharePoint desde que ambos os calendários do SharePoint estejam na mesma coleção de sites. A capacidade de utilizar a sobreposição do calendário num calendário do SharePoint Online ou do SharePoint Server que é recuperado do Exchange Online não é suportada.

O SharePoint Online oferece uma experiência moderna com calendários de grupo microsoft 365. Para saber mais, consulte [Use a parte web do calendário do grupo.](https://support.microsoft.com/en-us/office/use-the-group-calendar-web-part-eaf3c04d-5699-48cb-8b5e-3caa887d51ce)
