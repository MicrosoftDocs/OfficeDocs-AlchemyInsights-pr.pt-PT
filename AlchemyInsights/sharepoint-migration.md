# <a name="migrate-options-to-sharepoint-online"></a>Migrar opções para SharePoint Online

Existem diferentes opções disponíveis para migrar o conteúdo para sharepoint on-line, dependendo do tamanho e quantidade de arquivos que você precisa mover, por favor, veja uma lista de opções [localizadas aqui](https://docs.microsoft.com/sharepointmigration/migrate-to-sharepoint-online).

Para mais informações sobre a migração de conteúdo, visite os links abaixo.

- [Ferramenta de migração de pontos de compartilhamento](https://docs.microsoft.com/sharepointmigration/introducing-the-sharepoint-migration-tool)

- [Comece com o Gerente de Migração](https://docs.microsoft.com/sharepointmigration/mm-get-started)

- [Sharepoint Online e ODB Migration Speed](https://docs.microsoft.com/sharepointmigration/sharepoint-online-and-onedrive-migration-speed)

- [Evite ser estrangulado ou bloqueado no SharePoint Online](https://docs.microsoft.com/sharepoint/dev/general-development/how-to-avoid-getting-throttled-or-blocked-in-sharepoint-online)

- [Ferramenta de Avaliação de Migração sharepoint (SMAT)](https://www.microsoft.com/download/details.aspx?id=53598&amp;751be11f-ede8-5a0c-058c-2ee190a24fa6=True)

**Nota:** Atualmente, a ferramenta SharePoint Migration só suporta migrações do SharePoint 2010 e 2013. Versão 2016 ou 2019 não são suportados neste momento.
