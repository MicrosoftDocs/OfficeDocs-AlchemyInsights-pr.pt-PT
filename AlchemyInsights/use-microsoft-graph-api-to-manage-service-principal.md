# <a name="use-microsoft-graph-api-to-manage-service-principal"></a>Utilize a Microsoft Graph API para gerir o principal de serviço

Com a [API principal](https://docs.microsoft.com/graph/api/resources/serviceprincipal)de serviço, pode gerir programáticamente casos de aplicações e controlar o que uma aplicação pode fazer dentro do seu inquilino.

Para obter mais informações sobre a relação entre aplicações e principais serviços, consulte [aplicações e objetos principais de serviço no Azure Ative Directory](https://docs.microsoft.com/azure/active-directory/develop/app-objects-and-service-principals).
