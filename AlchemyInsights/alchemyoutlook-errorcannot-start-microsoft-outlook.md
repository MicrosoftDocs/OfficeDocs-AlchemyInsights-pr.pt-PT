# <a name="outlook-error-cannot-start-microsoft-outlook"></a>Erro do Outlook: "Não é possível iniciar o Microsoft Outlook"

Está a ter problemas ao iniciar o Outlook de 2016, o Outlook 2013 ou o Outlook 2010 ou recepção "não é possível iniciar o Microsoft Outlook. Não é possível abrir a janela do Outlook"mensagens de erro?
  
Em primeiro lugar, utilize os passos de resolução de problemas [neste](https://support.office.com/en-us/article/I-can-t-start-Microsoft-Outlook-2016-2013-or-2010-or-receive-the-error-Cannot-start-Microsoft-Office-Outlook-Cannot-open-the-Outlook-Window-d1f69da6-b333-4650-97bf-4d77bd7abb85) artigo para ajudar a resolver estes problemas de arranque. 
  
Também pode executar a [ferramenta de diagnóstico avançadas](https://aka.ms/SaRA-OutlookAdvDiagnostics) no computador afectado.
