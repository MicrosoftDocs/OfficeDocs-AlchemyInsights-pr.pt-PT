# <a name="outlook-error-cannot-start-microsoft-outlook"></a>Erro do Outlook: Não é possível iniciar o Microsoft Outlook

Está com dificuldades em iniciar o Outlook 2016, Outlook 2013 ou Outlook 2010, ou receber "Não é possível iniciar o Microsoft Outlook. Não é possível abrir as mensagens de erro da janela Outlook?
  
Em primeiro lugar, use os passos de resolução de problemas [neste artigo](https://support.office.com/article/I-can-t-start-Microsoft-Outlook-2016-2013-or-2010-or-receive-the-error-Cannot-start-Microsoft-Office-Outlook-Cannot-open-the-Outlook-Window-d1f69da6-b333-4650-97bf-4d77bd7abb85) para ajudar a resolver estes problemas de arranque. 
  
Também pode executar a [ferramenta De Diagnóstico Avançado](https://aka.ms/SaRA-OutlookAdvDiagnostics) na máquina afetada.
