# <a name="got-a-parameter-cannot-be-found-error"></a>Tem um erro 'Parâmetro não é possível localizar'?

Se receber a mensagem de erro **"não é possível encontrar um parâmetro que corresponda ao nome do parâmetro ' EmailAddresses"**, isso significa que está a demorar um pouco mais tempo do Office 365 terminar de configurar o locatário ou o domínio personalizado se adicionou recentemente um. 
  
O processo de configuração pode demorar até quatro horas para ser concluído. Aguarde um pouco para que o processo de configuração seja concluído e, em seguida, tente novamente.
