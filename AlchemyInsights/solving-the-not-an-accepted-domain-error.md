# <a name="got-a-parameter-cannot-be-found-error"></a>Tem um erro de "Parâmetro não pode ser encontrado"?

Se receber a mensagem de erro **"Não é possível encontrar um parâmetro que corresponda ao nome de parâmetro 'EmailAddresses'** significa que está a demorar um pouco mais para a Microsoft 365 terminar a configuração do seu inquilino, ou o seu domínio personalizado se tiver adicionado recentemente um.
  
O processo de configuração pode demorar até quatro horas para ser concluído. Aguarde um pouco para que o processo de configuração seja concluído e, em seguida, tente novamente.
