# <a name="solutions-for-error-30016-4"></a>Soluções para o erro 30016-4

Quando tenta desinstalar o Office 365 ProPlus de um computador, poderá receber a seguinte mensagem de erro:
  
> "Alguma coisa correu errado 30016-4"
    
Este problema ocorre se a unidade **% temp %** é mapeada para uma unidade diferente de **% ProgramFiles %**. 
  
Para corrigir, remova completamente o Office do computador.
  
1. Transferir e instalar o [Assistente de recuperação e suporte](https://aka.ms/SARA-OfficeUninstall-Alchemy).
    
2. Quando é aberto, a partir de **produtos de desinstalar o Office** , seleccione a versão que pretende desinstalar e seleccione **seguinte**. 
    
3. Continuar a percorrer ecrãs e reinicie o computador quando lhe for pedido.
    
    Depois de reiniciar, apoio e Assistente de recuperação automaticamente abre novamente para concluir o processo de desinstalação.
    
4. Siga os pedidos restantes e reinstalar o Office ou fechar o assistente.
