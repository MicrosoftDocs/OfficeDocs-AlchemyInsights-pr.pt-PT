# <a name="how-to-create-a-file-request"></a>Como criar uma solicitação de arquivo

O recurso de solicitação de arquivo no OneDrive permite que você escolha uma pasta onde outras pessoas podem carregar arquivos usando um link que você os envia. Os usuários que você solicita arquivos não podem ver o que está na pasta - eles só podem carregar arquivos para ele.

[Criar uma solicitação de arquivo](https://support.office.com/article/create-a-file-request-f54aa7f8-2589-4421-b351-d415fc3b83af)
