# <a name="permissions-in-microsoft-stream"></a>Permissões no Microsoft Stream

Por padrão, quando faz o upload de um vídeo, as permissões são definidas para que o vídeo seja visível para toda a empresa. Se pretender especificar permissões a utilizadores/grupos específicos, pode personalizar as permissões de vídeo.

- [Permissões e privacidade no Microsoft Stream](https://docs.microsoft.com/stream/portal-permissions)

- [Permissões de grupo](https://docs.microsoft.com/stream/portal-permissions#group-permissions)

- [Permissões do canal](https://docs.microsoft.com/stream/portal-permissions#channel-permissions)
