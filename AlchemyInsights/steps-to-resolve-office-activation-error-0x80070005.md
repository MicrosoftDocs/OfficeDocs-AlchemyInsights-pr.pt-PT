# <a name="steps-to-resolve-office-activation-error-0x80070005"></a>Passos para resolver erro de ativação do escritório 0x80070005

Se o erro 0x80070005 ter começado após a atualização do Windows, pode corrigir o problema atualizando o registo.
  
Descarregue e instale o Assistente de [Suporte e Recuperação](https://aka.ms/SARA-OfficeActivation-Alchemy) e execute o **"Activation Troubleshooter".**
  
Para obter ajuda adicional, consulte: Código de [erro do escritório 0x800700005 ao ativar o Office](https://support.office.com/article/7aa7600f-df57-4aef-81d2-25509c66f865)
