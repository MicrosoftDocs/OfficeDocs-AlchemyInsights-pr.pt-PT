# <a name="create-a-folder"></a>Criar uma pasta

1. No painel da esquerda, seleccione **Criar nova pasta**![a criar o novo botão da pasta](media/d8e28612-fbdb-4d28-a4d0-14f7834cfd97.png) junto a **pastas**. 
    
2. Na caixa pasta nova, introduza um nome para a pasta e prima Enter.
    
Para obter mais informações, consulte [criar uma pasta no Outlook.com](https://go.microsoft.com/fwlink/p/?linkid=873114).
