# <a name="upgrade-to-visio-2019"></a>Upgrade para Visio 2019

Os utilizadores que instalaram a aplicação visio desktop como parte de uma subscrição Visio Online já têm a versão mais recente. 

Para atualizar uma versão não subscrição como o Visio 2016, é necessário comprar uma subscrição Visio Online ou uma compra única da Visio Standard ou Vision Professional 2019. Ver [Comparar opções visio](https://products.office.com/visio/microsoft-visio-plans-and-pricing-compare-visio-options).

Se já adquiriu uma versão mais recente e está pronto para a instalar, consulte [a Install Visio](https://support.office.com/article/f98f21e3-aa02-4827-9167-ddab5b025710?wt.mc_id=OfficeAdm_ClientDIA_Alchemy1798) e siga os passos para a sua versão. 

Para instalar o Visio lado a lado com o Office, consulte [cenários suportados para instalar diferentes versões de Office, Project e Visio no mesmo computador](https://docs.microsoft.com/deployoffice/install-different-office-visio-and-project-versions-on-the-same-computer).
