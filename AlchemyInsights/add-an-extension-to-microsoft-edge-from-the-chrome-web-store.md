# <a name="add-an-extension-to-microsoft-edge-from-the-chrome-web-store"></a>Adicione uma extensão ao Microsoft Edge a partir da loja web do Chrome

Para adicionar uma extensão ao Microsoft Edge a partir da loja web do Chrome, complete os seguintes passos:

1. Vá à Loja Chrome.
2. No banner no topo da página, selecione **Permitir extensões de outras lojas**.
3. Selecione a extensão que pretende adicionar e, em seguida, clique em **Adicionar ao Chrome**.
4. No diálogo de confirmação, reveja cuidadosamente as modificações que a extensão pode escoar e, em seguida, clique em **Adicionar extensão**.
O diálogo final confirmará que a extensão foi adicionada.
