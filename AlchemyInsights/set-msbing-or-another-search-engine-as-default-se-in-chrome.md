# <a name="set-microsoft-bing-or-another-search-engine-as-the-default-search-engine-in-google-chrome"></a>Desafine o Microsoft Bing ou outro motor de busca como o motor de busca padrão no Google Chrome

Para definir o seu motor de busca padrão no Google Chrome:

1. Abra o Google Chrome.
1. No topo direito, selecione o ícone que mostra três pontos verticais e, em seguida, selecione **Definições**.
1. No **motor de busca,** ao lado **do motor de busca** utilizado na barra de endereços, selecione a seta para baixo e, em seguida, selecione **Bing** ou outro motor de busca.

Mudar para um motor de busca diferente de Bing não remove a extensão Bing, por isso ainda pode pesquisar facilmente conteúdo relacionado com o trabalho através do flyout orientado por extensão.
