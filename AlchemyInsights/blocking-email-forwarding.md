# <a name="block-email-forwarding"></a>Bloquear o encaminhamento de e-mail

Para desabilitar o encaminhamento de e-mail para uma caixa de correio específica, consulte [Configurar encaminhamento de e-mail](https://docs.microsoft.com/office365/admin/email/configure-email-forwarding).

Para bloquear o encaminhamento de email para toda a organização, [as várias maneiras de bloquear o encaminhamento de email automático no Exchange Online](https://blogs.technet.microsoft.com/exchange/2017/12/22/the-many-ways-to-block-automatic-email-forwarding-in-exchange-online/).
