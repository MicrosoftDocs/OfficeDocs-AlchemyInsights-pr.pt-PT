# <a name="block-email-forwarding"></a>Bloquear o reencaminhamento de correio electrónico

Para desactivar a mensagem de correio electrónico reencaminhar para uma caixa de correio específica, consulte [configurar o reencaminhamento de correio electrónico](https://support.office.com/client/15abf81d-5c5d-49da-ac81-1b4daa1809f6).

Para bloquear o reencaminhamento de correio electrónico para toda a organização, [várias formas de bloquear automático de correio de reencaminhamento no Exchange Online](https://blogs.technet.microsoft.com/exchange/2017/12/22/the-many-ways-to-block-automatic-email-forwarding-in-exchange-online/).
