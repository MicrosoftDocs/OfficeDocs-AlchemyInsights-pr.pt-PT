# <a name="block-email-forwarding"></a>Bloquear reencaminhamento de e-mail

Para desativar o envio de e-mail para uma caixa de correio específica, consulte o [envio de e-mail configure](https://docs.microsoft.com/office365/admin/email/configure-email-forwarding).

Para bloquear o reencaminhamento de e-mails para toda a organização, [as muitas formas de bloquear o reencaminhamento automático de e-mails no Exchange Online](https://blogs.technet.microsoft.com/exchange/2017/12/22/the-many-ways-to-block-automatic-email-forwarding-in-exchange-online/).
