# <a name="401-unauthorized-error-in-sharepoint"></a>401 Erro não autorizado no SharePoint

Se receber o erro "(401) Não Autorizado" no SharePoint, poderá estar relacionado com a depreciação de TLS 1.0/1.1. Para obter mais informações, consulte:

[Preparando-se para o TLS 1.2 no Office 365 e Office 365 GCC](https://docs.microsoft.com/microsoft-365/compliance/prepare-tls-1.2-in-office-365)

[Erros de autenticação ocorrem se o cliente não tiver suporte TLS 1.2](https://review.docs.microsoft.com/sharepoint/troubleshoot/administration/authentication-errors-tls12-support)
