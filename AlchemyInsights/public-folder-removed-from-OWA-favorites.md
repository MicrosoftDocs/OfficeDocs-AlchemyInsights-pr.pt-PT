# <a name="public-folder-removed-from-outlook-on-the-web-owa-favorites"></a>Pasta pública removida dos Outlook favoritos na Web (OWA)

Existe um problema conhecido em que as pastas públicas são removidas dos favoritos no OWA em cada início de sessão ou atualização do ecrã. O problema será corrigido em breve. Por enquanto, adicione a pasta para acesso a partir do OWA.
