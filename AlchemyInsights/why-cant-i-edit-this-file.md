# <a name="why-cant-i-edit-this-file"></a>Por que não posso editar este ficheiro?

Se um documento, folha de cálculo ou apresentação estiver bloqueado para edição e não puder ser editado nas aplicações do windows desktop, pode ser que o ficheiro esteja a ser editado por outra pessoa, ou que tenha o ficheiro verificado.

Veja rapidamente algumas das razões pelas quais [poderá não conseguir editar o ficheiro](https://support.office.com/article/why-can-t-i-edit-this-file-97315f48-aa5e-49d3-a4ae-a14b73daf87b).
