# <a name="configure-stay-signed-in-for-accounts"></a>Configurar Manter a conta com a assinatura assinada para contas

A indicação Manter a minha conta com êxito (KMSI) apresenta um aviso Continuar com a **assinatura?** após um utilizador ter ativado a sua assinatura. Se um utilizador responder **Sim** a este pedido, o serviço KMSI fornece-lhe um token de atualização persistente. 

Para obter mais informações sobre a configuração de KMSI, consulte Configurar o pedido "Continuar com a assinatura?" para contas [do Azure AD.](/azure/active-directory/fundamentals/keep-me-signed-in)
