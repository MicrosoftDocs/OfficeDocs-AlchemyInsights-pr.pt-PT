# <a name="refresh-using-web-connector-doesnt-work-properly"></a>Refresh usando o conector Web não funciona corretamente

Se tiver um script de conector web que esteja a usar a função [Web.Page,](https://msdn.microsoft.com/library/mt260924.aspx) e tiver atualizado o seu conjunto de dados ou relatório após 18 de novembro de 2016, precisa de usar um gateway para que a atualização funcione corretamente.

Para mais informações: [https://docs.microsoft.com/power-bi/refresh-troubleshooting-refresh-scenarios](https://docs.microsoft.com/power-bi/refresh-troubleshooting-refresh-scenarios)
