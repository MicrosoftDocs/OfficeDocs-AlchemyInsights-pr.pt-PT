# <a name="problems-opening-office-files-from-sharepoint"></a>Problemas ao abrir ficheiros do Office a partir do SharePoint

Se tiver problemas ao abrir ficheiros do Office e documentos do SharePoint no seus locais clientes do Office, consulte as hiperligações abaixo. 

[Verificar limitações de ficheiro para garantir o ficheiro é suportada](https://support.office.com/article/Invalid-file-names-and-file-types-in-OneDrive-OneDrive-for-Business-and-SharePoint-64883a5d-228e-48f5-b3d2-eb39e07630fa).

[Eliminar a cache de documentos do office](https://support.office.com/article/Delete-your-Office-Document-Cache-b1d3765e-d71b-4bb8-99ca-acd22c42995d).

[Execute uma reparação no Office](https://support.office.com/Article/Repair-an-Office-application-7821d4b6-7c1d-4205-aa0e-a6b40c5bb88b).

Para obter mais informações ou detalhes, consulte o artigo, [corrigir problemas ao abrir documentos em bibliotecas do SharePoint](https://support.office.com/article/Fix-problems-opening-documents-in-SharePoint-libraries-31329FA1-4AD0-47FC-95D8-BB0C5B12A536).

**Nota**: Se estiver a ter problemas com documentos que abrir no browser ou cliente por predefinição, poderá ter de alterar a definição de biblioteca. Para tal, navegue para o comportamento >Default para a opção de documentos preparados para browser.
