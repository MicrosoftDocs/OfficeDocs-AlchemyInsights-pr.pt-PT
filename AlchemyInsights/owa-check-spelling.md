# <a name="use-your-browser-to-check-spelling"></a>Use o seu navegador para verificar a ortografia

Não existe nenhuma opção de verificação de feitiços no Outlook na web. Para verificar a sua ortografia, terá de usar o seu navegador web. A verificação ortográfica está disponível nas versões Microsoft Edge, Internet Explorer 10 e posteriores, e nas versões atuais do Firefox, Chrome e Safari. Consulte as opções para o seu navegador web para saber mais sobre como verificar a ortografia.
