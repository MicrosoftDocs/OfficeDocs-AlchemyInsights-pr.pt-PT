# <a name="activating-a-new-device-running-windows-10"></a>Ativando um novo dispositivo rodando o Windows 10

Se você comprou um novo dispositivo que executa o Windows 10, ele será ativado automaticamente quando você conectar seu dispositivo à Internet. Se os servidores de ativação estiverem ocupados, talvez seja necessário esperar um pouco para que o Windows seja ativado.

Para verificar o status de ativação, escolha a**atualização** > de **configurações** >  **iniciosas** &**ativação**de segurança.

**Nota:** Em alguns casos, você pode precisar inserir a chave do produto Windows 10 que veio com a embalagem do dispositivo, ou incluída como um cartão ou no Certificado de Autenticidade (COA) anexado ao dispositivo.

[Saiba mais sobre ativação no Windows 10](https://support.microsoft.com/help/12440)
