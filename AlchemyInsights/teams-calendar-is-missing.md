# <a name="teams-calendar-is-missing"></a>Falta calendário de equipas

O administrador das equipas pode ter desativado o calendário, por isso, primeiro verifique se está disponível para si.

Se estiver disponível, tente mudar a sua versão de Equipas.

- Se estiver a utilizar a versão web, mude para a versão instalada.
- Se estiver a utilizar a versão instalada, experimente equipas na web.
