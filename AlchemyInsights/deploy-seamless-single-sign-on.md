# <a name="enable-seamless-sso"></a>Ativar sso sem costura

O Azure Active Directory (Azure AD) Seamless Single Sign-On (Seamless SSO) assina automaticamente os usuários quando estão em seus desktops corporativos conectados à sua rede corporativa.
  
[Siga as etapas deste artigo para habilitar o SSO sem costura](https://docs.microsoft.com/azure/active-directory/connect/active-directory-aadconnect-sso-quick-start)
