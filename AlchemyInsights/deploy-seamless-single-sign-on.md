# <a name="enable-seamless-sso"></a>Ativar SSO sem emenda

O Azure Ative Directory (Azure AD) Seamless Single Sign-On (Seamless SSO) assina automaticamente nos utilizadores quando estão nos seus desktops corporativos ligados à sua rede corporativa.
  
[Siga os passos deste artigo para ativar o SSO sem emenda](https://docs.microsoft.com/azure/active-directory/connect/active-directory-aadconnect-sso-quick-start)
