# <a name="managing-microsoft-stream-licenses"></a>Gerenciamento de licenças do Microsoft Stream

**Importante:** Você precisa estar no papel de Administrador inquilino para executar as operações descritas neste tópico.

Você precisa ser um administrador global ou um administrador de gerenciamento de usuários. Para mais informações, veja [sobre as funções de administração](https://docs.microsoft.com/office365/admin/add-users/about-admin-roles).

Você pode atribuir licenças a contas de usuário com o [Office 365 PowerShell](https://go.microsoft.com/fwlink/p/?linkid=850410).
