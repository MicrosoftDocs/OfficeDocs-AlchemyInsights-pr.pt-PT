# <a name="profile-property-synchronization"></a>Sincronização de propriedade de perfil

Não é possível mapear propriedades adicionais para a AD Import sincronizar do Azure Ative Directory para a Aplicação de Perfil do Utilizador. A AD Import está limitada a um conjunto de propriedades pré-configuradas para garantir um desempenho consistente do trabalho do temporizador.
  
[Saiba mais sobre as propriedades de perfil que sincronizam](https://go.microsoft.com/fwlink/?linkid=875671)
