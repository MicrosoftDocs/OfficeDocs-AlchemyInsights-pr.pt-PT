# <a name="profile-property-synchronization"></a>Sincronização de perfis de propriedade

Não é possível mapear propriedades adicionais para a importação de AD sincronizar a partir do Active Directory Azure para a aplicação do perfil de utilizador. Importação de AD está limitada a um conjunto de propriedades para garantir um desempenho consistente da tarefa de temporizador pré-configurado.
  
[Obter mais informações sobre as propriedades de perfil que sincronizar](https://go.microsoft.com/fwlink/?linkid=875671)
