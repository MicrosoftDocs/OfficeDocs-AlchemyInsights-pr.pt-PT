# <a name="move-a-folder"></a>Mover uma pasta

Pode arrastar pastas **Favoritos** reordená-las.
  
Não é possível reordenar pastas em **pastas** ou mover uma subpasta para se tornar uma pasta.

*Nota:* Se mover uma pasta seja uma subpasta, crie uma nova pasta e copie o correio electrónico da sub-pasta para a nova pasta.
  
Para obter mais informações, consulte [trabalhar com pastas de mensagens em Outlook.com](https://support.office.com/article/6bb0723a-f39f-4a8d-bb3f-fab5dcc2510a).
