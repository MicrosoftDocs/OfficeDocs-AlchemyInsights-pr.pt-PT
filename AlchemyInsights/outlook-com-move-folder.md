# <a name="move-a-folder-in-outlookcom"></a>Mova uma pasta em Outlook.com

Pode reencomendar **pastas em "Favoritos",** mas não é possível reencomendar pastas nas **pastas** ou fazer uma sub-dobragem uma pasta. Se mover acidentalmente uma pasta para outra pasta, crie uma nova pasta e copie todo o seu e-mail da sub-página para a nova pasta.
  
Para saber mais, leia [Trabalhar com pastas de mensagens em Outlook.com](https://support.office.com/article/6bb0723a-f39f-4a8d-bb3f-fab5dcc2510a?wt.mc_id=Office_Outlook_com_Alchemy).
