# <a name="move-a-folder-in-outlookcom"></a>Mover uma pasta na Outlook.com

Pode reordenar pastas nos **Favoritos**, mas não é possível reordenar pastas em **pastas** ou criar uma subpasta de uma pasta. Se mover acidentalmente uma pasta para outra pasta, crie uma nova pasta e copie o correio electrónico da subpasta para a nova pasta.
  
Para obter mais informações, leia a [trabalhar com pastas de mensagens em Outlook.com](https://support.office.com/article/6bb0723a-f39f-4a8d-bb3f-fab5dcc2510a?wt.mc_id=Office_Outlook_com_Alchemy).
