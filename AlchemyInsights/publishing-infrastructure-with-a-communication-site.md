# <a name="publishing-infrastructure-with-a-communication-site"></a>Publicação de infraestrutura com um site de comunicação


Habilitar o recurso de infraestrutura de publicação de servidor espartilho sharepoint não é **suportado** com sites de comunicação. 
  
Para mais informações, [consulte a mudança de sites de publicação para sites](https://docs.microsoft.com/sharepoint/publishing-sites-classic-to-modern-experience)de comunicação.
