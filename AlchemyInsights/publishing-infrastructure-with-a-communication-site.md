# <a name="publishing-infrastructure-with-a-communication-site"></a>Infra-estrutura de publicação com um Site de comunicação


Activar a funcionalidade de infra-estrutura de publicação de servidor do SharePoint é **não suportada** com Sites de comunicação. 
  
Para obter mais informações, consulte a [mover a publicação sites para sites de comunicação](https://docs.microsoft.com/sharepoint/publishing-sites-classic-to-modern-experience).
