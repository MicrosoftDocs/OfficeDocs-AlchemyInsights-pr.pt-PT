# <a name="publishing-infrastructure-with-a-communication-site"></a>Infraestrutura de publicação com site de comunicação


Ativar a funcionalidade de Infraestrutura de publicação do servidor SharePoint não é **suportada** com sites de comunicação. 
  
Para obter mais informações, consulte [movendo-se de sites de publicação para sites de comunicação.](https://docs.microsoft.com/sharepoint/publishing-sites-classic-to-modern-experience)
