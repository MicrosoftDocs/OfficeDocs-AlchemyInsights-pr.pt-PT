# <a name="cannot-access-sharepoint"></a>Incapacidade de aceder ao SharePoint

Existem várias razões pelas quais o SharePoint ou o OneDrive podem tornar-se inacessíveis. Se não conseguir aceder ao SharePoint ou ao OneDrive, utilize o seguinte guia para resolver este problema.

- [Não é possível aceder ao SharePoint ou ao OneDrive](https://docs.microsoft.com/sharepoint/troubleshoot/sharing-and-permissions/sharepoint-online-inaccessible)
