# <a name="turn-off-focused-inbox-in-email-clients"></a>Desativar a caixa de entrada focada em clientes de email

Para desativar a caixa de entrada focada no Outlook 2016, Outlook na Web, Outlook.com/Hotmail.com, Outlook 2016 para Mac ou mail para Windows 10, consulte [desativar a caixa de entrada focada](https://support.office.com/article/f714d94d-9e63-4217-9ccb-6cb2986aa1b2.aspx).

Para desativar a caixa de entrada focada para todos na organização, consulte [Configurar a caixa de entrada focada para todos na sua organização](https://docs.microsoft.com/office365/admin/setup/configure-focused-inbox).
