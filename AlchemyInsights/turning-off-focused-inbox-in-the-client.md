# <a name="turn-off-focused-inbox-in-email-clients"></a>Desactivar direccionadas para a pasta a receber de clientes de correio electrónico

Para desactivar direccionadas para a pasta a receber no Outlook de 2016, o Outlook na web, Outlook.com/Hotmail.com, 2016 do Outlook para Mac ou o correio for Windows 10, consulte [Desactivar direccionadas para a pasta a receber](https://support.office.com/article/f714d94d-9e63-4217-9ccb-6cb2986aa1b2.aspx).

Para desactivar direccionadas para a pasta a receber para todos os utilizadores na organização, consulte [Configurar direccionadas para a pasta a receber para todos os utilizadores na organização](https://docs.microsoft.com/office365/admin/setup/configure-focused-inbox).
