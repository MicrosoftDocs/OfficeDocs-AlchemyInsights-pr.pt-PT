# <a name="what-is-autosave"></a>O que é autosave?

O AutoSave é um recurso disponível nos assinantes do Excel, Word e PowerPoint para o Office 365 que salva seu arquivo automaticamente, a cada poucos segundos, enquanto você trabalha. 

Para perguntas frequentes, veja [o que é autoSave?](https://support.office.com/article/6d6bd723-ebfd-4e40-b5f6-ae6e8088f7a5)
