# <a name="what-is-autosave"></a>O que é a gravação automática?

Gravação automática é uma nova funcionalidade disponível no Excel, Word e PowerPoint para subscritores Office 365 guarda automaticamente, o ficheiro de cada alguns segundos, à medida que trabalha. 

Para perguntas mais frequentes, consulte [o que é AutoSave?](https://support.office.com/article/6d6bd723-ebfd-4e40-b5f6-ae6e8088f7a5)
