# <a name="what-is-autosave"></a>O que é AutoSave?

O AutoSave é uma funcionalidade disponível no Excel, Word e PowerPoint para os subscritores do Microsoft 365 que guarda o seu ficheiro automaticamente, a cada poucos segundos, enquanto trabalha. 

Para perguntas frequentes, veja [o que é AutoSave?](https://support.office.com/article/6d6bd723-ebfd-4e40-b5f6-ae6e8088f7a5)
