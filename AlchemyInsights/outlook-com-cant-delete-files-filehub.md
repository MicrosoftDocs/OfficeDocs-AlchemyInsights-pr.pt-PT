# <a name="cant-delete-files-or-photos-from-files"></a>Não é possível eliminar ficheiros ou fotografias de ficheiros

Ficheiros mostra os ficheiros e fotografias anexadas a mensagens de correio. Não é possível eliminar ficheiros ou fotografias directamente a partir de ficheiros. Para eliminar ficheiros e fotografias, terá de eliminar as mensagens de correio electrónico que estão anexadas:

Para mais informações, leia [como eliminar ficheiros ou fotografias em Outlook.com](https://support.office.com/article/bae0531f-040f-4c42-90b9-786ca718c16d.aspx).
