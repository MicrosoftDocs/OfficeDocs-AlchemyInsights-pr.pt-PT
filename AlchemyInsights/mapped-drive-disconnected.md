# <a name="sharepoint-libraries-mapped-to-network-drives-become-disconnected"></a>As bibliotecas do SharePoint mapeadas para as unidades de rede ficam desligadas

Mapear uma biblioteca de documentos SharePoint para uma unidade de rede é temporário. Deve ocasionalmente abrir o site SharePoint no Internet Explorer e selecionar **Stay in in** para evitar que a sessão expire. 
  
[Obtenha ajuda para configurar e resolver problemas de unidades de rede mapeadas](https://docs.microsoft.com/sharepoint/support/administration/troubleshoot-mapped-network-drives).
  
Para trabalhar com uma biblioteca no File Explorer, [sincronizar ficheiros SharePoint com o novo cliente de sincronização OneDrive](https://support.office.com/article/6de9ede8-5b6e-4503-80b2-6190f3354a88.aspx) que inclui [ficheiros on-demand](https://support.office.com/article/0e6860d3-d9f3-4971-b321-7092438fb38e.aspx). Aceda a todos os seus ficheiros no OneDrive sem utilizar o espaço de armazenamento local.
