# <a name="sync-a-sharepoint-library-with-onedrive-sync"></a>Sync uma Biblioteca SharePoint com OneDrive Sync

Para que os seus ficheiros SharePoint se sincronizem com o disco rígido local, complete os seguintes passos:

1. Navegue para a biblioteca de documentos que gostaria de sincronizar com o seu disco rígido local.
2. Selecione o botão Sync a partir das opções do menu.
3. Siga passo a passo para completar a sincronização.
