# <a name="dashboard-doesnt-reflect-changes-after-refresh"></a>Dashboard não reflete alterações após atualização

Se o painel não refletir alterações após a atualização, aguarde cerca de 10-15 minutos para que a atualização seja refletida nos azulejos do painel de instrumentos. Se ainda não estiver a aparecer, re-fixar a visualização no painel de instrumentos.

Para obter mais informações, consulte [cenários de atualização de resolução de problemas](https://docs.microsoft.com/power-bi/refresh-troubleshooting-refresh-scenarios).
