# <a name="prevent-files-from-being-downloaded-from-sharepoint-or-onedrive"></a>Impedir que os ficheiros a ser transferido a partir do SharePoint ou OneDrive

Existem vários métodos que podem ser utilizados para impedir ou bloquear transferências ocorra. Visite as hiperligações abaixo para os respectivos métodos:

- [Transferência de ficheiro do bloco com acesso condicional Azure AD](https://docs.microsoft.com/cloud-app-security/use-case-proxy-block-session-aad#create-a-block-download-policy-for-unmanaged-devices)

- [Controlar o acesso de dispositivos não geridos](https://docs.microsoft.com/sharepoint/control-access-from-unmanaged-devices)

- [Configurar política IRM em bibliotecas de documentos do SharePoint para limitar a transferência de conteúdo](https://docs.microsoft.com/office365/securitycompliance/set-up-irm-in-sp-admin-center)
