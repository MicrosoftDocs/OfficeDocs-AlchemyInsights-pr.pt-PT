# <a name="prevent-files-from-being-downloaded-from-sharepoint-or-onedrive"></a>Evite que os ficheiros sejam descarregados do SharePoint ou do OneDrive

Existem vários métodos que podem ser usados para impedir que ocorram transferências. Visite os links abaixo para os respetivos métodos:

- [Download de Ficheiros de Blocos com acesso condicional AD Azure](https://docs.microsoft.com/cloud-app-security/use-case-proxy-block-session-aad#create-a-block-download-policy-for-unmanaged-devices)

- [Controlar o acesso de dispositivos não geridos](https://docs.microsoft.com/sharepoint/control-access-from-unmanaged-devices)

- [Configurar políticas de IRM nas bibliotecas de documentos SharePoint para limitar o descarregamento de conteúdos](https://docs.microsoft.com/microsoft-365/compliance/set-up-irm-in-sp-admin-center)
