# <a name="screen-sharing-with-skype-consumer-users"></a>Partilha de ecrã com utilizadores de consumidores skype

Para instruções gerais sobre como partilhar o seu programa, para aprender sobre restrições com partilha e para resolução de recursos, [reveja Partilhe o seu ecrã no Skype for Business](https://support.office.com/article/Share-your-screen-in-Skype-for-Business-2d436dc9-d092-4ef1-83f1-dd9f7a7cd3fc).  

Atualmente, o Skype para utilizadores empresariais não conseguem fazer o ecrã partilhar com os utilizadores do Skype. Clique [aqui](https://www.skypefeedback.com/forums/299913-generally-available/suggestions/12335259-enable-screen-sharing-to-consumer-skype-users) para fornecer feedback sobre como permitir a partilha de ecrãs com os utilizadores do Skype do consumidor.
