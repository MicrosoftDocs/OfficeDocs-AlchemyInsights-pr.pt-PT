# <a name="give-users-access-to-sharepoint-site"></a>Dar aos usuários acesso ao site SharePoint

O recurso de solicitação de acesso permite que as pessoas solicitem acesso a conteúdo que atualmente não têm permissão para ver. Como proprietário do site, você pode configurar o recurso para enviar-lhe e-mail quando alguém solicita acesso a um site. Para mais informações, visite [Configurar e gerenciar solicitações de acesso.](https://support.office.com/article/set-up-and-manage-access-requests-94b26e0b-2822-49d4-929a-8455698654b3)
