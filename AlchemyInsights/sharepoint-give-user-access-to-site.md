# <a name="give-users-access-to-sharepoint-site"></a>Dar aos utilizadores acesso ao SharePoint site

A funcionalidade de pedido de acesso permite às pessoas a pedir acesso ao conteúdo que não têm actualmente permissão para ver. Como proprietário do site, pode configurar a funcionalidade para enviar correio, quando alguém pedidos de acesso a um site. Para mais informações, visite [definir uma cópia de segurança e gerir pedidos de acesso](https://support.office.com/article/set-up-and-manage-access-requests-94b26e0b-2822-49d4-929a-8455698654b3).
