# <a name="give-users-access-to-sharepoint-site"></a>Dar aos utilizadores acesso ao site SharePoint

A funcionalidade de pedido de acesso permite que as pessoas solicitem acesso a conteúdos que não têm atualmente autorização para ver. Como proprietário de um site, pode configurar a funcionalidade para lhe enviar correio quando alguém pede acesso a um site. Para mais informações, visite [Configurar e gerir pedidos](https://support.office.com/article/set-up-and-manage-access-requests-94b26e0b-2822-49d4-929a-8455698654b3)de acesso.
