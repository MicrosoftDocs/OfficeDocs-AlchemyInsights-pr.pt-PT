# <a name="site-is-not-available"></a>Site não está disponível

Se o site SharePoint ou OneDrive não estiver disponível para vários usuários, pode haver um problema de serviço temporário. Verifique o painel de Saúde do [Serviço](https://admin.microsoft.com/AdminPortal/Home#/servicehealth) para ver se seu locatário está afetado.
