# <a name="site-is-not-available"></a>O site não está disponível

Se o seu site SharePoint ou OneDrive não estiver disponível para vários utilizadores, poderá haver um problema de serviço temporário. Consulte o [painel de instrumentos de saúde](https://admin.microsoft.com/AdminPortal/Home#/servicehealth) do serviço para ver se o seu inquilino está afetado.
