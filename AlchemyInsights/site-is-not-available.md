# <a name="site-is-not-available"></a>Site não está disponível

Se o site SharePoint ou OneDrive não está disponível para vários utilizadores, poderá existir um problema de serviço temporária. Verifique se o[Serviço de saúde Dashboard](https://admin.microsoft.com/AdminPortal/Home#/servicehealth) para ver se o locatário é afectado.
