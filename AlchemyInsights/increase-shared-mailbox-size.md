# <a name="increase-the-size-of-a-shared-mailbox"></a>Aumentar o tamanho da caixa de correio partilhada

Pode aumentar o tamanho de uma caixa de correio partilhada do tamanho predefinido de 50 GB para 100 GB, atribuindo a caixa de correio partilhada uma licença de P2 Online do Exchange (que é incluída no licenças E3 e E5). Quando atribui a licença de acesso P2 na caixa de correio partilhada no [utilizadores activos](https://portal.office.com/adminportal/home), o tamanho serão automaticamente aumentado no prazo de 15 minutos.
