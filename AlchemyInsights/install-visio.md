# <a name="install-visio"></a>Instalar visio

A Visio é vendida como uma aplicação autónoma, por isso, se ainda não a tiver, [compre ou experimente a Visio.](https://products.office.com/visio) 

**Nota:** O Visio Online Plan 1 não inclui uma versão desktop da Visio, pelo que nenhuma instalação é possível.

Se tiver uma subscrição do Visio Online Plan 2, tem de [atribuir uma licença](https://docs.microsoft.com/office365/admin/subscriptions-and-billing/assign-licenses-to-users?wt.mc_id=OfficeAdm_ClientDIA_Alchemy1764) a um utilizador ou não terá opção de instalar o Visio a partir das minhas **instalações** no [portal da minha conta.](https://portal.office.com/account#installs) 

As versões de licença sem volume da Visio Standard ou visio Professional estão associadas às contas da Microsoft e não podem ser geridas no centro de administração. Para instalar estas versões, os utilizadores devem iniciar sessão para [account.microsoft.com](https://account.microsoft.com) com a conta Microsoft associada a esta versão do Visio.

Para obter ajuda mais detalhada, consulte a [Install Visio](https://support.office.com/article/f98f21e3-aa02-4827-9167-ddab5b025710?wt.mc_id=OfficeAdm_ClientDIA_Alchemy1764) e siga os passos para a sua versão do Visio.
