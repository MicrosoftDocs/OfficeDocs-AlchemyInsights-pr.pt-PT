# <a name="how-to-stop-signing-in-automatically"></a>Como parar de iniciar sessão automaticamente

Para parar de iniciar sessão para Outlook.com automaticamente, desverifique a caixa para **me manter inscrito** na página onde insere a sua palavra-passe.
  
Para saber mais, veja [como iniciar sessão dentro ou fora de Outlook.com](https://support.office.com/article/e08eb8ac-ac27-49f4-a400-a47311e1ee7e?wt.mc_id=Office_Outlook_com_Alchemy).
