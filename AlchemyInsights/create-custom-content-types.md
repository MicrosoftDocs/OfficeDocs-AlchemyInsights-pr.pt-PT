# <a name="create-custom-content-types"></a>Criar tipos de conteúdo personalizado

1. Selecione a lista com a quais pretende trabalhar no Lançamento Rápido ou clique em **Definições** (ícone de engrenagem) e, em seguida, clique nas **definições do Site**.
2. Clique nos tipos de **conteúdo do site** nas **Galerias Web Designer.**
3. Na página 'Tipos de Conteúdo do Site', clique em Criar.
4. Na página New Site Content Type, preencha as informações para o seu tipo de conteúdo personalizado.

Para obter mais informações, consulte [os tipos de conteúdo personalizados.](https://support.microsoft.com/office/e1277a2e-a1e8-4473-9126-91a0647766e5#__toc323548991)
