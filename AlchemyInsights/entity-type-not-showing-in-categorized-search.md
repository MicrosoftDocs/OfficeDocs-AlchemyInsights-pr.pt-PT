# <a name="entity-type-not-showing-in-search-results"></a>Tipo de entidade não aparecer nos resultados da procura

Para um tipo de entidade Mostrar na procura por categorias de procura ou relevância na Interface de unificado do Dynamics 365, este deve ser pela primeira vez adicionado o módulo de aplicação.

Nota: Quando procura anexos na procura de relevância, a entidade de anexo deve ser adicionada ao módulo de aplicação mesmo que o registo principal é o que é apresentado nos resultados.
