# <a name="get-in-app-support-for-the-outlook-mobile-app"></a>Obtenha suporte na aplicação para a aplicação móvel Outlook

1. Na aplicação móvel Outlook, abra o **Menu**.

2. Selecione o ponto de interrogação para abrir o ecrã **&amp; de Feedback de Ajuda.**

3. Toque **no Suporte de Contacto** e explique o seu problema.

4. Toque **em Enviar** e receberá uma notificação in-app quando um agente de suporte o contactar.

Para obter instruções detalhadas, consulte [obter ajuda na aplicação para Outlook para iOS e Android](https://support.office.com/article/218a22d1-9fa5-4889-b689-de1c63493243.aspx#ID0EAABAAA=Contact_Support).
