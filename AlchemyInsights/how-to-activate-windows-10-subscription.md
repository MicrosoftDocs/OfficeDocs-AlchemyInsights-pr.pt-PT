# <a name="activating-windows-10-subscriptions"></a>Ativando assinaturas do Windows 10

Microsoft 365 Business, Microsoft 365 Education, Microsoft 365 e Enterprise fornecem licenças digitais para o Windows 10 como parte da assinatura. Para ativar a assinatura, seu dispositivo já deve ter ativado o Windows 10 Pro (1703 ou posterior). Para ver uma lista completa de requisitos para ativar uma assinatura do Windows 10, consulte os requisitos de ativação de assinatura do [Windows 10.](https://docs.microsoft.com/windows/deployment/windows-10-subscription-activation#requirements)
