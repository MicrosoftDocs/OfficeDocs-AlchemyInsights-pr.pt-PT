# <a name="troubleshooting-missing-files-or-folders-in-onedrive-or-sharepoint"></a>Resolução de problemas de ficheiros ou pastas em OneDrive ou do SharePoint em falta

- [Verifique a Reciclagem do site](https://support.office.com/article/restore-deleted-items-from-the-site-collection-recycle-bin-5fa924ee-16d7-487b-9a0a-021b9062d14b)

- [Restaurar itens na Reciclagem de um site SharePoint](https://support.office.com/article/Restore-deleted-files-or-folders-in-OneDrive-949ada80-0026-4db3-a953-c99083e6a84f)



**Funcionalidade de restauro de ficheiros**

Se muitos dos ficheiros de OneDrive ou SharePoint obter eliminados, substituídos, danificados ou infectados por software malicioso, pode restaurar a biblioteca inteira de OneDrive ou do Sharepoint para uma hora anterior utilizando a funcionalidade de restauro de ficheiros.

- [Restaurar uma biblioteca de OneDrive](https://support.office.com/article/restore-your-onedrive-fa231298-759d-41cf-bcd0-25ac53eb8a15)

- [Restaurar uma biblioteca de documentos](https://support.office.com/article/restore-a-document-library-317791c3-8bd0-4dfd-8254-3ca90883d39a)

**Utilizar registo de auditoria ou o painel de actividade do ficheiro para verificar o histórico do ficheiro.**

[Verifique os relatórios de auditoria](https://docs.microsoft.com/office365/securitycompliance/search-the-audit-log-in-security-and-compliance?redirectSourcePath=%252fen-us%252farticle%252fsearch-the-audit-log-in-the-office-365-protection-center-0d4d0f35-390b-4518-800e-0c7ec95e946c) </a> pelo comando [aqui](https://protection.office.com/#/unifiedauditlog)

Utilize o painel de [actividade do ficheiro](https://support.office.com/article/File-activity-in-a-document-library-6105ecda-1dd0-4f6f-9542-102bf5c0ffe0) para verificar o histórico do ficheiro.

Verifique se o cliente de sincronização de OneDrive no computador local.  Se estiver a sincronizar os ficheiros no seu computador através de cliente de sincronização de OneDrive, verifique a pasta de sincronização local para se certificar de que este foi correctamente carregado. Certifique-se de que também verificar a ' Reciclagem ' no computador local.
