# <a name="troubleshooting-missing-files-or-folders-in-onedrive-or-sharepoint"></a>Solução de problemas arquivos ou pastas em OneDrive ou SharePoint

- [Verifique a lixeira do site](https://support.office.com/article/restore-deleted-items-from-the-site-collection-recycle-bin-5fa924ee-16d7-487b-9a0a-021b9062d14b)

- [Restaurar itens no lixo reciclado de um site SharePoint](https://support.office.com/article/Restore-deleted-files-or-folders-in-OneDrive-949ada80-0026-4db3-a953-c99083e6a84f)



**Recurso de restauração de arquivos**

Se muitos de seus arquivos OneDrive ou SharePoint forem excluídos, substituídos, corrompidos ou infectados por malware, você poderá restaurar toda a biblioteca OneDrive ou Sharepoint para um tempo anterior usando o recurso de restauração de arquivos.

- [Restaurar uma biblioteca OneDrive](https://support.office.com/article/restore-your-onedrive-fa231298-759d-41cf-bcd0-25ac53eb8a15)

- [Restaurar uma biblioteca documentada](https://support.office.com/article/restore-a-document-library-317791c3-8bd0-4dfd-8254-3ca90883d39a)

**Use a registro de auditoria ou o painel de atividade do arquivo para verificar o histórico do arquivo**

[Verifique os relatórios](https://docs.microsoft.com/office365/securitycompliance/search-the-audit-log-in-security-and-compliance?redirectSourcePath=%252fen-us%252farticle%252fsearch-the-audit-log-in-the-office-365-protection-center-0d4d0f35-390b-4518-800e-0c7ec95e946c) </a> de auditoria navegando [aqui](https://protection.office.com/#/unifiedauditlog)

Use o painel de atividade do [arquivo](https://support.office.com/article/File-activity-in-a-document-library-6105ecda-1dd0-4f6f-9542-102bf5c0ffe0) para verificar o histórico do arquivo.

Verifique o cliente OneDrive Sync em sua máquina local.  Se você estiver sincronizando os arquivos em seu computador através do cliente OneDrive Sync, verifique a pasta de sincronização local para garantir que ele tenha sido devidamente carregado. Certifique-se de verificar também a lixeira na sua máquina local.
