# <a name="set-passwords-to-never-expire"></a>Palavras-passe de conjunto para nunca expirar

Definir palavras-passe para nunca expirar do [segurança &amp; Privacy](https://portal.office.com/adminportal/home#/settings/security). Clique em 'Editar' junto a política de palavra-passe e activar o controlo de deslize para "ligado".
  
Para obter informações mais detalhadas, consulte: [definir a política de expiração de palavra-passe para a sua organização](https://support.office.com/article/set-the-password-expiration-policy-for-your-organization-0f54736f-eb22-414c-8273-498a0918678f).
