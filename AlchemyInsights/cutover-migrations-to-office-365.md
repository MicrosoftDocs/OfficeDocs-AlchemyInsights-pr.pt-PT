# <a name="cutover-migrations-to-office-365"></a>Migrações de transferência para o Office 365

Ao inscrever-se no Office 365, pode migrar os conteúdos das caixas de correio dos utilizadores de uma organização local do Exchange para o Office 365. Quando faz isto de uma só vez, esta denomina-se migração de transferência. A migração de transferência poderá ser adequada para si se estiver a executar uma versão suportada do Exchange, e se tiver menos de 2.000 caixas de correio. Para saber mais sobre migrações de transferência, consulte: [Migração de Transferência para o Office 365](https://support.office.com/article/9496e93c-1e59-41a8-9bb3-6e8df0cd81b4.aspx).
