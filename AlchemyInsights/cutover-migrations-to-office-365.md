# <a name="cutover-migrations-to-office-365"></a>Migrações cutover para o Office 365

Quando se inscreve Office 365, pode migrar o conteúdo de caixas de correio do utilizador de uma organização do Exchange no local para Office 365. Quando o faz tudo isto de uma só vez, é denominado uma migração de cutover. Uma migração cutover pode ser adequada para si, se estiver a executar uma versão suportada do Exchange e tiver menos de 2.000 caixas de correio. Para saber mais acerca de migrações de cutover, observe: [migração de Cutover para o Office 365](https://support.office.com/article/9496e93c-1e59-41a8-9bb3-6e8df0cd81b4.aspx).
