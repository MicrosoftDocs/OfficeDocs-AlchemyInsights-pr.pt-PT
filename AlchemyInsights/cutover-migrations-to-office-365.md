# <a name="cutover-migrations-to-office-365"></a>Migrações de transferência para o Office 365

Ao inscrever-se no Office 365, pode migrar os conteúdos das caixas de correio dos utilizadores de uma organização local do Exchange para o Office 365. Quando faz isto de uma só vez, esta denomina-se migração de transferência. A migração de transferência poderá ser adequada para si se estiver a executar uma versão suportada do Exchange, e se tiver menos de 2.000 caixas de correio. Para saber mais sobre migrações de transferência, consulte: [Migrar o e-mail utilizando o método de migração de transferência do Exchange](https://docs.microsoft.com/Exchange/mailbox-migration/cutover-migration-to-office-365).
