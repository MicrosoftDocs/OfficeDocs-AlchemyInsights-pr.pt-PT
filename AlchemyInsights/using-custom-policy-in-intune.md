# <a name="using-custom-policy-in-intune"></a>Utilizar política personalizada no Intune

Intune tem muitos incorporada de definições para dispositivos Android, iOS, macOS e no Windows. Pode também criar um perfil personalizado e, em seguida, implementar estas definições personalizadas para os dispositivos a utilizar o Intune. Perfis personalizados são concebidos para adicionar as definições do dispositivo e funcionalidades que não estão incorporadas Intune.

[Definições personalizadas para os dispositivos Android da Intune](https://docs.microsoft.com/intune/custom-settings-android)

[Definições personalizadas para os dispositivos Android empresa em Intune](https://docs.microsoft.com/intune/custom-settings-android-for-work)

[Definições personalizadas para os dispositivos de iOS da Intune](https://docs.microsoft.com/intune/custom-settings-ios)

[Definições personalizadas para os dispositivos de macOS da Intune](https://docs.microsoft.com/intune/custom-settings-macos)

[Definições personalizadas para dispositivos Windows 10 Intune](https://docs.microsoft.com/intune/custom-settings-windows-10)

[Definições personalizadas para o Windows holográfica para dispositivos de negócio no Intune](https://docs.microsoft.com/intune/custom-settings-windows-holographic)
