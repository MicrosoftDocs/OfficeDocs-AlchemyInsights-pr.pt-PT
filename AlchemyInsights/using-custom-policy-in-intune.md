# <a name="using-custom-policy-in-intune"></a>Usando a política personalizada em Intune

Intune tem muitas configurações incorporadas para dispositivos Android, iOS, macOS e Windows. Também pode criar um perfil personalizado e, em seguida, implementar estas definições personalizadas para os seus dispositivos usando Intune. Os perfis personalizados são projetados para adicionar configurações e funcionalidades do dispositivo que não estão incorporadas no Intune.

[Configurações personalizadas para dispositivos Android em Intune](https://docs.microsoft.com/intune/custom-settings-android)

[Configurações personalizadas para dispositivos Android Enterprise em Intune](https://docs.microsoft.com/intune/custom-settings-android-for-work)

[Configurações personalizadas para dispositivos iOS em Intune](https://docs.microsoft.com/intune/custom-settings-ios)

[Configurações personalizadas para dispositivos macOS em Intune](https://docs.microsoft.com/intune/custom-settings-macos)

[Definições personalizadas para dispositivos Windows 10 em Intune](https://docs.microsoft.com/intune/custom-settings-windows-10)

[Definições personalizadas para Windows Holographic para dispositivos empresariais em Intune](https://docs.microsoft.com/intune/custom-settings-windows-holographic)
