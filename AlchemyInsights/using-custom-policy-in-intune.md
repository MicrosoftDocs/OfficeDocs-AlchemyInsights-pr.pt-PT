# <a name="using-custom-policy-in-intune"></a>Usando a política personalizada em Intune

A Intune tem muitas configurações incorporadas para dispositivos Android, iOS, macOS e Windows. Também pode criar um perfil personalizado e, em seguida, implementar estas definições personalizadas para os seus dispositivos usando o Intune. Os perfis personalizados são projetados para adicionar configurações e funcionalidades do dispositivo que não são incorporadas ao Intune.

[Configurações personalizadas para dispositivos Android em Intune](https://docs.microsoft.com/intune/custom-settings-android)

[Configurações personalizadas para dispositivos Android Enterprise em Intune](https://docs.microsoft.com/intune/custom-settings-android-for-work)

[Definições personalizadas para dispositivos iOS em Intune](https://docs.microsoft.com/intune/custom-settings-ios)

[Definições personalizadas para dispositivos macOS no Intune](https://docs.microsoft.com/intune/custom-settings-macos)

[Definições personalizadas para dispositivos Windows 10 no Intune](https://docs.microsoft.com/intune/custom-settings-windows-10)

[Definições personalizadas para Windows Holographic para dispositivos Empresariais em Intune](https://docs.microsoft.com/intune/custom-settings-windows-holographic)
