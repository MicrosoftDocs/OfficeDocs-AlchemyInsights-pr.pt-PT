# <a name="create-and-use-a-shared-mailbox"></a>Criar e usar uma caixa de correio partilhada

Uma caixa de correio partilhada permite que mais de uma pessoa envie e receba e-mails para um pseudónimo, como sales@contoso.com, sem atribuir outra licença.
  
- [Siga estes passos](https://portal.office.com/AdminPortal/Home#/AssistedGuide/addemailoptions) no centro de administração microsoft 365 para começar. 

Também pode ler o artigo [Criar uma caixa de correio partilhada,](https://docs.microsoft.com/microsoft-365/admin/email/create-a-shared-mailbox) que lhe dirá tudo o que precisa de saber sobre caixas de correio partilhadas, incluindo como usá-las.
