# <a name="resolving-print-failures-in-word"></a>Resolução de falhas de impressão no Word

Se estiver a receber erros ao imprimir no Word, tente imprimir a partir de um novo documento de teste para determinar se o seu problema está relacionado com um documento específico.

- Se conseguir imprimir no documento de teste mas ainda não conseguir imprimir o documento original, o documento original poderá estar danificado. Siga as instruções em [Como resolver os problemas dos documentos danificados no Word](https://docs.microsoft.com/office/troubleshoot/word/damaged-documents-in-word#update-microsoft-office-and-windows).
- Se ainda não conseguir imprimir, siga as instruções em [falhas de impressão de resolução de problemas no Word](https://docs.microsoft.com/office/troubleshoot/word/print-failures-in-word) para diagnosticar e corrigir o problema.
