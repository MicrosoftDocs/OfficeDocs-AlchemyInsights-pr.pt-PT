# <a name="sharepoint-publishing-infrastructure"></a>Infraestrutura de publicação sharepoint


SharePoint inclui um conjunto clássico de recursos que permitem que você autor e publicar páginas web ricas para a sua intranet. Ativar a infra-estrutura de publicação SharePoint no nível de coleta do site para ligá-los.

Para mais informações, visite os links abaixo.

[Ativar recursos de publicação](https://support.office.com/article/Enable-publishing-features-479677A6-8B33-4AC7-907D-071C1C7E4518)

[Publicação de recursos](https://support.office.com/article/Features-enabled-in-a-SharePoint-Online-publishing-site-3AB3810C-3C2C-4361-9D0E-0CBE666EA0B0?wt.mc_id=O365_Portal_MMaven#__toc336865553)

Notas:

- Os modelos clássicos do site de publicação incluem os recursos de publicação por padrão.

- Os recursos clássicos de publicação não estão disponíveis em sites modernos. Sites modernos fornecem uma alternativa mais nova por padrão.
