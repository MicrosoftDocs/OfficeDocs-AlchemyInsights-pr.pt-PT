# <a name="sharepoint-publishing-infrastructure"></a>Infra-estrutura de publicação do SharePoint


<p>O SharePoint inclui um conjunto de funcionalidades que permitem-lhe criar e publicar páginas Web completas para a intranet clássico. Active a infra-estrutura de publicação de SharePoint ao nível da colecção de sites para activá-los.</p>  <p>Para obter mais informações, visite as hiperligações abaixo.</p>  <ul>  <li><a href="https://support.office.com/en-us/article/Enable-publishing-features-479677A6-8B33-4AC7-907D-071C1C7E4518">Activar funcionalidades de publicação</a></li>  <li><a href="https://support.office.com/en-us/article/Features-enabled-in-a-SharePoint-Online-publishing-site-3AB3810C-3C2C-4361-9D0E-0CBE666EA0B0?wt.mc_id=O365_Portal_MMaven#__toc336865553">Funcionalidades de publicação</a></li>  </ul>  <p>Notas:</p>  <ul>  <li>Os modelos de site de publicação classic incluem as funcionalidades de publicação por predefinição.</li>  <li>Funcionalidades de publicação clássicas não estão disponíveis em sites modernas. Modernos sites fornecem uma alternativa mais recente por predefinição.</li>  </ul>
