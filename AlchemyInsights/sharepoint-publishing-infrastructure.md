# <a name="sharepoint-publishing-infrastructure"></a>Infraestrutura de publicação SharePoint


O SharePoint inclui um conjunto clássico de funcionalidades que lhe permitem autorizar e publicar páginas web ricas para a sua intranet. Ative a Infraestrutura de Publicação SharePoint ao nível da recolha do site para as ligar.

Para mais informações visite os links abaixo.

[Ativar funcionalidades de publicação](https://support.office.com/article/Enable-publishing-features-479677A6-8B33-4AC7-907D-071C1C7E4518)

[Funcionalidades de publicação](https://support.office.com/article/Features-enabled-in-a-SharePoint-Online-publishing-site-3AB3810C-3C2C-4361-9D0E-0CBE666EA0B0?wt.mc_id=O365_Portal_MMaven#__toc336865553)

Notas:

- Os modelos clássicos do site de publicação incluem as funcionalidades de publicação por padrão.

- As funcionalidades de publicação clássicas não estão disponíveis em sites modernos. Os sites modernos fornecem uma alternativa mais recente por defeito.
