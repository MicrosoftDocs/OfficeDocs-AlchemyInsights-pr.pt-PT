# <a name="how-to-select-all-in-outlookcom"></a>Como "seleccionar todos os" no Outlook.com

Só pode seleccionar todos se todas as mensagens estão visíveis. Isto é evitar acidentais acções nas mensagens que não é possível ver.

**CTRL + clique** selecciona várias mensagens individuais.

**SHIFT + clique** selecciona todas as mensagens entre a primeira e última mensagem que seleccionar.
