# <a name="solutions-for-error-30179-25"></a>Soluções para o erro 30179-25

Poderá ver um erro com a mensagem "não foi possível transmitir em sequência Office" (código de erro [30179-25](https://support.office.com/article/e40d3c7d-98f6-4284-94a0-882beaa44593?wt.mc_id=Alchemy_ClientDIA)) se o disco rígido está a ficar sem espaço.
  
[Livre espaço no disco rígido](https://support.microsoft.com/help/12425/windows-10-free-up-drive-space) e, em seguida, [reinstalar o Office.](https://portal.office.com/OLS/MySoftware.aspx)
